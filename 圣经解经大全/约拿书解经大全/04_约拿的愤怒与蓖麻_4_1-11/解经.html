约拿书 4:1-11 解经：约拿的愤怒与蓖麻

一、经文概览

项目内容
书卷约拿书（Jonah）
段落4:1-11
主题约拿因神怜悯尼尼微而发怒，神用蓖麻、虫子和东风教导他怜悯的功课
历史背景承接第3章，尼尼微悔改后
文学体裁叙事文学（先知故事）；以对话和象征行动为主要形式

二、经文结构

``` 4:1-3 约拿的愤怒与求死 4:4-5 神的质问与约拿的等待 4:6-8 蓖麻、虫子与东风 4:9-11 神的最终教导——怜悯的功课 ```

三、逐段解经

1. 约拿的愤怒与求死（4:1-3）

这事约拿大大不悦，且甚发怒。（4:1）

关键词解析：

  • "大大不悦"（וַיֵּרַע אֶל יוֹנָה רָעָה גְדוֹלָה）：原文直译为"在约拿眼中看为极大的恶"。讽刺的是，尼尼微人的"恶"（רָעָה，1:2）曾达到神面前，现在约拿自己的"恶"（同一个词）也爆发了。
  • "且甚发怒"（וַיִּחַר לוֹ）：原文意为"他的鼻孔燃烧"，是愤怒的强烈表达。约拿的愤怒不是小情绪，而是强烈的义愤。

"耶和华啊，我在本国的时候岂不是这样说吗？我知道你是有恩典、有怜悯的神，不轻易发怒，有丰盛的慈爱，并且后悔不降所说的灾，所以我急速逃往他施去。"（4:2）

关键词解析：

  • "我在本国的时候岂不是这样说吗"：约拿承认他在以色列时就已经预见到了这个结果。他的逃跑不是出于恐惧，而是出于对神的认识。
  • "有恩典、有怜悯的神"：这是对神本性的准确描述，引用了出埃及记34:6的神圣宣告。约拿知道神的本质是怜悯，但他不希望这个怜悯临到尼尼微。
  • "不轻易发怒，有丰盛的慈爱"：完整的神圣宣告（参出34:6；民14:18；尼9:17；诗86:15；103:8；145:8；珥2:13）。约拿对神的认识是正确的，但他的应用是错误的。
  • "并且后悔不降所说的灾"：约拿知道神可能会"后悔"（参3:10），所以他不愿来。
  • "所以我急速逃往他施去"：约拿的逃跑不是出于恐惧，而是出于对神怜悯的抵抗。这是一个惊人的告白——约拿害怕的不是神的审判，而是神的怜悯。

"耶和华啊，现在求你取我的性命吧！因为我死了比活着还好。"（4:3）

关键词解析：

  • "求你取我的性命"：约拿的愤怒达到了求死的程度。这与以利亚在迦密山后的求死类似（王上19:4）。
  • "死了比活着还好"：约拿宁愿死，也不愿看到神怜悯他的敌人。这是一个令人心碎的告白，揭示了民族主义如何扭曲了一个人的信仰。

神学要点：

约拿书4:2是全书的神学核心——它揭示了约拿逃跑的真正动机。约拿对神的认识是正确的（有恩典、有怜悯），但他的回应是错误的（不愿神怜悯尼尼微）。这提醒我们：正确的神学知识，如果不受爱的引导，可以成为冷酷的教条。

2. 神的质问与约拿的等待（4:4-5）

耶和华说："你这样发怒合乎理吗？"（4:4）

关键词解析：

  • "合乎理吗？"（הַהֵיטֵב）：原文直译为"这样好吗？"神用温和的问题来回应约拿的愤怒，而不是直接责备。这反映了神的耐心和教导的方式。
  • 这个问题是整卷约拿书的核心问题——约拿的愤怒是否合理？神的怜悯是否值得愤怒？

于是约拿出城，在城的东边搭了一座棚，坐在棚的荫下，要看看那城究竟如何。（4:5）

关键词解析：

  • "出城，在城的东边"：约拿离开尼尼微城，在城外等候。这与1:3的逃跑形成对比——之前他逃离神，现在他逃离尼尼微。
  • "搭了一座棚"：临时性的棚子，暗示约拿打算短暂等待。
  • "要看看那城究竟如何"：约拿仍然希望神会改变主意，降灾给尼尼微。他在等待，但等待的方向是错误的——他在等待审判，而不是盼望怜悯。

3. 蓖麻、虫子与东风（4:6-8）

耶和华神安排一棵蓖麻，使其发生高过约拿，影儿遮盖他的头，救他脱离苦楚。约拿因这棵蓖麻大大喜乐。（4:6）

关键词解析：

  • "安排"（וַיְמַן）：这个词在约拿书中反复出现（1:17大鱼；4:6蓖麻；4:7虫子；4:8东风），强调神对整个故事的主权掌控。
  • "蓖麻"（קִיקָיוֹן）：可能是蓖麻树（ricinus communis）或一种快速生长的植物。蓖麻的特点是生长迅速，能在一夜之间长出。
  • "救他脱离苦楚"：蓖麻的荫凉使约拿免受烈日的曝晒。神在约拿愤怒时仍然供应他的需要。
  • "大大喜乐"：约拿因蓖麻而喜乐，这与他因尼尼微得救而发怒形成对比——他关心自己的舒适超过关心百万灵魂的命运。

神却安排一条虫子在黎明的时候来咬这蓖麻，以致枯槁。日头出来的时候，神安排炎热的东风，日头曝晒约拿的头，使他发昏。（4:7-8上）

关键词解析：

  • "安排一条虫子"：神再次"安排"。虫子咬蓖麻，使约拿失去荫凉。
  • "炎热的东风"：从东方沙漠吹来的热风（参结17:10；何13:15），带来极高的温度。
  • "日头曝晒约拿的头，使他发昏"：约拿的身体和精神都到了极限。

他就为自己求死，说："我死了比活着还好。"（4:8下）

关键词解析：

  • 约拿第二次求死（参4:3）。第一次是因神怜悯尼尼微，第二次是因失去蓖麻的荫凉。他对一棵植物的关心，超过了对百万灵魂的关心。

神学要点：

神使用三个"安排"来教导约拿： 1. 蓖麻——神供应约拿的需要 2. 虫子——神剥夺约拿的舒适 3. 东风——神使约拿经历困境

这三个"安排"都是出于神的主权，目的是教导约拿关于怜悯的功课。蓖麻的兴衰，成为约拿生命的缩影。

4. 神的最终教导——怜悯的功课（4:9-11）

神对约拿说："你因这棵蓖麻发怒合乎理吗？"他说："我发怒以至于死，都合乎理！"（4:9）

关键词解析：

  • "你因这棵蓖麻发怒合乎理吗？"：神重复了4:4的问题，但这次的对象从"尼尼微"变成了"蓖麻"。这是一个巧妙的教导策略——约拿为一棵蓖麻发怒当然不合理，那么为尼尼微发怒就合理吗？
  • "我发怒以至于死，都合乎理"：约拿坚持自己的愤怒是合理的。他没有领悟神的教导。

耶和华说："你尚且爱惜这棵蓖麻，这蓖麻不是你栽种的，也不是你培养的，一夜发生，一夜干死，你尚且爱惜它；何况这尼尼微大城，其中不能分辨左手右手的有十二万多人，并有许多牲畜，我岂能不爱惜呢？"（4:10-11）

关键词解析：

  • "你尚且爱惜"（אַתָּה חַסְתָּה）：原文"חַסְתָּה"与"חֶסֶד"（慈爱）同根。约拿对蓖麻表现出了一种扭曲的"慈爱"。
  • "这蓖麻不是你栽种的，也不是你培养的"：约拿对蓖麻没有付出任何劳动，却爱惜它。那么，神创造了尼尼微人，难道不应该更爱惜他们吗？
  • "一夜发生，一夜干死"：蓖麻的存在是短暂的，但约拿却如此珍惜。尼尼微人是有永恒灵魂的存在，神岂不应该更珍惜他们？
  • "不能分辨左手右手的有十二万多人"：这可能指幼儿（太小还不能分辨左右手），也可能指灵性上无知的人。无论如何，他们是需要怜悯的人。
  • "并有许多牲畜"：连牲畜也在神的关怀之中。这与3:8的禁食相呼应——连牲畜也参与了悔改的仪式。
  • "我岂能不爱惜呢？"：这是整卷约拿书的最终答案。神的怜悯不是软弱，而是出于他本性的爱。神对万有的爱，远远超过人对一棵植物的爱。

神学要点：

约拿书以一个问题结束，没有给出约拿的回应。这个开放式的结尾邀请读者自己回答：

1. 你是否同意神的怜悯？ 2. 你是否愿意神怜悯你的"尼尼微"？ 3. 你的爱是否像神的爱一样宽广？

四、神学主题

1. 神的怜悯

  • 神的怜悯是他本性的一部分（出34:6）
  • 神的怜悯不局限于以色列，而是涵盖万族
  • 神的怜悯不是软弱，而是出于爱

2. 民族主义与信仰

  • 约拿的民族主义扭曲了他的信仰
  • 他关心以色列超过关心神的旨意
  • 信仰应该超越民族、种族、文化的界限

3. 神的主权与教导

  • 神"安排"了大鱼、蓖麻、虫子、东风
  • 一切都是为了教导约拿关于怜悯的功课
  • 神使用环境来塑造他的仆人

4. 爱的优先次序

  • 约拿爱蓖麻超过爱尼尼微人
  • 神爱万有，包括约拿不愿爱的人
  • 我们的爱是否像神的爱一样宽广？

5. 开放式的结尾

  • 约拿书没有给出约拿的回应
  • 读者被邀请自己回答神的问题
  • 每个读者都需要面对这个挑战

五、关键经文深度解读

约拿书 4:2 — 约拿的告白

"我知道你是有恩典、有怜悯的神，不轻易发怒，有丰盛的慈爱，并且后悔不降所说的灾。"

这是旧约中对神本性最准确的描述之一，引用了出埃及记34:6的神圣宣告。约拿书4:2与出埃及记34:6的对应关系：

出埃及记 34:6约拿书 4:2
有怜悯有恩典
有恩典有怜悯
不轻易发怒不轻易发怒
有丰盛的慈爱有丰盛的慈爱
赦免罪孽后悔不降所说的灾

约拿对神的认识是完全正确的，但他的应用是完全错误的。他知道神是怜悯的，但他不希望这个怜悯临到他的敌人。这提醒我们：正确的神学知识，如果不被爱引导，可以成为冷酷的教条。

约拿书 4:10-11 — 神的最终答案

"你尚且爱惜这棵蓖麻……何况这尼尼微大城……我岂能不爱惜呢？"

这节经文揭示了神的爱的逻辑：

1. 约拿爱蓖麻（虽然他没有栽种它） 2. 神创造了尼尼微人（他们是神的作品） 3. 因此，神爱尼尼微人（这是逻辑的必然）

这个逻辑挑战了约拿的民族主义——如果约拿能爱一棵他没有栽种的植物，那么神——创造了尼尼微人的神——岂不应该更爱他们吗？

约拿书 4:11 — 不能分辨左手右手的人

"其中不能分辨左手右手的有十二万多人。"

这个描述有几种可能的解释：

1. 幼儿：太小还不能分辨左右手。如果这样，那么十二万只是幼儿的数量，总人口可能超过六十万。 2. 灵性无知的人：在灵性上不能分辨是非的人。这强调了他们的需要——他们是需要怜悯的人。 3. 整体性的无知：整个尼尼微城在灵性上处于无知的状态。这呼应了使徒行传17:30——"世人蒙昧无知的时候，神并不监察。"

无论哪种解释，重点是：这些人需要怜悯，而不是审判。

六、与新约的联系

马太福音 5:44-45 — 爱仇敌

"要爱你们的仇敌，为那逼迫你们的祷告。这样，就可以作你们天父的儿子。因为他叫日头照好人，也照歹人；降雨给义人，也给不义的人。"

耶稣的教导直接回应了约拿的问题。神的爱是普世的，不局限于义人或选民。约拿不愿神怜悯他的敌人，但耶稣教导我们要爱我们的敌人。

马太福音 9:36 — 耶稣的怜悯

"他看见许多的人，就怜悯他们，因为他们困苦流离，如同羊没有牧人一般。"

耶稣对人的怜悯，正是约拿书4:11中神的怜悯的体现。耶稣看到人的需要，就动了慈心。约拿看到尼尼微人的需要，却发了怒。

路加福音 15:7 — 天上的喜乐

"我告诉你们，一个罪人悔改，在天上也要这样为他欢喜，较比为九十九个不用悔改的义人，欢喜更大。"

约拿不愿看到尼尼微人悔改，但天上的天使却为罪人的悔改欢喜。约拿的愤怒与天上的喜乐形成尖锐的对比。

罗马书 9:15-16 — 神的怜悯主权

"我要怜悯谁，就怜悯谁；要恩待谁，就恩待谁。据此看来，这不在乎那定意的，也不在乎那奔跑的，只在乎发怜悯的神。"

保罗引用出埃及记33:19来说明神的怜悯主权。约拿书正是这个真理的例证——神选择怜悯尼尼微，不是因为他们配得，而是因为他是怜悯的神。

七、实际应用

1. 面对神的怜悯

  • 我们是否像约拿一样，不愿某些人得到神的怜悯？
  • 我们是否限制了神的怜悯的范围？
  • 神的怜悯是否挑战了我们的偏见？

2. 民族主义与信仰

  • 约拿的民族主义扭曲了他的信仰
  • 我们是否也让政治、种族、文化影响了我们的信仰？
  • 信仰应该超越一切人为的界限

3. 爱的优先次序

  • 约拿爱蓖麻超过爱尼尼微人
  • 我们是否也关心自己的舒适超过关心他人的灵魂？
  • 我们的爱是否像神的爱一样宽广？

4. 神的教导方式

  • 神使用蓖麻、虫子、东风来教导约拿
  • 神也使用我们生命中的环境来教导我们
  • 我们是否在环境中学到了神要教导的功课？

5. 开放式的挑战

  • 约拿书以问题结束，邀请读者自己回答
  • 你是否同意神的怜悯？
  • 你是否愿意神怜悯你的"尼尼微"？

八、思考问题

1. 约拿为什么发怒？他的愤怒是否合理？ 2. 约拿对神的认识（4:2）是正确的，但他的应用是错误的。我们是否也有类似的矛盾？ 3. 蓖麻的比喻如何帮助我们理解神的怜悯？ 4. 约拿书为什么以问题结束？这对我们有什么挑战？ 5. 在你的生命中，谁是你的"尼尼微人"？你是否愿意神怜悯他们？

九、祈祷

主啊，感谢你，因为你是有恩典、有怜悯的神。你爱万有，包括我不愿爱的人。求你赦免我的狭隘和偏见。帮助我像你一样，有宽广的爱。当我像约拿一样发怒时，求你用蓖麻的功课教导我。让我明白，你的怜悯不是软弱，而是出于你本性的爱。奉耶稣基督的名祷告，阿们。
