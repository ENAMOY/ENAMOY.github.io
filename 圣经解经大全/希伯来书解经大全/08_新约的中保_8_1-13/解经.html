希伯来书 8:1-13 解经：新约的中保

一、经文概览

项目内容
书卷希伯来书（Πρὸς Ἑβραίους）
段落8:1-13
主题基督在天上真帐幕中作大祭司——新约的中保
历史背景读者熟悉耶路撒冷圣殿的敬拜制度，需要理解基督在天上的事奉
文学体裁讲章式书信，天上／地下对比，引用耶31:31-34作为新约的根基

二、经文结构

``` 8:1-2 主题：在天上真帐幕作大祭司 8:3-5 地上祭司的影儿与天上真像 8:3-4 地上献礼物的祭司 8:5 地上帐幕是天上形状的影像 8:6 基督更美之约的中保 8:7-12 引用耶31:31-34——新约的应许 8:7 前约有瑕疵 8:8-9 不像与以色列所立的约 8:10 我要将我的律法放在他们里面 8:11 他们不用各人教导自己的乡邻 8:12 我要宽恕他们的不义 8:13 既说新约，就以前约为旧了 ```

三、逐段解经

1. 在天上真帐幕作大祭司（8:1-2）

我们所讲的事，其中第一要紧的，就是我们有这样的大祭司，已经坐在天上至大者宝座的右边，在圣所，就是真帐幕里作执事。这帐幕是主所支的，不是人所支的。（8:1-2）

8:1 — "第一要紧的"（κεφάλαιον）

  • "第一要紧的"（κεφάλαιον）：这个词原意是"头"或者"主要的点"。作者在总结了第7章的论证（基督是麦基洗德等次的大祭司）后，给出了整卷书的"中心论题"。
  • "有"（ἔχομεν）：现在时，持续拥有。我们不是"希望有"或"将会有"——我们确实有。
  • "这样的大祭司"（τοιοῦτον ἀρχιερέα）：指前面描述的那一位——圣洁、无邪恶、无玷污、长远活着、完全到永远的。
  • "已经坐在天上至大者宝座的右边"：这是基督升高状态的最终描述。"坐下了"（ἐκάθισεν）——表明工作已经完成。

8:2 — "真帐幕"（τῆς σκηνῆς τῆς ἀληθινῆς）

  • "真帐幕"（ἀληθινός）：不是与"假"相对的真，而是与"影子和预表"相对的本体（real, genuine）。地上的会幕是"影像"（ὑπόδειγμα）和"影子"（σκιά），天上的是"真"的。
  • "执事"（λειτουργός）：公共的侍奉者——如同在圣殿中事奉的祭司。但这里他是在"真帐幕"（天上）事奉。
  • "是主所支的，不是人所支的"：强调天上帐幕的神圣起源——不是人手所造的（参9:11"不是人手所造"）。

2. 地上的影子与天上的真像（8:3-5）

凡大祭司都是为献礼物和祭物设立的，所以这位大祭司也必须有所献的。他若在地上，必不得为祭司，因为已经有照律法献礼物的祭司。他们供奉的事本是天上事的形状和影像，正如摩西将要造帐幕的时候，蒙神警戒他，说："你要谨慎，作各样的物件都要照着在山上指示你的样式。"（8:3-5）

8:3 — "凡大祭司都是为献礼物和祭物设立的"

  • 如果耶稣是大祭司，他必须有所献——他的献祭就是自己（9:14,28）。
  • 基督的献祭与旧约献祭的根本区别在于——他"献上"（προσφέρειν）的是一次性的（7:27），而旧约的是反复的。

8:4 — "他若在地上，必不得为祭司"

  • 如果耶稣在地上（按照地上的制度），他不能成为祭司——因为他不是利未支派的。但在天上，他不需要地下的职称。这再次强调了祭司等次的完全转换。

8:5 — "照着在山上指示你的样式"

  • 引用出埃及记25:40（七十士译本）。摩西在山上的"样式"（τύπος）——视觉化的蓝图——是天上实体的复制品。
  • "形状"（ὑπόδειγμα）和"影像"（σκιά）：地上帐幕是"在轮廓中"跟随天上实体的——像是物体的轮廓或影子。影子不是无用的——它证明物体的存在，并且提供物体的形状信息——但它不是物体本身。

3. 更美之约的中保（8:6）

如今耶稣所得的职任是更美的，正如他作更美之约的中保。这约原是凭更美之应许立的。（8:6）

  • "更美的职任"（διαφορωτέρας λειτουργίας）：比较级——比地上祭司的职任更美。不同之处不在于技术上的优越，而在于本质上的超越。
  • "中保"（μεσίτης）：中间人。基督是神与人之间的唯一中保（提前2:5）。地上的祭司也是中保，但基督是更美之约的中保——这约是凭"更美之应许"立的。

4. 新约的应许（8:7-12）

那前约若没有瑕疵，就无处寻求后约了。所以主指责他的百姓说："日子将到，我要与以色列家和犹大家另立新约，不像我拉着他们祖宗的手，领他们出埃及的时候，与他们所立的约。因为他们不恒心守我的约，我也不理他们。这是主说的。"主又说："那些日子以后，我与以色列家所立的约乃是这样：我要将我的律法放在他们里面，写在他们心上；我要作他们的神，他们要作我的子民。他们不用各人教导自己的乡邻和自己的弟兄，说：'你该认识主'；因为他们从最小的到至大的，都必认识我。我要宽恕他们的不义，不再记念他们的罪愆。"（8:7-12）

8:7 — "前约若有瑕疵"

  • "瑕疵"（μέμψις）：责备、指责。不是旧约本身有道德瑕疵——律法是圣洁的（罗7:12）——而是在约的建立和执行中，人没有能力遵守。旧约的"缺陷"不是律法有错，而是它不能赋予遵守的能力。

8:8-9 — "另立新约"

  • 引用耶利米书31:31-34——这是旧约中最重要的关于新约的预言之一。
  • "新约"（διαθήκην καινήν）："新"（καινή）——与"旧"相对，指性质上的新、不同种类的新的。
  • "不像我拉着他们祖宗的手，领他们出埃及的时候，与他们所立的约"：新约不同于西奈之约。西奈之约是写在石版上的（外在的、要求性的），新约是写在心版上的（内在的、赋予能力的）。

8:10 — "放在他们里面，写在他们心上"

  • 新约的独特之处在于"内在化"——律法不再是外在的规条，而是成为内在的引导。这是圣灵的工作（结36:26-27——"我必将我的灵放在你们里面"）。
  • "我要作他们的神，他们要作我的子民"：这是约的核心公式（出6:7；利26:12；耶31:33）。在旧约中，这个公式指向将来的完全实现。

8:11 — "都必认识我"

  • "认识"（γνῶθι）——不是头脑中的知识，是关系性的认识（יָדַע），如同亚当"认识"夏娃。
  • "从最小的到至大的"：新约的知识普及是突破性的——不再是少数人（祭司、先知）拥有对神的认识，而是所有与神立约的人。

8:12 — "不再记念他们的罪愆"

  • 新约的核心：罪的完全赦免。"记念"（μνησθῶ）不是神"忘记"（像人因记忆力差而忘记），而是神有意识地决定"不再想起"（赛43:25）。在约的框架中，罪造成了关系的破裂；罪被赦免意味着破裂被修复。
  • "宽恕"（ἵλεως）——与"挽回祭"（ἱλάσκεσθαι, 2:17）的词根相同。赎罪的最终目的是宽恕。

5. "前约为旧"（8:13）

既说新约，就以前约为旧了；但那渐旧渐衰的，就必快归无有了。（8:13）

  • "新约"（καινήν）的宣告本身意味着旧约已经"旧了"（πεπαλαίωκεν）。
  • "快归无有了"（ἐγγὺς ἀφανισμοῦ）：作者可能指着地上的耶路撒冷圣殿说的——在主后70年被罗马人摧毁。这节经文对第一读者来说具有紧迫的现实意义——他们以为牢不可破的制度，实际上正在消失。

四、神学主题

1. 天上的真帐幕

地上的会幕和圣殿是从属的、预表性的。基督事奉的场所是"天上的真帐幕"——不是人手所造的，是神亲自设立的。

2. 新约取代旧约

耶利米书31:31-34的预言在基督里应验。新约的本质是：内在的律法、直接的知识、完全的赦免。

3. 基督是中保

中保的角色不仅是在神人之间传达信息，而是调解关系、扮演和谐者。基督作为"更美之约的中保"，他的中保工作是完美且完全的。

4. 影子和实体

地上的是"影子"（σκιά）和"形状"（ὑπόδειγμα），天上的是"真像"（ἀληθινός）。影子指向实体，但不能代替实体。

五、关键经文深度解读

希伯来书 8:5 — "照着在山上指示你的样式"

摩西被指示的"样式"（τύπος）——这个词在希腊文中可以指"印记"、"模型"、"图案"。出25:40的原始语境中，神告诉摩西要按照"天上的样式"建造会幕。希伯来书的作者将"样式"解释为"拷贝"（copy）和"影子"（shadow）。地上的会幕不是神"居住"的地方——它是天上实体的"立体照片"——真实指向（但不等同于）那更伟大的实体，即基督在天上的事奉。

希伯来书 8:10 — "我要将我的律法放在他们里面"

新约的"律法"（νόμος）不是被取消，而是被内在化。耶利米书预言中的"放在里面"和"写在心上"指向圣灵的内住和重生（参结36:26-27）。这不是反律主义（antinomianism）——而是恩典使律法的义得以成全（罗8:3-4）。

六、与新约的联系

1. 耶利米书31:31-34——新约的预言

这是整卷希伯来书引用最长的旧约段落，也是全书新约神学的核心根基。

2. 路加福音22:20——主餐中的新约

"这杯是用我血所立的新约"——耶稣在最后的晚餐中引用耶31:31，宣告新约在祂的血中生效。

3. 哥林多后书3:3-6——写在心版上

"你们明显是基督的信……不是用墨写的，乃是用永生神的灵写的；不是写在石版上，乃是写在心版上。"——保罗同样表达了新约的内在化特征。

七、实际应用

1. 不要回头看

如果新约已经来到，就不需要再回望旧约的影子和预表。基督已经成就了一切。

2. 内在的转变

新约不只是外在的规范——它是内在的改变。神的律法不是在我们外面——在我们里面。

3. 罪被赦免的确据

"不再记念他们的罪愆"——我们不需要活在罪的阴影中。神已经选择不再记念我们的罪。

4. 直接认识神

新约之下，我们不需要通过其他中保（祭司、圣徒、天使）来认识神。我们直接通过基督来到神面前。

八、思考问题

1. "天上真帐幕"对比"地上的影子"——这如何改变你对教会建筑或敬拜场所的理解？ 2. 旧约"写在石版上" vs 新约"写在心版上"——你感受到内在你对神的律法的爱慕吗？ 3. "都必认识我"——你今天对神的认识是间接的（通过别人）还是直接的（亲身经历）？ 4. 耶利米书31:34"不再记念他们的罪"——这对你的自我控告和罪疚感有何安慰？ 5. 基督是"更美之约的中保"——你生命中是否有任何事情是他不能作为中保的？

九、祈祷

主耶稣基督，你是更美之约的中保，坐在天上真帐幕中神的右边。感谢你为我们立了这约——不是写在石版上，而是写在我们的心版上。谢谢你宽恕我们的不义，不再记念我们的罪愆。求你使我们对你的认识成为亲密的、活的关系——不是二手知识，而是从心里发出的认识。帮助我们在新约的自由中站立得稳，不回到旧约的捆绑中。奉你的名祷告，阿们。
