约翰福音解经：伯大尼膏抹与骑驴进京

经文范围

约翰福音 12:1-50

主题概述

这段经文记录了耶稣在伯大尼受膏、骑驴进耶路撒冷，以及他对众人最后的公开教导。这是耶稣公开事工的高潮，也是他走向十字架的转折点。

经文与逐句解经

一、伯大尼的膏抹（12:1-11）

【经文】12:1-3 逾越节前六日，耶稣来到伯大尼，就是他叫拉撒路从死里复活之处。有人在那里给耶稣预备筵席；马大伺候，拉撒路也在那同耶稣坐席的人中。马利亚就拿着一斤极贵的真哪哒香膏，抹耶稣的脚，又用自己的头发去擦，屋里就满了膏的香气。

逐句解经：

  • "逾越节前六日"：耶稣最后一次进入耶路撒冷。
  • "马大伺候"：马大用行动服事。
  • "拉撒路也在那同耶稣坐席的人中"：拉撒路的复活成为活生生的见证。
  • "一斤极贵的真哪哒香膏"：价值三十两银子（约一年工资）。
  • "抹耶稣的脚"：表达极大的敬意和爱。

【经文】12:4-6 有一个门徒，就是那将要卖耶稣的加略人犹大，说：「这香膏为什么不卖三十两银子周济穷人呢？」他说这话，并不是挂念穷人，乃因他是个贼，又带着钱囊，常取其中所存的。

逐句解经：

  • "为什么不卖三十两银子周济穷人？"：犹大的建议看似合理。
  • "他是个贼，又带着钱囊"：犹大的真实动机是贪心。

【经文】12:7-8 耶稣说：「由她吧！她是为我安葬之日存留的。因为常有穷人和你们同在，只是你们不常有我。」

逐句解经：

  • "她是为我安葬之日存留的"：马利亚的膏抹是为耶稣的安葬做准备。
  • "常有穷人和你们同在"：耶稣不是轻看穷人，而是指出他的时候近了。

【经文】12:9-11 有许多犹太人知道耶稣在那里就来了，不但是为耶稣的缘故，也是要看他从死里所复活的拉撒路。但祭司长商议连拉撒路也要杀了；因有好些犹太人为拉撒路的缘故，回去信了耶稣。

逐句解经：

  • "也是要看他从死里所复活的拉撒路"：拉撒路成为吸引人的见证。
  • "但祭司长商议连拉撒路也要杀了"：他们的愤怒到了极点——连死而复活的人也要杀。

二、骑驴进京（12:12-19）

【经文】12:12-16 第二天，有许多上来过节的人听见耶稣将到耶路撒冷，就拿着棕树枝出去迎接他，喊着说：「和散那！奉主名来的以色列王是应当称颂的！」耶稣得了一个驴驹，就骑上，如经上所记的说：「锡安的民哪，不要惧怕！你的王骑着驴驹来了。」这些事门徒起先不明白，等到耶稣得了荣耀以后才想起这话是指着他写的，并且众人果然向他这样行了。

逐句解经：

  • "拿着棕树枝"：棕树枝象征胜利和欢乐。
  • "和散那"（希伯来文：הוֹשִׁיעָה נָּא，hoshiya na）：意思是"求你拯救"（诗篇118:25）。
  • "奉主名来的以色列王"：群众承认耶稣是王。
  • "耶稣得了一个驴驹，就骑上"：引用撒迦利亚书9:9，弥赛亚骑驴驹进城，象征和平。
  • "门徒起先不明白"：他们后来才明白。

【经文】12:17-19 当耶稣呼唤拉撒路、叫他从死复活出坟墓的时候，同耶稣在那里的众人就作见证。众人因听见耶稣行了这神迹，就去迎接他。法利赛人彼此说：「看哪，你们是徒劳无益，世人都随从他去了。」

逐句解经：

  • "众人就作见证"：目击者传播消息。
  • "法利赛人彼此说：你们是徒劳无益"：他们承认耶稣的影响力。

三、一粒麦子（12:20-26）

【经文】12:20-22 那时，上来过节礼拜的人中，有几个希腊人。他们来见加利利伯赛大的腓力，求他说：「先生，我们愿意见耶稣。」腓力去告诉安得烈，安得烈同腓力去告诉耶稣。

逐句解经：

  • "有几个希腊人"：外邦人来寻求耶稣。
  • "我们愿意见耶稣"：他们的请求简单而真诚。

【经文】12:23-26 耶稣说：「人子得荣耀的时候到了。我实实在在地告诉你们，一粒麦子不落在地里死了，仍旧是一粒；若是死了，就结出许多子粒来。爱惜自己生命的，就失丧生命；在这世上恨恶自己生命的，就要保守生命到永生。若有人服事我，就当跟从我；我在哪里，服事我的人也要在哪里；若有人服事我，我父必尊重他。」

逐句解经：

  • "人子得荣耀的时候到了"：耶稣的"得荣耀"是通过十字架。
  • "一粒麦子不落在地里死了，仍旧是一粒"：基督的死将带来丰盛的生命。
  • "爱惜自己生命的，就失丧生命"：自私的结果是失去生命。
  • "在这世上恨恶自己生命的，就要保守生命到永生"：舍己的结果是得着生命。

四、从天上来的声音（12:27-36）

【经文】12:27-30 「我现在心里忧愁，我说什么才好呢？父啊，救我脱离这时候；但我原是为这时候来的。父啊，愿你荣耀你的名！」当时就有声音从天上来，说：「我已经荣耀了我的名，还要再荣耀。」站在旁边的众人听见，就说：「打雷了。」还有人说：「有天使对他说话。」耶稣说：「这声音不是为我，是为你们来的。」

逐句解经：

  • "我现在心里忧愁"：耶稣的人性——他为即将来临的苦难忧愁。
  • "父啊，救我脱离这时候"：耶稣表达他的痛苦。
  • "但我原是为这时候来的"：耶稣知道他的使命。
  • "父啊，愿你荣耀你的名！"：耶稣最终的祷告是愿神的名被荣耀。
  • "当时就有声音从天上来"：父神回应。
  • "这声音不是为我，是为你们来的"：这个神迹是为了坚固众人的信心。

【经文】12:31-36 「现在这世界受审判，这世界的王要被赶出去。我若从地上被举起来，就要吸引万人来归我。」耶稣这话原是指着自己将要怎样死说的。众人回答说：「我们听见律法上有话说基督是永存的，你怎么说人子必须被举起来呢？这人子是谁呢？」耶稣对他们说：「光在你们中间还有不多的时候，应当趁着有光行走，免得黑暗临到你们；那在黑暗里行走的，不知道往何处去。你们应当趁着有光，信从这光，使你们成为光明之子。」

逐句解经：

  • "这世界的王要被赶出去"：撒但的权势将被击败。
  • "我若从地上被举起来"：十字架是得胜。
  • "就要吸引万人来归我"：十字架将吸引所有人。
  • "光在你们中间还有不多的时候"：耶稣呼召他们珍惜机会。

五、约翰的总结（12:37-50）

【经文】12:37-41 他虽然在他们面前行了许多神迹，他们还是不信他。这是要应验先知以赛亚的话说：「主啊，我们所传的有谁信呢？主的膀臂向谁显露呢？」他们所以不能信，因为以赛亚又说：「主叫他们瞎了眼，硬了心，免得他们眼睛看见，心里明白，回转过来，我就医治他们。」以赛亚因为看见他的荣耀，就指着他说这话。

逐句解经：

  • "他们还是不信他"：神迹不能强迫信心。
  • "这是要应验先知以赛亚的话"：以赛亚书53:1和6:10。
  • "主叫他们瞎了眼，硬了心"：神容许人刚硬自己的心。

【经文】12:42-43 虽然如此，官长中却有好些信他的，只因法利赛人的缘故就不承认，恐怕被赶出会堂。这是因他们爱人的荣耀过于爱神的荣耀。

逐句解经：

  • "官长中却有好些信他的"：有些人信了，但不敢公开。
  • "恐怕被赶出会堂"：他们害怕社会排斥。
  • "他们爱人的荣耀过于爱神的荣耀"：他们选择人的认可，不是神的认可。

【经文】12:44-50 耶稣大声说：「信我的，不是信我，乃是信那差我来的。人看见我，就是看见那差我来的。我到世上来乃是光，叫凡信我的，不住在黑暗里。若有人听见我的话不遵守，我不审判他。我来本不是要审判世界，乃是要拯救世界。弃绝我、不领受我话的人，有审判他的——就是我所讲的道在末日要审判他。因为我没有凭着自己讲，惟有差我来的父已经给我命令，叫我说什么，讲什么。我也知道他的命令就是永生。故此，我所讲的话正是照着父对我所说的。」

逐句解经：

  • "信我的，不是信我，乃是信那差我来的"：信心的对象是父神。
  • "人看见我，就是看见那差我来的"：基督是神的显现。
  • "我到世上来乃是光"：基督是光。
  • "我所讲的道在末日要审判他"：基督的话语将成为审判的标准。

神学要点总结

1. 基督的牺牲：一粒麦子的比喻——他的死带来丰盛的生命。 2. 十字架是得胜：被举起来的基督将吸引万人。 3. 信心的呼召：趁着有光，信从这光。 4. 神的荣耀：基督的使命是荣耀神的名。

对当年读者的意义

  • 对犹太人：他们需要在光和黑暗之间做出选择。
  • 对所有读者：我们是否趁着有光信从，还是留在黑暗中？
