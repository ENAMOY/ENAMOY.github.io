智慧人的试验（传道书3:1-4:16）

经文范围

传道书3:1-4:16——凡事都有定期、永世安置在人心、日光之下各样事务。

核心神学命题

1. 凡事都有定期：神掌管时间和历史。 2. 永世安置在人心：人有永恒的渴望。 3. 日光之下的不公：人生的不公平需要神的审判来纠正。

上文回顾

上一段（1:1-2:26）宣告万事虚空。现在传道者探索时间的意义。

经文与逐句解经

一、凡事都有定期（3:1-8）

3:1 凡事都有定期，天下万务都有定时。

  • לַכֹּל זְמָן（lakol zman）：凡事都有定期——时间在神的手中。
  • זְמָן（zman）：定期、季节——出现29次。
  • עֵת（et）：时候、时机——每件事都有神指定的时间。
  • 古代近东背景：埃及和美索不达米亚智慧文学也探讨时间的循环，但传道书强调神掌管时间。

3:2 生有时，死有时；栽种有时，拔出所栽种的也有时。

  • לָלֶדֶת（laladet）：生——出生
  • לָמוּת（lamut）：死——死亡
  • לִנְטֹעַ（linto'a）：栽种——种植
  • לַעֲקוֹר（la'aqor）：拔出——拔起
  • 生与死、栽种与拔出：每件事都有神指定的时间
  • 古代近东背景：吉尔伽美什史诗也探讨生死的循环

3:3 杀戮有时，医治有时；拆毁有时，建造有时。

  • לַהֲרוֹג（laharog）：杀戮——杀死
  • לִרְפּוֹא（lirpo'）：医治——医治
  • לִהֲרוֹס（liharos）：拆毁——拆毁
  • לִבְנוֹת（livnot）：建造——建造
  • 杀戮与医治、拆毁与建造：每件事都有神指定的时间

3:4 哭有时，笑有时；哀恸有时，跳舞有时。

  • לִבְכּוֹת（livkot）：哭——哭泣
  • לִשְׂחוֹק（lischok）：笑——欢笑
  • לְהֵסְפֵּד（lehesped）：哀恸——哀悼
  • לִרְקוֹד（lirqod）：跳舞——跳舞
  • 哭与笑、哀恸与跳舞：每件事都有神指定的时间

3:5 抛掷有时，堆聚石头有时；怀抱有时，不怀抱有时。

  • לְהַשְׁלִיךְ（lehashlich）：抛掷——扔掉
  • לִכְנוֹס（liknos）：堆聚——聚集
  • לַחֲבוֹק（lachavoq）：怀抱——拥抱
  • לִרְחֹק（lirchoq）：不怀抱——远离
  • 抛掷与堆聚、怀抱与不怀抱：每件事都有神指定的时间

3:6 寻找有时，失落有时；保守有时，舍弃有时。

  • לְבַקֵּשׁ（levaqesh）：寻找——寻求
  • לְאַבֵּד（le'abed）：失落——失去
  • לִשְׁמוֹר（lishmor）：保守——守卫
  • לְהַשְׁלִיךְ（lehashlich）：舍弃——扔掉
  • 寻找与失落、保守与舍弃：每件事都有神指定的时间

3:7 撕裂有时，缝补有时；静默有时，言语有时。

  • לִקְרוֹעַ（liqro'a）：撕裂——撕开
  • לִתְפּוֹר（litpor）：缝补——缝合
  • לַחֲשׁוֹת（lachashot）：静默——沉默
  • לְדַבֵּר（ledabber）：言语——说话
  • 撕裂与缝补、静默与言语：每件事都有神指定的时间

3:8 喜爱有时，恨恶有时；争战有时，和好有时。

  • לֶאֱהֹב（le'ehov）：喜爱——爱
  • לִשְׂנֹא（lisno'）：恨恶——恨
  • מִלְחָמָה（milchamah）：争战——战争
  • שָׁלוֹם（shalom）：和好——和平
  • 喜爱与恨恶、争战与和好：每件事都有神指定的时间
  • 这不是宿命论，而是承认神掌管时间的每一个方面
  • 基督论预表：基督道成肉身也有"时候"（约7:6"我的时候还没有"）

二、永世安置在人心（3:9-15）

3:9 做工的人在他的劳碌上有什么益处呢？

  • יִתְרוֹן（yitron）：益处、利润——传道书的关键问题
  • עָמַל（amal）：劳碌——辛劳、劳苦
  • 做工的人有什么益处：这是传道书的核心问题
  • 古代近东背景：埃及智慧文学也探讨劳碌的意义
  • 基督论预表：基督说"我就是生命的粮，到我这里来的，必定不饿"（约6:35）

3:10 我见神叫世人劳苦，使他们在这其中受经练。

  • עִנְיָן（inyan）：劳苦、忧虑——人生的重担是神允许的试炼
  • עָנָה（anah）：受经练——试炼、磨练
  • 神叫世人劳苦：这是神允许的试炼
  • 古代近东背景：约伯记也探讨苦难的意义
  • 基督论预表：基督也经历苦难，学习顺从（来5:8）

3:11 神造万物，各按其时成为美好，又将永生安置在世人心里。然而神从始至终的作为，人不能参透。

  • אֶת־הָעֹלָם נָתַן בְּלִבָּם（et-ha'olam natan b'libbam）：将永世安置在人心。
  • עוֹלָם（olam）：永世、永恒——人心中有永恒的渴望。
  • 这是传道书最深刻的神学洞察：人有永恒的渴望，但日光之下不能满足。
  • 基督论预表：唯有基督能满足人心中永恒的渴望（约4:13-14"我所赐的水要在他里头成为泉源"）。
  • 古代近东背景：吉尔伽美什史诗中英雄追求永生却失败，传道书指出永恒在人心中，但需要神来满足。

3:12-13 我知道世人，莫强如终身喜乐吃喝，并且人人吃喝，在他一切劳碌中享福，这也是神的恩赐。我知道神一切所做的都必永存；无所增添，无所减少。神这样行，是要人在他面前存敬畏的心。

  • טוֹב אֲשֶׁר־יֹאכַל וְשָׁתָה（tov asher-yokhal v'shatah）：人莫强如吃喝——享受当下的恩赐
  • שָׂמַח（samach）：喜乐——欢乐
  • יָרֵא（yare'）：敬畏——敬畏神
  • 享受当下的恩赐：这是神的恩赐
  • 神的作为永存：神的计划不会改变
  • 敬畏的心：享受人生的同时要敬畏神
  • 古代近东背景：埃及智慧文学也强调享受当下的恩赐
  • 基督论预表：基督说"我来了，是要叫羊得生命，并且得的更丰盛"（约10:10）

3:14-15 我知道神一切所做的都必永存；无所增添，无所减少。神这样行，是要人在他面前存敬畏的心。现今的事早先就有了，将来的事早已也有了，并且神使已过的事重新再来。

  • עוֹלָם（olam）：永存——永恒
  • יָרֵא（yare'）：敬畏——敬畏神
  • בָּקַשׁ（baqash）：使...重新再来——寻求、追回
  • 神的作为永存：过去、现在、未来都在神的手中
  • 神使已过的事重新再来：历史在神的手中循环
  • 古代近东背景：古代近东也有循环历史观，但传道书强调神掌管历史
  • 基督论预表：基督是阿拉法，是俄梅戛，是首先的，是末后的（启1:8）

三、神的审判（3:16-22）

3:16-17 我又见日光之下，在审判之处有奸恶，在公义之处也有奸恶。我心里说：神必审判义人和恶人，因为在那里，各样事务，一切工作，都有定时。

  • מִשְׁפָּט（mishpat）：审判——公义的审判
  • רֶשַׁע（resha）：奸恶——邪恶
  • צֶדֶק（tsedeq）：公义——正义
  • אֱלֹהִים יִשְׁפֹּט（elohim yishpot）：神必审判——这是传道书的盼望
  • 日光之下的不公：审判之处有奸恶，公义之处也有奸恶
  • 古代近东背景：古代近东智慧文学也探讨公义与邪恶的问题
  • 基督论预表：神必审判——基督是审判者（徒17:31）

3:18-19 我心里说：这乃为世人的缘故，是神要试验他们，使他们觉得自己不过像兽一样。因为世人遭遇的，兽也遭遇，所遭遇的都是一样：这个怎样死，那个也怎样死，气息都是一样。

  • בֶּן־אָדָם（ben-adam）：世人——人子、人类
  • בְּהֵמָה（behemah）：兽——牲畜
  • נְשָׁמָה（nishmah）：气息——呼吸
  • 人与兽的比较：在日光之下，人和兽的结局一样
  • 古代近东背景：吉尔伽美什史诗也探讨人与兽的区别

3:20-21 都归一处，都是出于尘土，也都归于尘土。谁知道人的灵是往上升，兽的魂是下入地呢？

  • עָפָר（aphar）：尘土——泥土
  • רוּחַ（ruach）：灵——气息、灵
  • נֶפֶשׁ（nefesh）：魂——生命
  • 人与兽的结局：都归尘土
  • 传道书的疑问：人和兽有什么区别？
  • 答案：人有永恒的渴望（3:11），兽没有
  • 基督论预表：基督是人子，他胜过死亡，使我们得永生（约11:25-26）

3:22 故此，我见人莫强如在他经营的事上喜乐，因为这是他的份。他身后的事，谁能使他回来得见呢？

  • חֶלְקוֹ（chelqo）：他的份——分配、份额
  • שָׂמַח（samach）：喜乐——欢乐
  • מַעֲשֶׂה（ma'aseh）：经营——工作、事业
  • 享受当下的工作是神的恩赐
  • 古代近东背景：埃及智慧文学也强调享受当下的恩赐
  • 基督论预表：基督说"我来了，是要叫羊得生命，并且得的更丰盛"（约10:10）

四、日光之下各样事务（4:1-16）

4:1 我又转念，见日光之下所行的一切欺压。看哪，受欺压的流泪，且无人安慰。

  • עֹשֶׁק（osheq）：欺压——压迫、剥削
  • דִּמְעָה（dim'ah）：流泪——眼泪
  • נָחַם（nacham）：安慰——安慰、抚慰
  • 日光之下的不公：受欺压的流泪，且无人安慰
  • 古代近东背景：埃及智慧文学《伊普味陈辞》也描述了社会不公和压迫
  • 基督论预表：基督是安慰者（林后1:3-4），他安慰一切受压迫的人

4:2 因此，我赞叹那早已死的死人，胜过那还活着的活人。

  • שָׁבַח（shavach）：赞叹——称赞
  • 传道者的感叹：死人比活人更有福——因为死人不再受欺压
  • 这是传道书的悲观论调，但不是最终答案

4:3 并且我以为那未曾生的，就是未见过日光之下恶事的，比这两种人更强。

  • 未曾生的比活人更强：这是极度悲观的感叹
  • 日光之下恶事：人生中的苦难和不公

4:4 我又见人为一切的劳碌和各样灵巧的工作就被邻舍嫉妒。这也是虚空，也是捕风。

  • קִנְאָה（qin'ah）：嫉妒——妒忌
  • רְעוּת רוּחַ（re'ut ruach）：捕风——追求风
  • 人为劳碌被嫉妒：这是虚空
  • 古代近东背景：埃及智慧文学也警告嫉妒的危险

4:5 愚昧人抱着手，吃自己的肉。

  • חָבַק（chavaq）：抱着——交叉
  • בָּשָׂר（basar）：肉——身体
  • 愚昧人抱着手：懒惰的人
  • 吃自己的肉：消耗自己的生命

4:6 满了一把，得享安静，强如满了两把，劳碌捕风。

  • כַּף（kaf）：手——手掌
  • נָחַת（nachat）：安静——平安
  • 满了一把得享安静：知足胜过贪心
  • 基督论预表：基督说"凡劳苦担重担的人，可以到我这里来，我就使你们得安息"（太11:28）

4:7-8 我又转念，见日光之下有一件虚空的事：有人孤单无二，无子无兄，竟劳碌不息，眼目也不以钱财为足。他说：我劳劳碌碌，刻苦自己，不享福乐，到底为谁呢？这也是虚空，是极重的劳苦。

  • אֶחָד（echad）：孤单——一个
  • בֵּן（ben）：儿子——孩子
  • אָח（ach）：兄弟——弟兄
  • עָמָל（amal）：劳苦——辛劳
  • 孤单之人的悲哀：劳碌却不享受，不知为谁劳碌
  • 古代近东背景：埃及智慧文学也警告不要只为财富劳碌
  • 基督论预表：基督说"人若赚得全世界，赔上自己的生命，有什么益处呢？"（太16:26）

4:9-12 两个人总比一个人好，因为二人劳碌同得美好的果效。若是跌倒，这人可以扶起他的同伴。若是孤身跌倒，没有别人扶起他来，这人就有祸了。再者，二人同睡，就都暖和；一人独睡，怎能暖和呢？有人攻胜孤身一人，若有二人便能敌挡他；三股合成的绳子不容易折断。

  • טוֹבִים הַשְׁנַיִם מִן־הָאֶחָד（tovim hashnayim min-ha'echad）：两个人总比一个人好——团契的价值
  • נָפַל（nafal）：跌倒——倒下
  • קוּם（qum）：扶起——站起来
  • חַבְלִים（chavlim）：绳子——绳索
  • 两个人总比一个人好：团契的价值
  • 三股合成的绳子：团契的力量
  • 古代近东背景：埃及智慧文学也强调友谊的价值
  • 基督论预表：基督差遣门徒两个两个地出去（可6:7），教会是基督的身体，信徒彼此配搭

4:13-14 贫穷而有智慧的少年人，胜过年老不肯纳谏的愚昧王。因为有人能监牢中出来作王；那在国中生的，却变为贫穷。

  • מֶלֶךְ（melek）：王——君王
  • זָקֵן（zaqen）：年老——年长
  • אֱוִיל（evil）：愚昧——愚笨
  • עָנִי（ani）：贫穷——贫乏
  • 智慧胜过权力：贫穷而有智慧的少年人胜过愚昧的王
  • 古代近东背景：约瑟从监牢出来作宰相（创41章）
  • 基督论预表：基督从死里复活，成为万有的主（腓2:9-11）

4:15-16 我见一切活的人，就是日光之下行走的人，都随从那少年人，就是起来代替老王的。他所治理的众人，就是他的百姓，多得无数。后来的人尚且不喜悦他。这真是虚空，也是捕风。

  • חַי（chay）：活的——活着
  • יָלַךְ（yalakh）：行走——走
  • עָמַד（amad）：起来——站立
  • 众人随从少年人：但这也是虚空
  • 即使作王，也不能满足人心
  • 基督论预表：基督是真正的王，他的国度存到永远（但7:14）

神学要点总结

1. 凡事都有定期：神掌管时间的每一个方面。 2. 永世安置在人心：人有永恒的渴望，唯有神能满足。 3. 神必审判：日光之下的不公需要神的审判来纠正。 4. 团契的价值：两个人总比一个人好。

对当年读者的意义

  • 对以色列人：他们需要认识到神掌管时间，享受当下是神的恩赐。
  • 对所有读者：我们是否有永恒的渴望？唯有神能满足。

深化反思与应用

一、关键词解析

  • זְמָן (zman)：定期、时间——神掌管时间
  • עֵת (et)：时候、时机——每件事都有神指定的时间
  • עוֹלָם (olam)：永世、永恒——人心中有永恒的渴望
  • יִתְרוֹן (yitron)：益处、利润——传道书的关键问题

二、古代近东背景

  • 吉尔伽美什史诗：英雄追求永生却失败
  • 埃及智慧文学：时间的循环和人生的虚空
  • 传道书与这些文学有相似的主题，但指向以色列的神

三、基督论预表

  • 3:11"永世安置在人心"——人有永恒的渴望，唯有基督能满足（约4:13-14）
  • 基督道成肉身也有"时候"（约7:6）
  • 神必审判——基督是审判者（徒17:31）

四、默想经文

"神造万物，各按其时成为美好，又将永生安置在世人心里。"（传3:11）

默想要点：人有永恒的渴望，唯有神能满足。
