以斯拉记解经（三）：以斯拉归回

经文范围

经文范围：以斯拉记 7:1-8:36

本段记录了以斯拉——一位祭司和文士——带领第二批犹大人归回耶路撒冷的经过。以斯拉的使命不仅是带领归回者踏上旅程，更重要的是教导神的律法、改革以色列人的信仰生活。这段经文突出了以斯拉"定志考究遵行"神律法的心志，以及神在归回旅程中的信实保护。

核心神学命题

1. 神的话语是信仰和生活的最高权威——以斯拉"定志考究遵行"律法，表明信仰的根基在于神的话语。 2. 神的护理在每一个细节中工作——从王的诏书到平安的旅程，神的手始终引导。 3. 信心的祷告胜过人的保护——以斯拉选择用祷告而不是兵马来寻求神的保护。

上文回顾

圣殿在大流士王第六年（公元前516年）建成。此后约五十八年，以斯拉记没有记录任何事件。这段时间里，犹大社群在圣殿的敬拜中逐渐成长，但也面临信仰松懈的挑战。以斯拉的归回发生在亚达薛西王第七年（公元前458年），比尼希米的归回早约十三年。以斯拉的到来标志着一个新的阶段——从建筑的恢复转向信仰的重建。

主题概述

本段的核心主题是神话语的权威与神的护理。以斯拉不是一般的归回者，他是一个"通达耶和华以色列神所赐摩西的律法书"的文士（7:6）。他的使命是将神的律法教导给归回的以色列人，使他们的信仰生活有根有基。同时，本段也展示了神在每一个细节中的护理——从王的允准到平安的旅程。

逐句解经

一、以斯拉的背景（7:1-10）

"这事以后，波斯王亚达薛西年间，有个以斯拉，他是西莱亚的儿子。"（7:1）

"以斯拉"（希伯来文 Ezra / עֶזְרָא）：名字意为"帮助"或"神帮助"。这个名字本身就是他使命的写照——他是神赐给以色列人的帮助。以斯拉是祭司家族的后裔，他的家谱一直追溯到亚伦（7:1-5），表明他有合法的祭司身份。

"亚达薛西"（希伯来文 Artachshashta / אַרְתַּחְשַׁשְׂתָּא）：亚达薛西一世（Artaxerxes I，公元前465-424年在位）。以斯拉在亚达薛西第七年（公元前458年）归回。

"以斯拉是敏捷的文士，通达耶和华以色列神所赐摩西的律法书。"（7:6）

"敏捷的文士"（希伯来文 sofer mahir / סֹפֵר מָהִיר）：希伯来文"sofer"（文士）不仅是抄写员，更是律法的教师和诠释者。"mahir"意为"敏捷的、熟练的"。以斯拉不是一般的文士，而是一个精通律法的学者。

历史背景：文士在被掳后的犹太社会中扮演了极其重要的角色。在没有圣殿的日子里，会堂逐渐成为犹太人聚集敬拜的地方，而文士则是会堂中教导律法的核心人物。以斯拉是这个传统的奠基人之一。

基督论预表：以斯拉作为文士和祭司，预表基督——那真正的教师和祭司。基督在会堂中教导神的话语（路4:16-21），祂也是永远的大祭司（来7:24-25）。以斯拉的三重使命——考究、遵行、教训——在基督身上得到了完美的实现。

"以斯拉定志考究遵行耶和华的律法，又将律例典章教训以色列人。"（7:10）

"定志"（希伯来文 kuwn / כּוּן）：原文意为"坚定、确立、预备"。以斯拉不是偶然成为文士的，而是坚定地立志要考究、遵行、教导神的律法。这个"定志"表明他的使命是出于深思熟虑的委身。

"考究"（希伯来文 darash / דָּרַשׁ）：原文意为"寻求、研究、探求"。这个动词在旧约中常用于描述对神的话语的深入研究（参代下34:15，约西亚王在圣殿中发现律法书后"求问耶和华"）。以斯拉不是一个表面的读者，而是一个深入的研究者。

"遵行"（希伯来文 la'asot / לַעֲשׂוֹ�）：知识若没有实践就是空洞的。以斯拉不仅研究律法，还身体力行。这提醒我们：真正的信仰不是只停留在头脑中，而是要付诸行动。

"教训"（希伯来文 lelammed / לְלַמֵּד）：以斯拉的使命不仅是个人的灵修，更是群体的教导。他要将律法传给以色列人，使整个信仰群体有根有基。

讲道应用：以斯拉的三重使命——考究、遵行、教训——是每一个信徒的榜样。我们是否"定志"研究神的话语？我们是否在生活中"遵行"？我们是否愿意"教训"他人？

二、亚达薛西王的诏书（7:11-26）

"亚达薛西王达于祭司以斯拉，通达天上神律法大德的文士。"（7:12）

"天上神的律法"（希伯来文 Torat Elohe hashshamayim）：这个短语在诏书中出现了四次（7:12, 14, 21, 23）。外邦君王承认以色列的神是"天上的神"，并且承认祂的律法具有权威。这表明神的名在外邦中被尊崇。

历史背景：亚达薛西的诏书是一份正式的官方文件，授权以斯拉带领犹大人归回，并给予物质支持和政治保护。诏书的内容包括：允许犹大人归回（7:13），拨款支持圣殿（7:15-20），免除祭司的税赋（7:24），授权以斯拉设立审判官（7:25-26）。

"王和七个谋士降旨……准他们与你同去。"（7:14-15）

"七个谋士"：波斯王有七个谋士（参斯1:14），这是波斯帝国的制度。七在圣经中是完全的数字，七个谋士代表了帝国的最高决策层。

基督论预表：外邦君王的支持预表福音要传到万邦。正如亚达薛西支持以斯拉的使命，神也要使用外邦的权势来推动祂国度的扩展。

"你掌管河西一切的金银、宝物和牲畜。"（7:21-23）

"河西"（希伯来文 abar naharah / עֲבַר נַהֲרָה）：指幼发拉底河以西的地区，包括犹大、撒玛利亚、叙利亚等地。以斯拉被授权管理这个地区的资源，表明他的权柄不仅是宗教的，也是政治的。

三、以斯拉的感恩（7:27-28）

"耶和华我们列祖的神是应当称颂的！因他使王起这心意修饰耶路撒冷耶和华的殿。"（7:27）

"使王起这心意"（希伯来文 nathan lemelekh / נָתַן לְמֶלֶךְ）：以斯拉将王的慷慨归因于神的感动。他不是说"王真好"，而是说"神感动了王"。这表明以斯拉有属灵的洞察力——他能看见神在人背后的作为。

讲道应用：我们是否能看见神在我们生活中的作为？当好事发生时，我们是感谢人还是感谢神？以斯拉的感恩提醒我们：一切美善都来自神的感动和供应。

四、归回者的名单（8:1-14）

"以下是同以斯拉从巴比伦上来的人的家谱。"（8:1）

详细记录：以斯拉详细记录了归回者的家谱和人数。这表明每个人在神眼中都是重要的。名单中的每一个名字都是一个家庭、一段故事、一个信心的决定。

历史背景：在古代近东，家谱是身份认同的重要根据。归回者需要确认自己的以色列血统，才能在信仰群体中有份。详细的名单也确保了归回者的真实性。

五、在亚哈瓦河边聚集（8:15-20）

"我招聚这些人在流入亚哈瓦的河边。"（8:15）

"亚哈瓦"（希伯来文 Ahava / אֲהַוָא）：这条河的确切位置不详，可能在巴比伦附近。以斯拉在河边聚集归回者，可能是为了检查人员和物资，准备出发。

"我见百姓和祭司中没有利未人。"（8:15上）

没有利未人：以斯拉发现归回者中没有利未人。利未人是圣殿事奉的重要人员，没有他们，圣殿的敬拜就无法正常进行。以斯拉派人去召利未人来（8:16-20），表明他认识到建殿需要每个职分的人。

讲道应用：神的工作需要每个人参与。以斯拉不满足于"够了"，而是追求"完全"。今天教会也需要每个恩赐的人参与事奉。

六、禁食祷告（8:21-23）

"那时，我在亚哈瓦河边宣告禁食，为要在我们神面前克苦己心，求他使我们和妇人孩子，并一切所有的，都得平坦的道路。"（8:21）

"禁食"（希伯来文 tsom / צוֹם）：禁食是谦卑和依靠神的表现。以斯拉带领归回者在神面前禁食，表明他们完全依靠神。禁食不是表演，而是在神面前"克苦己心"。

"我求王拨派兵马护送我们，因为曾对王说：'我们神施恩的手必帮助一切寻求他的；但他的能力和忿怒必攻击一切离弃他的。'"（8:22）

信心的考验：以斯拉曾向王夸耀神的保护，所以不敢再向王求兵马。这是一个信心的考验——他是否真的相信神会保护他们？他选择了信心的道路，用祷告代替兵马。

讲道应用：信心不是口头上的宣告，而是行动上的选择。以斯拉曾对王说"神会保护"，如今他必须在行动上证明自己的信心。你是否愿意在行动上证明你的信心？

"这样，我们禁食祈求我们的神，他就应允了我们。"（8:23）

"应允了我们"（希伯来文 na'aneh lanu / נַעֲנֶה לָנוּ）：神应允了他们的祷告。这表明神信实地保护了归回者。信心的祷告不是冒险，而是信靠那位信实的神。

七、归回旅程（8:24-36）

"我分派祭司长十二人……将金银和器皿都称了交给他们。"（8:24-30）

忠心的管理：以斯拉仔细地分派金银和器皿的管理责任。他将金银交给十二个祭司长和其他管理人员，并要求他们在耶路撒冷殿中称重核对。这表明对神的财物需要忠心的管理。

讲道应用：对神所托付的财物，我们需要忠心的管理。以斯拉对金银的管理提醒我们：信徒在财物上的忠心是信仰生活的重要部分。

"我们到了耶路撒冷，在那里住了三天。第四日，在我们神的殿里把金银和器皿都称了。"（8:32-33）

平安到达：归回者平安到达耶路撒冷。从巴比伦到耶路撒冷的旅程约四个月（参7:9），途中可能有强盗和野兽的威胁。但神信实地保护了他们。

"称了"：以斯拉将金银和器皿在圣殿中称重核对，确保数量无误。这表明他的管理是透明的、负责任的。

"他们将王的谕旨交给王所派的总督与河西的省长，他们就帮助百姓，又供给神殿里所需用的。"（8:36）

王的谕旨：总督和省长执行了王的谕旨，帮助归回者。这再次表明神在外邦权势中的工作。

神学要点总结

一、神的话语的权威

以斯拉"定志考究遵行"神的律法，表明神的话语是信仰和生活的最高权威。信仰不是感觉，而是建立在神话语的根基上。

二、神的护理

从亚达薛西王的诏书到平安的旅程，神在每一个细节中工作。祂的护理是确实的、具体的、持续的。

三、信心的祷告

以斯拉选择用祷告而不是兵马来寻求保护。这表明真正的信心在于依靠神，而不是依靠人的力量。

四、忠心的管理

以斯拉仔细管理金银和器皿。对神的财物需要忠心的管理，这是信仰生活的重要部分。

对当年读者的意义

对归回者而言，以斯拉的到来带来了新的希望。他不仅带来了更多的归回者和物资，更重要的是带来了神的话语。在被掳归回后的重建时期，以色列人需要有根有基的教导来重建他们的信仰生活。

对后来的犹太社群而言，以斯拉是文士传统的奠基人。他的"考究、遵行、教训"三重使命成为犹太教育的核心。后来的拉比传统将以斯拉视为律法的伟大教师，与摩西并列。

深化反思与应用

一、定志考究遵行

以斯拉的三重使命是每一个信徒的榜样。我们是否"定志"研究神的话语？研究而不遵行是空洞的，遵行而不教导是自私的。完整的信仰生活需要研究、实践和传递。

二、信心的祷告

以斯拉选择用祷告而不是兵马来寻求保护。这提醒我们：真正的信心不是等到万全之策才行动，而是在没有"安全网"的时候仍然信靠神。你是否愿意放下人的保护，完全依靠神？

三、忠心的管理

以斯拉对金银的管理是透明的、负责任的。这提醒我们：在教会和家庭中，忠心的管理是信仰的见证。

四、感恩的态度

以斯拉将王的慷慨归因于神的感动。这提醒我们：一切美善都来自神，我们的感恩应当归给神。

下文预告

以斯拉记接下来将记录以斯拉回到耶路撒冷后发现的问题——以色列人与外邦人通婚，违背了神的律法。以斯拉以极大的悲痛和祷告来面对这个问题，最终带领百姓悔改和改革。
