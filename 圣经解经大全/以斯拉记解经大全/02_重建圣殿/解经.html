以斯拉记解经（二）：重建圣殿

经文范围

经文范围：以斯拉记 3:1-6:22

本段记录了圣殿重建的完整过程：从筑坛献祭、立定根基，到敌人的拦阻导致停工，再到先知哈该和撒迦利亚的鼓励、大流士王的诏书支持，最终圣殿建成并守逾越节。这段经文是被掳归回时期最核心的叙事，揭示了圣殿重建的神学意义——不仅是建筑的恢复，更是敬拜的恢复、约的关系的恢复。

核心神学命题

1. 恢复敬拜是重建的第一优先——归回者首先筑坛献祭，表明与神的关系比任何建筑都重要。 2. 神的工程必然经历属灵争战——敌人的拦阻是必然的，但不能拦阻神旨意的成就。 3. 神使用先知和外邦君王来完成祂的工作——哈该、撒迦利亚的鼓励和大流士王的诏书都是神护理的方式。

上文回顾

第一批归回者在所罗巴伯和耶书亚的带领下，从巴比伦回到耶路撒冷。他们带来了五千四百件圣殿器皿和甘心奉献的金银。他们面对的是一个荒凉的故土——耶路撒冷城墙被毁，圣殿成为废墟，四周有敌人的威胁。然而，他们带着神的应许和古列王的诏书，开始了重建的工作。

主题概述

本段的核心主题是圣殿重建的艰难与神的信实。重建的过程不是一帆风顺的，而是充满了争战、灰心和拦阻。但神通过先知的鼓励和外邦君王的诏书，最终使圣殿建成。这个过程教导我们：神的工作必然经历困难，但神的旨意终将成就。

逐句解经

一、筑坛献祭（3:1-7）

"到了七月，以色列人住在各城，那时他们如同一人，聚集在耶路撒冷。"（3:1）

"如同一人"（希伯来文 k'ish echad）：这个表达强调了归回者的合一。他们虽然来自不同的家族和城邑，但在信仰上是合一的。这种合一是神子民重建的基础。

"到了七月"：七月是以色列宗教历的重要月份，住棚节就在这个月（利23:34）。归回者选择在七月聚集，表明他们重视神所设立的节期。

历史背景：归回者在被掳七十年间无法在圣殿中敬拜。如今他们回到耶路撒冷，第一件事就是恢复敬拜。这表明他们认识到：敬拜是信仰生活的核心，没有敬拜就没有真正的恢复。

"约萨答的儿子耶书亚和他的弟兄众祭司，并撒拉铁的儿子所罗巴伯与他的弟兄，都起来建筑以色列神的坛，要照神人摩西律法书上所写的，在坛上献燔祭。"（3:2）

"建筑以色列神的坛"：在建殿之前先筑坛。坛是献祭的地方，献祭是人与神和好的途径。归回者首先恢复的是与神的关系，而不是自己的家园。这表明他们的优先次序是正确的。

"照神人摩西律法书上所写的"：他们的敬拜不是随意的，而是严格按照律法的要求。这表明归回者重视神的话语，以律法为敬拜的标准。在被掳期间，律法书可能被忽视或遗忘，如今他们重新回到神的话语中。

基督论预表：坛上的燔祭预表基督的十字架。燔祭是无瑕疵的祭牲被献上，为罪人赎罪。基督是那"无瑕疵、无玷污的羔羊"（彼前1:19），祂在十字架上将自己献上，成为永远的赎罪祭。归回者在坛上献祭，指向基督在十字架上完成的救赎。

"他们在原有的根基上筑坛，因惧怕邻国的民。"（3:3）

"原有的根基"：坛建在所罗门圣殿原有的根基上。这表明归回者不是另起炉灶，而是在先人的基础上恢复。信仰的传承需要连接过去和现在。

"因惧怕邻国的民"：归回者在敌人环伺的环境中敬拜。他们的敬拜不是在安全的环境中，而是在危险中。这表明真正的敬拜不是等到环境完美才开始，而是在困难中仍然坚持。

"又照律法书上所写的守住棚节。"（3:4）

守住棚节：住棚节（希伯来文 Sukkot / סֻּכּוֹת）是纪念以色列人在旷野漂流四十年的节期（利23:42-43）。归回者守住棚节，表明他们认识到自己的处境如同在旷野——离开了被掳之地，但尚未完全进入安息。住棚节也是欢乐的节期，感谢神在旷野中的供应和保护。

历史背景：经文特别指出"从嫩的儿子约书亚的时候直到这日，以色列人没有这样行过"（3:4下）。这表明从约书亚时代到归回时期，以色列人已经有数百年没有认真守住棚节了。被掳之后的恢复，是对古老信仰传统的重新拾起。

二、立定殿的根基（3:8-13）

"百姓到了耶路撒冷神殿的地方。第二年二月，撒拉铁的儿子所罗巴伯，约萨答的儿子耶书亚和其余的弟兄，就是祭司、利未人，并一切被掳归回耶路撒冷的人，都兴工建造。"（3:8）

"第二年二月"：归回后第二年的二月，即公元前536年。所罗门建造圣殿也是在二月开工（王上6:1）。这暗示归回者效法所罗门的榜样。

"兴工建造"（希伯来文 hechelim / הֵחֵל）：原文意为"开始、着手"。这是一个充满希望的动词——在废墟之上，神的子民开始重建。建造不仅是物理上的工程，更是信心的行动。

基督论预表：归回者在废墟上"兴工建造"，预表基督在罪的废墟上建立祂的教会。基督说："我要把我的教会建造在这磐石上"（太16:18）。教会的根基不是人的努力，而是基督自己。

"他们赞美耶和华的时候，众祭司都吹号。"（3:10）

"吹号"（希伯来文 chatstsotsrah / חֲצֹצְרָה）：银号是祭司在圣殿中使用的乐器（民10:2）。吹号不仅是音乐，更是宣告——宣告神的国度、宣告神的同在。归回者在立根基的时候吹号，表明这是一个神圣的时刻。

"他们彼此唱和，赞美称谢耶和华说：他本为善，他向以色列人永发慈爱。"（3:11）

"慈爱"（希伯来文 chesed / חֶסֶד）：这是旧约中最重要的神学词汇之一，指神与祂子民之间立约的爱、忠实的爱、不离不弃的爱。归回者赞美神的chesed，因为他们经历了神的信实——七十年的被掳之后，神仍然记念祂的约，带领他们归回。

讲道应用：在困难中赞美神，不是因为环境好，而是因为神好。归回者在废墟上赞美神的慈爱，这提醒我们：赞美不是等到问题解决才开始，而是在困难中仍然宣告神的美善。

"然而有许多祭司、利未人、族长，就是见过旧殿的老年人，现在亲眼看见立这殿的根基，便大声哭号。"（3:12）

"见过旧殿的老年人"：这些老年人见过所罗门建造的辉煌圣殿。如今的新殿根基与旧殿相比，显得微不足道。他们的情感是复杂的——既有对恢复的感恩，又有对失去的哀伤。

历史背景：所罗门的圣殿是古代近东最辉煌的建筑之一。它用了七年的建造时间，动用了数万工人，使用了大量黄金、香柏木和宝石。归回者所建的新殿，无论在规模还是华丽程度上，都无法与旧殿相比。

"甚至百姓不能分辨欢呼的声音和哭号的声音。"（3:13）

"不能分辨"：欢呼与哭号交织在一起，无法分辨。这深刻地表达了信仰生活中的一种张力——喜乐与哀伤并存，盼望与遗憾交织。信仰不是简单的"一切都好"，而是在复杂的情感中仍然信靠神。

讲道应用：你是否也有过这样的经历——在感恩中带着忧伤，在盼望中带着遗憾？这是真实的信仰生活。神接纳我们复杂的情感，祂不需要我们假装一切都好。

三、敌人的拦阻（4:1-24）

"犹大和便雅悯的敌人听说被掳归回的人为耶和华以色列的神建造殿宇，就去见所罗巴伯和以色列的族长，对他们说：'请容我们与你们一同建造。'"（4:1-2）

"敌人"（希伯来文 tsar / צַר）：这些"敌人"是亚述帝国迁入撒玛利亚地区的外邦民族（王下17:24-41）。他们混合了外邦宗教和耶和华信仰，形成了后来的撒玛利亚人。他们的"帮助"实际上是试探——如果他们参与建殿，就会使圣殿的信仰变得混杂。

基督论预表：敌人的"帮助"预表教会历史中异端和世俗势力的渗透。他们表面上是"帮助"，实际上是要稀释信仰的纯正。基督警告说："假先知……外面披着羊皮，里面却是残暴的狼"（太7:15）。

"我们建造神的殿与你们无干，我们自己为耶和华以色列的神协力建造。"（4:3）

"与你们无干"：所罗巴伯和耶书亚拒绝了敌人的帮助。这不是种族歧视，而是信仰的分别——圣殿的敬拜必须保持纯正，不能与混合的宗教信仰掺杂。

讲道应用：在信仰上保持纯正有时意味着拒绝"帮助"。当世俗的价值观试图渗入教会时，我们需要有分辨的智慧和拒绝的勇气。

"那地的民，就在犹大人建造的时候，使他们的手发软，扰乱他们。"（4:4）

"使他们的手发软"（希伯来文 merappim / מְרַפִּים）：原文意为"使松懈、使灰心"。敌人通过恐吓和干扰来打击归回者的士气。属灵争战往往不是正面冲突，而是通过灰心和恐惧来削弱神子民的力量。

历史背景：敌人的拦阻持续了很长时间——从古列王到大流士王，大约十六年。在这期间，建殿工程停滞不前。百姓转向建造自己的房屋（该1:4），圣殿的废墟被遗忘了。

"亚达薛西年间……上本控告犹大人和耶路撒冷的居民。"（4:6-7）

历史背景：这里的"亚达薛西"可能不是后来允许以斯拉归回的亚达薛西一世，而是一位更早的波斯王（可能是假亚达薛西，历史上有争议）。敌人用政治性的语言控告犹大人，诬告他们是叛乱的威胁。

"那时河西的总督达乃和示他波斯乃，并他们的同党，上本奏告大流士王。"（5:3-5）

"达乃"（希伯来文 Tatnai）：河西总督达乃调查建殿的权柄。他的调查反而成为了建殿的支持——当大流士王查找古列的诏书后，不仅允许建殿继续，还下令从国库拨款支持。

神的护理：敌人的控告反而成为了建殿的助力。这表明神可以将拦阻变为祝福，将敌人的计谋变为祂旨意的成就。

四、先知的鼓励（5:1-2）

"那时，先知哈该和易多的孙子撒迦利亚，奉以色列神的名，向犹大和耶路撒冷的犹大人说劝勉的话。"（5:1）

"先知哈该"（希伯来文 Chaggai / חַגַּי）：哈该的名字意为"节期的"或"我的节期"。他的信息简短有力，责备百姓只顾自己的房屋，却让圣殿荒凉（该1:4）。他的使命是唤醒百姓的使命感。

"撒迦利亚"（希伯来文 Zekharyah / זְכַרְיָה）：名字意为"耶和华记念"。撒迦利亚的信息更加丰富，包含了一系列异象和对未来的盼望。他宣告神将复兴耶路撒冷，弥赛亚将要来到（亚9:9）。

基督论预表：撒迦利亚预言了一位谦卑的君王——"锡安的民哪，应当大大喜乐；耶路撒冷的民哪，应当欢呼。看哪，你的王来到你这里！他是公义的，并且施行拯救，谦谦和和地骑着驴"（亚9:9）。这预言在基督骑驴进耶路撒冷时应验（太21:1-11）。

"于是撒拉铁的儿子所罗巴伯和约萨答的儿子耶书亚，都起来动手建造耶路撒冷神的殿，有神的先知在那里帮助他们。"（5:2）

"有神的先知在那里帮助他们"：当百姓灰心、工程停滞的时候，神通过先知来鼓励他们。这表明神不会忘记祂的子民，祂会在适当的时候兴起先知来帮助他们。先知的角色不是代替百姓工作，而是鼓励他们起来工作。

五、大流士王的诏书（5:3-6:12）

"大流士王降旨，要寻察典籍库……在亚马他城的宫内寻得一卷。"（6:1-2）

历史背景：大流士一世（Darius I，公元前522-486年在位）是波斯帝国最伟大的君王之一。他下令在帝国的档案库中查找古列的诏书，结果找到了。这表明神在帝国的档案系统中工作，保存了那份关键的诏书。

"河西的总督达乃和示他波斯乃……当远离他们，不要拦阻这神殿的工作。"（6:6-7）

"不要拦阻"：大流士王的命令不仅允许建殿继续，还禁止任何人拦阻。敌人再也不敢干扰了。这表明神可以在外邦君王的心中工作，使祂子民的工作得以顺利进行。

"我又降旨，吩咐你们向犹大人的长老为建造神的殿当怎样行……从河西的款项中，急速拨取贡银作他们的经费。"（6:8）

从国库拨款：大流士王不仅允许建殿，还下令从国库中拨款支持。敌人原本想用金钱来诱惑王（3:9），但神反转了局面——王不仅不收钱，反而拨款支持。这再次表明神的反转大能。

基督论预表：外邦君王的拨款支持圣殿的建造，预表福音要传到万邦，外邦人也要参与神子民的建造。保罗说："你们外邦人……与圣徒同国，是神家里的人了"（弗2:19）。

六、圣殿建成（6:13-22）

"犹大长老因先知哈该和易多的孙子撒迦利亚所说劝勉的话，就建造这殿，凡事亨通。他们遵着以色列神的命令和波斯王古列、大流士、亚达薛西的旨意，建造完毕。"（6:14）

"凡事亨通"（希伯来文 matsliach / מַצְלִיחַ）：原文意为"顺利、成功"。圣殿的建成是神信实的见证。从停工到完成，神的手一直在引导。

"遵着以色列神的命令和……旨意"：经文同时提到神的命令和波斯王的旨意，表明神的旨意和人的制度并不冲突。神可以在人的制度中工作，使祂的旨意成就。

"以色列的祭司和利未人，并其余被掳归回的人都欢欢喜喜地行奉献神殿的礼。"（6:16）

"行奉献的礼"（希伯来文 chanak / חָנַךְ）：原文意为"开始使用、奉献"。这个动词在旧约中出现次数不多，但每次都与重要的建筑或工程相关。圣殿的奉献标志着一个新时代的开始。

历史背景：奉献礼献上了一百只公牛、二百只公绵羊、四百只母羊，又照以色列支派的数目献十二只公山羊作赎罪祭。这个规模表明归回者对圣殿的重视。

"正月十四日，被掳归回的人守逾越节。"（6:19）

守逾越节：逾越节（希伯来文 Pesach / פֶּסַח）是纪念以色列人从埃及得拯救的节期（出12章）。归回者守逾越节，表明他们认识到自己的归回如同第二次出埃及——从巴比伦的捆绑中得释放。

基督论预表：逾越节的羔羊预表基督。保罗说："我们逾越节的羔羊基督已经被杀献祭了"（林前5:7）。归回者守逾越节，指向基督在十字架上完成的终极拯救。

"耶和华使亚述王的心归向他们，坚固他们的手，作以色列殿的工程。"（6:22）

"耶和华使……心归向"：经文再次强调是神激动了君王的心。从古列到大流士，每一位波斯王的心都在神的掌管中。神是历史的主。

神学要点总结

一、恢复敬拜是第一优先

归回者首先筑坛献祭，然后才建造圣殿。这表明敬拜是信仰生活的核心，没有敬拜就没有真正的恢复。

二、属灵争战是必然的

敌人从一开始就试图拦阻建殿的工作。属灵争战是神子民必须面对的现实，但争战的结果已经由神决定。

三、神使用先知来鼓励灰心的人

当百姓灰心、工程停滞时，神兴起先知哈该和撒迦利亚来鼓励他们。神的话语是灰心时的安慰和力量。

四、神的反转大能

敌人想用金钱诱惑王，神却使王拨款支持。敌人想拦阻建殿，神却使建殿完成。神可以将一切拦阻变为祝福。

对当年读者的意义

对归回者而言，圣殿的建成是一个巨大的喜乐。他们经历了十六年的停工、敌人的拦阻、信心的软弱，但最终看到了圣殿的完成。这个过程教导他们：神的工作可能经历延迟，但绝不会被拦阻。

对后来的以色列人而言，圣殿的重建是信仰复兴的标志。圣殿不仅是建筑，更是神与人同在的象征。圣殿的恢复意味着神仍然与祂的子民同在，祂的约仍然有效。

深化反思与应用

一、在困难中坚持

建殿的过程充满了困难——敌人的拦阻、百姓的灰心、工程的停滞。但神通过先知和外邦君王的工作，最终使圣殿建成。这提醒我们：在困难中不要放弃，因为神的工作必会完成。

二、保持信仰的纯正

归回者拒绝了敌人的"帮助"，保持了信仰的纯正。这提醒我们：在信仰上不能妥协，即使妥协看起来能带来和平。

三、先知的重要性

当百姓灰心时，神兴起先知来鼓励他们。今天，神仍然通过祂的话语来鼓励我们。我们是否重视神的话语？

四、在废墟上赞美

归回者在圣殿的废墟上赞美神的慈爱。这提醒我们：赞美不是等到环境完美才开始，而是在困难中仍然宣告神的美善。
