07 赎罪日 (利未记 16:1-34)

经文范围

利未记 16:1-34

经文背景

赎罪日（Yom Kippur）是以色列历中最神圣的一天，也是每年唯一一次大祭司进入至圣所的日子。这一章紧接在拿答亚比户之死（10章）后面，表明进入至圣所是极其严肃和危险的事。 "耶和华晓谕摩西说：'要告诉你哥哥亚伦，不可随时进圣所的幔子内、到柜上的施恩座前，免得他死亡，因为我要从云中显现在施恩座上。'"（16:2）

经文分段

第一段：赎罪日的准备（16:1-5）

亚伦的特殊准备：
  • 不可随时进圣所（16:2）
  • 必须带赎罪祭的公牛犊和燔祭的公绵羊（16:3）
  • 必须用水洗身（16:4）
  • 穿圣衣（细麻布内袍、细麻布裤子、束腰、裹头巾）（16:4）
  • 不穿荣耀的圣衣（荣光外袍、以弗得、胸牌、冠冕）
  • 从以色列会众取两只公山羊为赎罪祭，一只公绵羊为燔祭（16:5）

第二段：为亚伦和他家赎罪（16:6-10）

赎罪祭（16:6, 11）：
  • 亚伦献公牛犊为自己和本家赎罪（16:6）
  • 宰杀公牛犊（16:11）
两只公山羊的命运（16:7-10）：
  • 两只公山羊拈阄（16:8）
  • 一阄归耶和华——献为赎罪祭（16:9）
  • 一阄归阿撒泻勒——活着送到旷野（16:10）
  • 归阿撒泻勒的羊要活着站在耶和华面前，为要除罪（16:10）

第三段：为圣所和会众赎罪（16:11-28）

为圣所赎罪（16:11-19）：
  • 宰杀公牛犊（16:11）
  • 取香炉盛火，取两把捣细的香料（16:12）
  • 把香放在火上，在施恩座前使烟遮掩施恩座（16:13）
  • 取公牛的血，用指头弹在施恩座前面七次（16:14）
  • 宰杀归耶和华的那只公山羊（16:15）
  • 把羊血带进幔子弹在施恩座前（16:15）
  • 因以色列人的污秽、过犯、罪孽，为至圣所赎罪（16:16）
  • 为会幕和坛赎罪（16:16-19）
两只公山羊的处理（16:20-22）：
  • 两只公山羊赎罪完毕（16:20）
  • 亚伦两手按在活着的公山羊头上（16:21）
  • 承认以色列人的罪孽、过犯、罪恶（16:21）
  • 把这罪归在羊头上（16:21）
  • 送到旷野无人之地（16:22）
  • "担当他们一切的罪孽"（16:22）
仪式的完成（16:23-28）：
  • 亚伦进会幕脱下细麻衣（16:23）
  • 洗澡（16:24）
  • 穿上衣服出来献燔祭（16:24-25）
  • 放羊的人要洗衣服、洗澡（16:26）
  • 赎罪祭的公牛犊和公山羊送到营外焚烧（16:27-28）

第四段：赎罪日的定例（16:29-34）

永久的定例（16:29-34）：
  • 每年七月初十日（16:29）
  • 你们要刻苦己心（16:29, 31）——禁食
  • 什么工都不可做（16:29, 31）
  • 这是永远的定例（16:29, 31, 34）
  • 一年一次为以色列人赎罪（16:34）

神学主题

1. 至圣所的进入

赎罪日是每年唯一一次大祭司进入至圣所的日子：
  • 施恩座（כַּפֹּרֶת, kapporeth）：约柜上的盖子，象征神的宝座
  • 云：神的临在（16:2）
  • 危险：不按规矩进入会死亡（16:2；10:1-2）
  • 中保：大祭司是神与人之间的中保

2. 赎罪的全面性

赎罪日的赎罪涵盖：
  • 大祭司自己和本家（16:6, 11）
  • 至圣所（16:16）
  • 会幕（16:16）
  • 坛（16:18-19）
  • 全会众（16:15, 20-21）

3. 阿撒泻勒

"阿撒泻勒"（עֲזָאזֵל, Azazel）的含义有不同解释：
  • 代罪羊：羊担当百姓的罪，被送到旷野
  • 罪的移除：罪被带到远离百姓的地方
  • 与神隔离：罪导致与神的分离

4. 刻苦己心

"你们要刻苦己心"（16:29, 31）：
  • 这不仅是禁食，更是内心的悔改
  • 在神面前谦卑
  • 承认自己的罪

基督的预表

1. 大祭司 → 基督是我们的大祭司（来4:14-16） 2. 进入至圣所 → 基督进入天上真正的圣所（来9:11-12） 3. 公牛犊 → 基督为我们的罪献上自己（来9:12-14） 4. 两只公山羊 → 一只死（基督的死），一只活（基督的复活，罪被除去） 5. 施恩座 → 基督是我们的挽回祭（罗3:25） 6. 一年一次 → 基督一次献上，成就永远的赎罪（来9:12） 7. 血的弹洒 → 基督的血洗净我们的罪（约一1:7） 8. 代罪羊 → 基督担当我们的罪（赛53:6；彼前2:24）

今日应用

1. 基督是我们的大祭司：我们不需要人间的中保，基督已经为我们开了一条又新又活的路（来10:19-22） 2. 基督的血完全充足：基督一次献上，成就永远的赎罪，我们不需要重复献祭 3. 悔改的心：赎罪日的"刻苦己心"提醒我们，悔改是信仰的核心 4. 罪的赦免：基督的死使我们的罪被除去，如同代罪羊把罪带到旷野 5. 坦然无惧：因为基督的赎罪，我们可以坦然来到神面前（来4:16）
