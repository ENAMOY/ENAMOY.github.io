03 赎罪祭与赎愆祭 (利未记 4:1-6:7)

经文范围

利未记 4:1 - 6:7

经文背景

前两章讲的是自愿献的祭（燔祭、素祭、平安祭），现在转向因罪而必须献的祭——赎罪祭和赎愆祭。这两种祭处理的是人因软弱而犯的罪，不是故意悖逆的罪（民15:30-31）。 "耶和华对摩西说：'你晓谕以色列人说：若有人在耶和华所吩咐不可行的什么事上误犯了一件……'"（4:1-2）——注意"误犯"一词，这是赎罪祭的前提。

经文分段

第一段：赎罪祭的条例（4:1-35）

四种情况的赎罪祭： 1. 为祭司赎罪（4:3-12） - 祭司犯罪，百姓就因此有了罪（4:3） - 祭物：没有残疾的公牛犊（4:3） - 特殊仪式：把血带进会幕，在圣所幔子弹血七次（4:6） - 把血倒在会幕门口坛脚那里（4:7） - 公牛全部烧在营外倒灰之处（4:12） 2. 为全会众赎罪（4:13-21） - 全会众误犯了罪（4:13） - 祭物：公牛犊（4:14） - 仪式与祭司赎罪相同 - 长老在牛犊头上按手（4:15） 3. 为官长赎罪（4:22-26） - 官长误犯了罪（4:22） - 祭物：没有残疾的公山羊（4:23） - 血涂在坛角上，其余倒在坛脚那里（4:25） 4. 为平民赎罪（4:27-35） - 平民误犯了罪（4:27） - 祭物：母山羊或母绵羊（4:28, 32） - 血涂在坛角上，其余倒在坛脚那里（4:30, 34） - 穷人可用一对斑鸠或雏鸽（4:32-35）

第二段：赎罪祭的补充条例（5:1-13）

几种特殊的赎罪情况：
  • 知道证人指证的事却不作证（5:1）
  • 摸了不洁之物却不自觉（5:2-3）
  • 冒失发誓（5:4）
  • 认罪后献祭（5:5-6）
  • 穷人可用细面代替羊（5:11-13）

第三段：赎愆祭的条例（5:14-6:7）

赎愆祭的几种情况： 1. 在圣物上误犯（5:15-16） - 赔偿圣物的价值，加上五分之一 - 献一只没有残疾的公绵羊 2. 误犯了耶和华的诫命（5:17-19） - 虽然不知道，仍是有了罪 - 献赎愆祭 3. 在邻舍交付之物上犯罪（6:1-3） - 欺骗、抢夺、勒索邻舍 4. 拾物说谎起假誓（6:3-5） - 犯了罪，有了罪 赎愆祭的要求（6:5-7）：
  • 归还所夺取的，加上五分之一
  • 献赎愆祭
  • 祭司为他赎罪

神学主题

1. 罪的严重性

赎罪祭（חַטָּאת, chatat）和赎愆祭（אָשָׁם, asham）的设立，表明：
  • 罪是真实的，不是小事
  • 即使是无意的罪，也需要赎罪
  • 罪影响人与神的关系
罪的层次：
  • 无意的罪（4:2, 13, 22, 27）——赎罪祭
  • 有意的罪（6:1-7）——赎愆祭
  • 故意悖逆的罪（民15:30-31）——没有赎罪祭

2. 赎罪的代价

赎罪祭强调：
  • 血的代价："因为活物的生命是在血中。我把这血赐给你们，可以在坛上为你们的生命赎罪；因血里有生命，所以能赎罪。"（17:11）
  • 祭司的中保：祭司代表百姓来到神面前
  • 代替的概念：祭牲代替罪人死

3. 赔偿的公义

赎愆祭强调：
  • 赎罪不仅仅是口头认罪，还需要实际行动
  • 必须归还所夺取的，加上五分之一（6:5）
  • 必须补偿受害者
  • 这体现了神的公义：罪必须被对付

4. 阶层的平等

四种赎罪祭的安排（祭司、全会众、官长、平民）表明：
  • 所有人都有罪，都需要赎罪
  • 地位越高，责任越大
  • 神不偏待人

基督的预表

1. 无残疾的祭牲 → 基督是无罪的（来4:15） 2. 血流出 → 基督流出宝血（太26:28） 3. 烧在营外 → 基督在城外受苦（来13:11-12） 4. 祭司的中保 → 基督是我们的中保（提前2:5；来7:25） 5. 代替罪人 → 基督代替我们成为罪（林后5:21） 6. 完全的赎罪 → 基督一次献上，成就永远的赎罪（来9:12）

今日应用

1. 认识罪的严重性：即使是无意的罪，也需要神的赦免 2. 认罪的必要：不认罪就无法得赦免（约一1:9） 3. 赔偿的行动：真正的悔改不仅口头认罪，还要有实际的行动 4. 基督的充足：基督的赎罪是完全的，我们不需要再献祭 5. 感恩的心：基督为我们付上了我们永远无法付上的代价
