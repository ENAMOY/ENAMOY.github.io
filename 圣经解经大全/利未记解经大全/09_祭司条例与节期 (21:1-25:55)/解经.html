09 祭司条例与节期 (利未记 21:1-25:55)

经文范围

利未记 21:1 - 25:55

经文背景

这一大段涵盖祭司的特殊条例、祭物的洁净要求、节期制度和禧年条例。这些条例进一步规范了以色列人的宗教生活和社会制度。

经文分段

第一段：祭司的特殊条例（21:1-24）

普通祭司的条例（21:1-9）：
  • 不可为民中的死人沾染自己（21:1）
  • 父母、儿女、兄弟、姐妹可以沾染（21:2-3）
  • 不可使头光秃，不可剃除胡须的周围（21:5）
  • 不可用刀划身（21:5）
  • 要归神为圣（21:6）
  • 不可娶妓女或被休的妇人为妻（21:7）
  • 祭司的女儿若行淫，要用火焚烧（21:9）
大祭司的特殊条例（21:10-15）：
  • 不可蓬头散发（21:10）
  • 不可撕裂衣服（21:10）
  • 不可挨近死人（21:11）
  • 不可出圣所（21:12）
  • 要娶处女为妻（21:13-14）
有残疾之人的限制（21:16-24）：
  • 有残疾的祭司不可献祭（21:17-21）
  • 有残疾的可以吃圣物（21:22）
  • 列举各种残疾（21:18-20）

第二段：祭物的洁净要求（22:1-33）

祭司的洁净（22:1-9）：
  • 不洁的祭司不可吃圣物（22:3-4）
  • 不洁的原因：漏症、梦遗、触摸不洁之物等（22:4-5）
  • 洗澡后可以吃圣物（22:6-7）
圣物的保护（22:10-16）：
  • 寄居的和雇工不可吃圣物（22:10）
  • 买来的奴隶可以吃（22:11）
  • 祭司的女儿嫁给外人，不可吃圣物（22:12）
  • 误吃了圣物，要加上五分之一归还（22:14-16）
祭牲的要求（22:17-33）：
  • 必须是没有残疾的（22:19-25）
  • 不可献有残疾的（22:20-24）
  • 不可在同一天宰杀母牛和小牛（22:28）
  • 不可迟延献祭（22:29-30）

第三段：节期制度（23:1-44）

安息日（23:3）：
  • 每七日的第七日
  • 什么工都不可做
逾越节和无酵节（23:4-8）：
  • 正月十四日黄昏——逾越节（23:5）
  • 正月十五日——无酵节（23:6）
  • 七日吃无酵饼（23:6-8）
初熟节（23:9-14）：
  • 收割庄稼时，将初熟的穗子带给祭司（23:10-11）
  • 献一岁的公羊羔为燔祭（23:12）
  • 素祭和奠祭（23:13）
  • 不可吃新穗子直到献祭（23:14）
五旬节（23:15-22）：
  • 从安息日的次日算起，第七个安息日（23:15-16）
  • 献两个摇祭的饼（23:17）
  • 献燔祭、赎罪祭、平安祭（23:18-19）
  • 留下田角给穷人（23:22）
吹角节（23:23-25）：
  • 七月初一日（23:24）
  • 什么工都不可做
  • 吹角纪念
赎罪日（23:26-32）：
  • 七月初十日（23:27）
  • 刻苦己心（23:27, 29, 32）
  • 什么工都不可做（23:28, 30-31）
住棚节（23:33-44）：
  • 七月十五日开始，共七天（23:34）
  • 第一天和第八天安息（23:35, 39）
  • 拿美好树上的果子、棕树枝、柳树枝欢乐（23:40）
  • 在棚里住七天（23:42）
  • 纪念以色列人住在棚里（23:43）

第四段：圣所的维护（24:1-9）

灯台的油（24:1-4）：
  • 从以色列人取纯橄榄油，使灯常常点着（24:2）
  • 灯台前常常点着（24:4）
陈设饼（24:5-9）：
  • 十二个饼，每安息日摆列（24:5-6）
  • 摆在耶和华面前（24:6）
  • 归给亚伦和他的子孙（24:9）

第五段：亵渎圣名的刑罚（24:10-23）

事件（24:10-12）：
  • 一个以色列妇人的儿子亵渎了圣名（24:11）
  • 被关押等候神的指示（24:12）
神的判决（24:13-23）：
  • 用石头打死（24:14, 16, 23）
  • "凡咒诅神的，必担当他的罪"（24:15）
  • "以伤还伤，以眼还眼，以牙还牙"（24:19-20）
  • 寄居的和本地人同归一例（24:22）

第六段：禧年条例（25:1-55）

安息年（25:1-7）：
  • 第七年为安息年（25:4）
  • 不可耕种田地（25:4）
  • 不可修理葡萄园（25:4）
  • 地要安息（25:5）
  • 地自长的可以吃（25:6-7）
禧年（25:8-55）：
  • 第五十年为禧年（25:10）
  • 每七安息年，七月初十日（25:8-9）
  • 吹角宣告自由（25:10）
  • 各人归回自己的产业（25:10, 13）
  • 不可欺压弟兄（25:14, 17）
禧年的经济条例：
  • 不可在禧年附近欺诈买卖（25:14-17）
  • 地不可永卖，地是我的（25:23）
  • 若贫穷卖地，至近亲属可以赎回（25:25-28）
  • 城内的房屋一年内可以赎回（25:29-30）
  • 利未人的城邑可以赎回（25:32-34）
  • 不可取利借钱给弟兄（25:35-38）
  • 不可使弟兄作奴隶（25:39-43）
  • 可以买外邦人为奴隶（25:44-46）
  • 寄居的可以赎回自己（25:47-55）

神学主题

1. 祭司的圣洁

祭司必须比普通百姓更加圣洁：
  • 不可沾染死亡
  • 不可有残疾
  • 必须保持洁净
  • 这表明事奉神是严肃的事

2. 节期的意义

节期制度教导：
  • 安息日：神创造的完成和人的安息
  • 逾越节：神的救赎
  • 初熟节：感恩和奉献
  • 五旬节：收割和感恩
  • 吹角节：纪念和呼召
  • 赎罪日：悔改和赦免
  • 住棚节：纪念旷野生活和神的供应

3. 禧年的社会公义

禧年条例表明：
  • 地属于神，人只是管家
  • 每五十年重新分配产业
  • 保护穷人和弱势群体
  • 防止永久性的贫穷和奴隶制度

4. 安息的原则

安息年和禧年教导：
  • 人和地都需要安息
  • 信任神的供应
  • 不是永无止境的劳作

基督的预表

1. 节期 → 基督是节期的实体（西2:16-17） 2. 初熟节 → 基督是初熟的果子（林前15:20, 23） 3. 禧年 → 基督宣告禧年（路4:18-19） 4. 安息 → 基督是我们的安息（太11:28-30） 5. 陈设饼 → 基督是生命的粮（约6:35） 6. 灯台 → 基督是世界的光（约8:12）

今日应用

1. 圣洁的事奉：事奉神需要圣洁的生活 2. 节期的纪念：虽然我们不过旧约节期，但要纪念神的救赎 3. 禧年的心肠：关心穷人和弱势群体 4. 安息的操练：在忙碌的生活中操练安息 5. 信任神的供应：安息年和禧年都需要信心
