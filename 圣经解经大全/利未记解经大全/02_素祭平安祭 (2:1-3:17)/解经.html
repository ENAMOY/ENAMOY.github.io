02 素祭与平安祭 (利未记 2:1-3:17)

经文范围

利未记 2:1 - 3:17

经文背景

继第一章的燔祭之后，神继续教导百姓献祭的条例。素祭和平安祭与燔祭不同——它们不是为了赎罪，而是为了表达感恩和团契。素祭是无血之祭，平安祭是共享之祭。

经文分段

第一段：素祭的种类（2:1-16）

三种素祭形式： 1. 细面素祭（2:1-3） - 材料：细面、油、乳香 - 取一把细面和油并所有的乳香烧在坛上 - 剩下的归给亚伦和他的子孙 2. 炉中烤的素祭（2:4） - 调油的无酵薄饼，或抹油的无酵饼 3. 铁鏊上做的素祭（2:5-6） - 用油调和的细面做成 4. 煎盘做的素祭（2:7） - 用油与细面做成 素祭的禁令（2:11-12）：
  • 不可有酵，不可有蜜
  • 不可烧酵或蜜作为火祭献给耶和华
  • 可以用盐（2:13）——"盐约"的概念
初熟的素祭（2:14-16）
  • 将初熟的穗子烘了轧了，献给耶和华

第二段：平安祭的条例（3:1-17）

从牛群献平安祭（3:1-5）：
  • 祭物：没有残疾的公牛或母牛（3:1）
  • 献祭者按手在祭牲头上（3:2）
  • 在会幕门口宰杀（3:2）
  • 祭司将血洒在坛的周围（3:2）
  • 内脏上的脂油和肝上的网子都要焚烧（3:3-4）
  • "是献与耶和华为食物的火祭"（3:5）
从羊群献平安祭（3:6-16）：
  • 公羊或母羊，没有残疾（3:6）
  • 仪式与献牛相同（3:7-11）
  • 从山羊中献也可以（3:12-16）
平安祭的禁令（3:17）：
  • "在你们一切的住处，脂油和血都不可吃"
  • 这是永远的定例

神学主题

1. 素祭：感恩的祭

"素祭"（מִנְחָה, minchah）字根意思是"礼物"或"贡品"。这是唯一没有血的祭，象征：
  • 劳动的成果：细面、油、乳香都是人劳作的产物
  • 感恩的心：不是为赎罪，而是为感谢
  • 生命的奉献：将日常生活中最好的献给神
无酵的象征：
  • 酵在圣经中通常象征罪和败坏（太16:6；林前5:6-8）
  • 无酵象征纯洁和真诚
  • 不可有蜜，因为蜜代表世俗的甜头
盐的象征：
  • 盐有防腐的作用，象征永恒
  • "盐约"（2:13）象征神与人之间永恒的约
  • 耶稣说基督徒是"世上的盐"（太5:13）

2. 平安祭：团契的祭

"平安祭"（שְׁלָמִים, shelamim）字根意思是"完全"或"和平"。这是唯一允许献祭者分享祭肉的祭：
  • 与神和好：平安祭表达献祭者与神之间的和睦关系
  • 团契共享：祭肉由献祭者、祭司和神（脂油焚烧）分享
  • 感恩与许愿：可以是感恩（7:12）、许愿（7:16）或自愿（7:16）
分享的神学：
  • 神得脂油（最好的部分）
  • 祭司得胸和右腿（7:31-34）
  • 献祭者得其余的肉
  • 象征神、祭司、人之间的团契

3. 血与脂油的禁令

"脂油和血都不可吃"（3:17）：
  • 脂油：代表最好的，属于神（3:16）
  • 血：代表生命（利17:11），属于神
  • 这禁令提醒百姓：生命和最好的都是神的

基督的预表

素祭：

1. 无酵 → 基督是无罪的（林后5:21） 2. 初熟的穗子 → 基督是初熟的果子（林前15:20, 23） 3. 人的劳动成果 → 基督完全的人性（来2:14-17） 4. 盐约 → 基督在永恒之约中作中保（来7:22）

平安祭：

1. 平安 → 基督是我们的平安（弗2:14） 2. 和好 → 基督使我们与神和好（林后5:18-19） 3. 团契 → 基督的身体和血成为我们与神团契的记号（林前10:16-17） 4. 分享祭肉 → 主餐中的共享（林前11:23-26）

今日应用

1. 感恩的生活：素祭提醒我们，日常生活中的一切都是从神而来的恩赐 2. 圣洁的生活：无酵象征我们应当除去罪恶，过圣洁的生活 3. 团契的祝福：平安祭提醒我们，与神和好是最大的平安 4. 分享的喜乐：我们不仅自己得享神的恩典，也要与他人分享 5. 初熟的奉献：将最好的献给神，而不是剩余的
