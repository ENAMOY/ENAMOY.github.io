尼希米记解经（一）：尼希米的祷告与归回

经文范围

经文范围：尼希米记 1:1-2:20

本段记录了尼希米听见耶路撒冷城墙被毁的消息后，禁食祷告、向王请求归回、并夜间巡查城墙的经过。尼希米的故事从一个在波斯王宫中身居高位的犹大人的悲伤开始，展示了代祷的心、基于应许的祷告、以及信心与行动的结合。

核心神学命题

1. 代祷的心是信仰领袖的根基——尼希米为同胞的困境悲伤，禁食祷告，表明真正的领袖关心神子民的需要。 2. 基于应许的祷告有大能——尼希米的祷告不是凭感觉，而是基于神的话语和应许。 3. 信心与行动必须结合——尼希米不仅祷告，还积极筹划和执行，表明信心不是被动的等待。

上文回顾

以斯拉记记录了第一批归回者在所罗巴伯的带领下归回耶路撒冷、重建圣殿的过程，以及以斯拉带领第二批归回者归回、进行信仰改革的事迹。以斯拉的改革处理了与外邦人通婚的问题，恢复了信仰的纯洁性。然而，耶路撒冷的城墙仍然是废墟，城门仍然是敞开的。城墙的被毁意味着耶路撒冷没有防御能力，居民暴露在敌人的威胁中。

尼希米的故事发生在以斯拉归回约十三年后（约公元前445年），亚达薛西王二十年。尼希米在波斯首都书珊城作王的酒政，一个有权势、有地位的位置。但他没有忘记自己的同胞和故土。

主题概述

本段的核心主题是代祷与行动的结合。尼希米听见同胞的困境后，不是采取冷漠的态度，而是以深深的悲痛和禁食祷告来回应。他的祷告基于神的话语和应许，他的行动则展现了信心的勇气和智慧。

逐句解经

一、听见困境（1:1-4）

"哈迦利亚的儿子尼希米的言语如下：亚达薛西王二十年基斯流月，我在书珊城的宫中。"（1:1）

"尼希米"（希伯来文 Nehemyah / נְחֶמְיָה）：名字意为"耶和华安慰"或"耶和华的安慰"。他的名字本身就是一个预言——神要通过尼希米来安慰祂的子民。

"书珊城"（希伯来文 Shushan / שׁוּשַׁן）：书珊是波斯帝国的首都之一，位于今伊朗西南部。它是波斯王冬季居住的地方，以华丽的宫殿闻名。尼希米在这个权力中心担任要职。

"酒政"（希伯来文 mashkeh / מַשְׁקֶה）：酒政不仅是品尝酒类的人，更是王身边信任的顾问。在波斯帝国，酒政是一个有权势的位置，可以直接接触王。

"那时，有我一个弟兄哈拿尼，同着几个人从犹大来。我问他们那些被掳归回剩下逃脱的犹大人和耶路撒冷的光景。"（1:2）

"被掳归回剩下逃脱的"：这个表达暗示归回者是少数中的少数。大多数犹大人选择留在波斯帝国，不愿意回到荒凉的故土。那些归回的人被称为"剩下逃脱的"，表明他们是经历了被掳的灾难后仍然存活的人。

"他们对我说：'那些被掳归回剩下的人在犹大省遭大难，受凌辱，并且耶路撒冷的城墙拆毁，城门被火焚烧。'"（1:3）

"遭大难，受凌辱"（希伯来文 bera'ah gedolah uvcherpah / בְּרָעָה גְדוֹלָה וּבְחֶרְפָּה）：这两个词描述了归回者面临的困境。"大难"指物质上的困难和敌人的攻击，"凌辱"指精神上的羞辱和蔑视。城墙被毁、城门被焚，意味着耶路撒冷完全暴露在敌人的威胁中。

历史背景：耶路撒冷城墙在公元前586年被巴比伦人摧毁后，一直没有被重建。归回者虽然重建了圣殿，但城墙仍然是废墟。没有城墙的保护，耶路撒冷的居民面临敌人的持续威胁——参巴拉、多比雅等邻近的领袖不断干扰和羞辱他们。

"我听见这话，就坐下哭泣，悲哀几日，在天上的神面前禁食祈祷。"（1:4）

"坐下哭泣"（希伯来文 yashav uvekhah / יָשַׁב וּבָכָה）：尼希米的反应是即时的、深刻的。他没有漠不关心，而是为同胞的困境悲伤。"坐下"表明他不是短暂的伤感，而是持续的哀伤。

"在天上的神面前禁食祈祷"：尼希米将悲伤转化为祈祷。他不是在人面前诉苦，而是在"天上的神面前"祈祷。"天上的神"这个称呼强调神的超越性和权能——地上的困境不能限制天上的神。

基督论预表：尼希米为同胞的悲伤预表基督为耶路撒冷的哀哭。基督看见耶路撒冷，"就为它哀哭"（路19:41）。尼希米的代祷和基督的代赎都出于对神子民的深爱。

二、认罪祷告（1:5-11）

"耶和华天上的神，大而可畏的神啊，你向爱你、守你诫命的人守约施慈爱。"（1:5）

"大而可畏"（希伯来文 gadol venora / גָּדוֹל וְנוֹרָא）：尼希米以赞美神的属性开始祷告。"大"强调神的伟大和权能，"可畏"强调神的威严和圣洁。尼希米的祷告不是从请求开始，而是从敬拜开始。

"守约施慈爱"（希伯来文 shomer habrit vehachesed / שׁוֹמֵר הַבְּרִית וְהַחֶסֶד）：这是旧约中描述神属性的核心表达。"约"（brit）指神与以色列人所立的西奈之约，"慈爱"（chesed）指神立约的爱、忠实的爱。尼希米的祷告基于神守约的信实。

"愿你睁眼看，侧耳听，你仆人昼夜在你面前为你众仆人以色列民的祈祷。"（1:6）

"睁眼看，侧耳听"：尼希米求神关注他们的困境。这不是说神需要被提醒，而是表达祷告者的恳切和迫切。

"我与我父家都有罪了。"（1:6下）

"我与我父家都有罪了"：尼希米将自己与百姓认同，说"我们"而不是"他们"。这是代祷的态度——不是站在高处审判他人，而是与罪人一同认罪。

基督论预表：尼希米的"我们都有罪"预表基督"凡事与他的弟兄相同"（来2:17）。基督虽然无罪，却将世人的罪归在自己身上（林后5:21）。代祷者需要与所代祷的人认同。

"求你记念所吩咐你仆人摩西的话。"（1:8）

"记念……摩西的话"：尼希米基于神的话语来祷告。他引用的是利未记26:40-45和申命记4:25-31，其中神应许：即使以色列人因罪被掳，只要他们悔改，神仍会记念与他们所立的约。

讲道应用：尼希米的祷告不是凭感觉，而是基于神的话语。我们的祷告也应该以神的话语为根基。当我们祷告时，我们是在与神的应许"对话"，而不是凭空祈求。

三、向王请求（2:1-8）

"亚达薛西王二十年尼散月，在王面前摆酒，我拿起酒来奉给王。我素来在王面前没有愁容。"（2:1）

"我素来在王面前没有愁容"：尼希米在王面前一直保持愉快的面容。这表明他是一个专业的、有自制力的人。但如今，他的悲伤太过深重，无法隐藏。

"王对我说：'你既没有病，为什么面带愁容呢？这不是别的，必是你心中愁烦。'"（2:2）

"面带愁容"（希伯来文 ra' panim / רַע פָּנִים）：在王面前面带愁容是危险的——这可能被视为对王的不满。在波斯帝国，进入王的面前需要保持愉快的面容，否则可能被视为不敬。

"于是我默祷天上的神。"（2:4下）

"默祷"（希伯来文 mitpallel / מִתְפַּלֵּל）：尼希米在回答王之前，先向神默祷。这是一个即时的、简短的祷告。这表明祷告可以是随时的，不一定是长时间的仪式。在关键时刻，一个简短的"默祷"也能连接天上的力量。

讲道应用：你是否养成了随时祷告的习惯？在面临压力和挑战时，你是否先转向神？尼希米的"默祷"提醒我们：祷告不需要华丽的语言或长时间的仪式，而是一颗随时转向神的心。

"王就允准我，因我神施恩的手帮助我。"（2:8）

"我神施恩的手"（希伯来文 yad Elohai hatovah / יַד אֱלֹהַי הַטּוֹבָה）：尼希米认识到，王的允准不是因为自己的口才或魅力，而是因为神施恩的手。"施恩的手"是尼希米记中反复出现的表达，强调神在每一个细节中的护理。

四、夜间巡查（2:12-16）

"我夜间起来，有几个人也一同起来。但神使我心里要为耶路撒冷做什么事，我并没有告诉人。"（2:12）

"夜间起来"：尼希米在夜间秘密巡查城墙。他没有立即开始行动，而是先了解情况。这表明他有智慧和策略，不是盲目行动。

"神使我心里要做什么事"：尼希米认识到，他的使命来自神的感动。他的行动不是出于个人的野心，而是出于神的呼召。

"我出了谷门，往野狗井去，到了粪门，察看耶路撒冷的城墙，见城墙拆毁，城门被火焚烧。"（2:13）

"谷门"、"野狗井"、"粪门"：这些是耶路撒冷城墙上的门和附近的地标。尼希米沿着城墙行走，亲眼察看了毁坏的程度。他的评估是全面的、具体的。

"我又往前察看，见城墙拆毁。"（2:14-15）

全面的评估：尼希米没有只看一处，而是沿着整个城墙巡查。他了解了敌情，为重建城墙做了充分的准备。这表明他是一个有策略的领袖。

五、号召百姓（2:17-20）

"我告诉他们我神施恩的手怎样帮助我，并王对我所说的话。他们说：'我们起来建造吧！'"（2:18）

"我们起来建造吧！"（希伯来文 nuqum venivneh / נָקוּם וְנִבְנֶה）：百姓被尼希米的话激励起来行动。这是一个信心的回应——他们不是在看到完美的计划后才行动，而是在听到神的呼召后就回应。

讲道应用：信心不是等到万事俱备才行动，而是在听到神的呼召后就回应。"我们起来建造吧！"——这不仅是以色列人的口号，也应该是每一个信徒面对神呼召时的回应。

"参巴拉和基善……嗤笑我们，藐视我们，说：'你们做什么呢？要背叛王吗？'"（2:19）

"嗤笑"（希伯来文 latsag / לָעַג）：参巴拉和多比雅用嘲笑来打击百姓的士气。嘲笑是敌人常用的策略——通过蔑视和讽刺来削弱神子民的信心。

基督论预表：参巴拉的嘲笑预表世人对基督和教会的嘲笑。基督在十字架上被人嘲笑（太27:39-44），但嘲笑不能改变神的旨意。

"天上的神必使我们亨通。我们作他仆人的，要起来建造。"（2:20）

"天上的神"：尼希米再次使用"天上的神"这个称呼。他将百姓的注意力从地上的困难转向天上的权能。地上的敌人不能拦阻天上的旨意。

"我们作他仆人的"：尼希米自称是"神的仆人"。这个身份定位是重要的——他们不是为自己建造，而是为神建造。

神学要点总结

一、代祷的心

尼希米为同胞的困境悲伤，禁食祷告。代祷需要一颗关心他人的心，需要与所代祷的人认同。

二、基于应许的祷告

尼希米的祷告基于神的话语和应许。我们的祷告也应该以神的话语为根基，而不是凭感觉。

三、信心与行动

尼希米不仅祷告，还积极筹划和执行。信心不是被动的等待，而是在祷告中采取行动。

四、神的护理

从王的允准到平安的旅程，神在每一个细节中工作。祂的护理是确实的、具体的。

对当年读者的意义

对归回者而言，尼希米的到来带来了新的希望。他们多年来面对敌人的嘲笑和威胁，城墙仍然是废墟。尼希米的异象和领导力激励他们起来建造。这个故事教导他们：神的工作需要领袖，也需要群体的回应。

深化反思与应用

一、关心他人的困境

尼希米在波斯王宫中身居高位，但他没有忘记自己的同胞。今天，我们是否关心教会和世界的需要？我们的舒适是否使我们对他人的困境漠不关心？

二、随时的祷告

尼希米在回答王之前"默祷天上的神"。这提醒我们：祷告不需要长时间的仪式，而是一颗随时转向神的心。

三、在嘲笑中坚持

当敌人嘲笑时，尼希米以信心回应："天上的神必使我们亨通。"今天，当世人嘲笑我们的信仰时，我们是否有同样的信心？

四、从祷告到行动

尼希米不仅祷告，还积极筹划和执行。信心与行动的结合是信仰生活的核心。
