约伯记解经：第二轮对话——比勒达、琐法与约伯

经文范围

约伯记 18:1-21:34

本段包含约伯记第二轮对话的后半部分：书亚人比勒达的第二次发言（18章）、约伯的回应（19章）、拿玛人琐法的第二次发言（20章）、约伯的回应（21章）。这是整个第二轮对话中最激烈、最震撼的一段——比勒达用恶人的命运来威胁约伯（18章），约伯发出了约伯记中最伟大的信仰宣告："我知道我的救赎主活着"（19:25），琐法再次描绘恶人的毁灭（20章），约伯则用恶人亨通的事实来反驳报应神学（21章）。

核心神学命题

1. 报应神学的最终困境——当比勒达（18章）和琐法（20章）用恶人的命运来威胁约伯时，约伯在21章中用一个简单而致命的事实来反驳：恶人反而亨通。这个事实使报应神学面临无法回避的困境。 2. 救赎主的信仰超越死亡——约伯在19:25-27中宣告"我知道我的救赎主活着"，这是约伯记中最重要的信仰时刻之一。约伯在极端苦难中、在死亡面前，宣告了一位活着的救赎主——即使在皮肉消灭之后，他仍要在肉体中见神。 3. 信仰允许质疑神的治理——约伯在21:7-34中的长篇质疑不是叛逆，而是对现实的诚实观察。恶人亨通、义人受苦的事实不容回避——约伯拒绝用"正确神学"来掩盖这个事实。

上文回顾

在第二轮对话的前半部分（15-17章），以利法第二次发言，语气比第一次更加尖锐——他直接指控约伯"不敬畏神"（15:4）、"喝罪孽如水"（15:16）。约伯在回应中控诉三友的残忍（16:1-5），描述神对他的攻击（16:6-17），并发出震撼的宣告："在天上有我的见证人"（16:19）。

第二轮对话的气氛已经显著升级。三友不再试图安慰约伯，而是用更恶毒的语言攻击他。约伯也不再只是表达痛苦，而是开始系统地反驳三友的神学。

主题概述

第二轮对话的后半部分包含四组发言：

  • 比勒达的第二次发言（18章）：比勒达用恶人的命运来威胁约伯，描绘恶人从兴盛到毁灭的全过程。
  • 约伯的第一次回应（19章）：约伯控诉神和三友对他的迫害（19:1-22），然后发出约伯记中最伟大的信仰宣告（19:25-27）。
  • 琐法的第二次发言（20章）：琐法描绘恶人的短暂繁荣和最终毁灭。
  • 约伯的第二次回应（21章）：约伯用恶人亨通的事实来反驳三友的报应神学。

逐句解经

比勒达的第二次发言（18章）

18:1-4

书亚人比勒达回答说："你寻索言语要到几时呢？你可以揣摩思想，然后我们就说话。我们为何算为畜生，在你眼中看作污秽呢？你这恼怒将自己撕裂的，难道大地为你见弃、磐石挪开原处吗？"

  • 你寻索言语要到几时（עַד־אָנָה תְּשִׂימוּן קִנְצֵה לְמִלִּין / ad-annah tesimun qintseh le-millin）：比勒达对约伯的长篇发言感到极度不耐烦。"קִנְץ"（qints）是"终结"——你什么时候才停止说话？
  • 你这恼怒将自己撕裂的（טֹרֵף נַפְשׁוֹ בְּאַפּוֹ / toref nafsho be-appo）：你在愤怒中撕裂自己——比勒达暗示约伯的愤怒是一种自我毁灭。
  • 难道大地为你见弃（הַלְמַעַנְךָ תֵּעָזֵב אָרֶץ / halma'ankha te'azev eretz）：难道大地要因你而被抛弃？——比勒达用讽刺的语气暗示约伯自视过高。
  • 磐石挪开原处吗（וְיִגַּע צוּר מִמְּקוֹמוֹ / ve-yigga' tsur mi-meqomo）：磐石要从原处挪开？——暗示约伯的遭遇不会改变宇宙的秩序。
  • 比勒达从一开始就否认了约伯发言的价值——他不是在对话，而是在压制。

18:5-21

恶人的灯必要熄灭，他的火焰必不照耀。他帐棚中的亮光要变为黑暗，他以上的灯也必熄灭。他坚强的脚步必见狭窄，自己的计谋必将他绊倒。因为他被自己的脚陷入网中，走在缠人的网罗上。圈套必抓住他的脚跟，机关必擒获他。活扣为他藏在地上，羁绊为他设在路上。四面的惊吓要使他害怕，并且追赶他的脚跟。他的力量必因饥饿衰败，祸患要在他旁边等候。他本身的肢体要被吞吃，死亡的长子要吞吃他的肢体。他要从所倚靠的帐棚被拔出来，赶到惊吓的王那里。不属他的火要焚烧他的帐棚，在他的住所上要撒上硫磺。他的根本要枯干，他的枝子要剪除。他的记念在地上必然灭亡，他的名字在街上也要存不住。他必从光明中被撵到黑暗里，必被赶出世界。在本民中必无子无孙，在寄居之地也无一人存留。以后来的要惊奇他的日子，好像以前去的受了惊骇。不义之人的住处就是这样，不认识神之人的所在就是如此。

  • 恶人的灯必要熄灭（נֵר רְשָׁעִים יִדְעָךְ / ner resha'im yida'）：恶人的灯必熄灭——在古代近东文化中，"灯"象征生命、繁荣和后代（参撒下21:17；王上11:36）。
  • 他帐棚中的亮光要变为黑暗：帐棚中的灯光变暗——暗示恶人的家庭将失去光明。
  • 他坚强的脚步必见狭窄（יִצְּרוּ צַעֲדֵי אוֹנּוֹ / yitstseru tsa'adei onno）：他强壮的脚步变狭窄——暗示恶人的道路越来越受限制。
  • 活扣为他藏在地上（תִּטְמֵן בָּאָרֶץ חַבְלּוֹ / titmen ba-arets chab-lo）：活扣藏在地上——陷阱为他预备。
  • 他要从所倚靠的帐棚被拔出来（יוּשַׁר בְּאָהֳלוֹ מִבְּלִי־לוֹ / yushar be-'oholo mi-beli-lo）：他要从帐棚中被连根拔起。
  • 赶到惊吓的王那里（וְתַצְעִדֵהוּ לְמֶלֶךְ בַּלָּהוֹת / ve-tats'idehu le-melekh ballahot）：赶到惊吓之王那里——暗示死亡或神的审判。
  • 不属他的火要焚烧他的帐棚：不属于他的火要烧他的帐棚——暗示毁灭来自外在的力量。
  • 在住所上要撒上硫磺（יָשַׁב שְׁמָמָה בְּאָהֳלוֹ / yashav shemamah be-'oholo）：他的住所变为荒凉——硫磺暗示神的审判（参创19:24）。
  • 他的记念在地上必然灭亡（לֹא־יִזָּכֵר שְׁמוֹ / lo-yizzakher shemo）：他的名字不再被记念——在古代文化中，名字的消失是最严重的诅咒。
  • 他必从光明中被撵到黑暗里：从光明到黑暗——暗示从繁荣到毁灭。
  • 以后来的要惊奇他的日子（יָשֹׁם עַל־יוֹם אָחֵיו אַחֲרוֹן / yashom 'al-yom achiv acharon）：后来的人要惊奇——恶人的命运成为后人的警戒。
  • 不义之人的住处就是这样（כֵּן אָרְחוֹת עַצְּלִי־אָוֶן / ken orchot 'atslei-aven）：不义之人的道路就是如此——比勒达以此总结恶人的命运。
  • 比勒达的整段话都是在描绘恶人的命运，但他的每一个细节都在暗示约伯就是这个恶人。这是一种"间接指控"的策略。

约伯的回应（19章）

19:1-6

约伯回答说："你们搅扰我的心，用言语压碎我要到几时呢？你们这十次羞辱我，你们苦待我也不以为耻。果真我有错，这错乃是在我。你们果然要向我夸大，以我的羞辱为证指责我，就该知道是神倾覆我，用网罗围绕我。"

  • 你们搅扰我的心（עַד־אָנָה תּוֹגְיוּן נַפְשִׁי / ad-annah tog'yun nafshi）：你们要搅扰我的灵魂到几时？——"תּוֹגַה"（togah）是忧伤、搅扰。
  • 用言语压碎我（וּתְדַכְּאוּנַנִּי בְמִלִּים / u-tedakka'uni be-millim）：用话语压碎我——三友的"安慰"实际上是一种言语暴力。
  • 你们这十次羞辱我（זֶה עֶשֶׂר פְּעָמִים תַּכְלִימֻנִּי / zeh 'eser pe'amim takhlimuni）：十次羞辱我——"十次"可能不是精确数字，而是表示"多次"。
  • 果真我有错，这错乃是在我（אַף־אָמְנָם שָׁגִיתִי / af-omnam shagiti）：即使我有错，错也只影响我自己——约伯在此暗示：即使他有错，也不影响三友的论证。
  • 就该知道是神倾覆我（אָמְנָם תֵּדְעוּ כִּי־אֵל עִוָּתָנִי / omnam ted'u ki-El 'ivvatani）：你们应该知道是神使我弯曲——约伯在此承认神是他的苦难的来源，但他的语气不是认罪，而是控诉。

19:7-12

我因委曲呼叫，却不蒙应允；我呼求，却不得公断。他用篱笆拦住我的道路，使我不得经过；又使我的路径黑暗。他剥去我的荣光，摘去我头上的冠冕。他在四围攻击我，我便归于死亡，将我的指望如树拔出来。他的忿怒向我发作，以我为敌人。他的军旅一齐上来，修筑战路攻击我，在我帐棚的四围安营。

  • 我因委曲呼叫，却不蒙应允（הֵן אֶצְעַק חָמָס וְלֹא אֵעָנֶה / hen 'ets'aq chamas ve-lo e'aneh）：我呼喊"暴力"，却得不到回应——约伯控诉神不回应他的呼求。
  • 他用篱笆拦住我的道路（גָּדַר בַּדַּרְכִּי וְלֹא אֶעֱבוֹר / gadar ba-darki ve-lo e'avor）：神用篱笆拦住我的道路——暗示约伯被困在他的苦难中，无处可逃。
  • 又使我的路径黑暗（וְעַל־נְתִיבוֹתַי חֹשֶׁךְ יָשִׂים / ve-'al-netivotai choshekh yasim）：使我的路径黑暗——暗示约伯看不到出路。
  • 他剥去我的荣光 (הִפְשִׁיט עָלַי לְבוּשִׁי / hifshit 'alai levushi)：他剥去我的衣服——在古代文化中，剥去衣服是羞辱的象征。
  • 摘去我头上的冠冕 (יָסִיר מִמֶּנִּי עֲטָרָתִי / yasir mimeni 'atarati)：摘去我的冠冕——冠冕象征荣耀和地位。
  • 将我的指望如树拔出来 (וַיִּתְּץ כָּעֵץ תִּקְוָתִי / va-yitets ka-'ets tiqvati)：将我的指望如树拔出——呼应14:7-9中树木的比喻。
  • 他的军旅一齐上来 (יַחַד עָלַי גְּדוּדָיו יָבֹאוּ / yachad 'alai gedudav yavo'u)：他的军队一齐上来——约伯将神描述为率领军队攻击他的统帅。

19:13-20

他使我的弟兄远离我，使我所认识的全然向我生疏。我的亲戚与我断绝，我的密友都忘记我。在我家寄居的和我的使女都以我为外人，我在他们眼中看为外邦人。我呼唤仆人，他不回答，虽用口恳求还是不听。我口的气味令我妻子厌恶，我的弟兄也厌弃我。连小孩子也藐视我，我若起来，他们就嘲笑我。我一切的密友都憎恶我，我所亲爱的也向我翻脸。我的骨头紧贴皮肉，我只剩牙皮逃脱了。

  • 他使我的弟兄远离我：神使约伯的家人远离他——在古代近东文化中，失去家族的支持是最严重的打击之一。
  • 我的密友都忘记我：密友的遗忘——最亲密的朋友也抛弃了约伯。
  • 我口的气味令我妻子厌恶（רוּחִי זָרָה לְאִשְׁתִּי / ruchi zarah le-ishti）：我的气息令妻子厌恶——暗示约伯的疾病导致口臭。
  • 我只剩牙皮逃脱了（רַק־עַל־עֹר שִׁנָּי / raq-'al-'or shinnai）：只剩牙皮逃脱——此句含义不确定，可能指"只剩下皮包骨"，或"只剩下牙龈上的皮"。约伯描述自己已经接近死亡。
  • 这段话展现了约伯的孤独达到了极致——不仅神攻击他，连家人、朋友、仆人、甚至妻子都远离他。

19:21-22

我的朋友啊，可怜我！可怜我！因为神的手攻击我。你们为什么仿佛神逼迫我，吃我的肉还不满足呢？

  • 可怜我！可怜我！（חָנֻּנִי חָנֻּנִי אַתֶּם רֵעָי / chanuni chanuni attem re'ai）：约伯向三友发出最后的恳求——可怜我吧！这是约伯在发出伟大宣告之前的最后恳求。
  • 因为神的手攻击我：神的手攻击我——约伯再次确认神是他的苦难的来源。
  • 你们为什么仿佛神逼迫我（כִּי יַד־אֱלוֹהַּ נָגְעָה בִּי / ki yad-Eloah nag'ah bi）：你们为什么像神一样逼迫我？——约伯控诉三友的行为等同于神的攻击。
  • 吃我的肉还不满足呢（לָמָּה תִּרְדְּפֻנִי כְמוֹ־אֵל / lammah tird'efuni kemo-El）：为什么你们像神一样追赶我，吃我的肉还不够？——"吃肉"是古代近东比喻中"伤害"的极端表达。

19:25-27

我知道我的救赎主活着，末了必站立在地上。我这皮肉灭绝之后，我必在肉体之外得见神。我自己要见他，亲眼要看他，并不像外人。我的心肠在我里面消灭了。

  • 我知道我的救赎主活着（וַאֲנִי יָדַעְתִּי גֹּאֲלִי חָי / va-ani yada'ti go'ali chai）：这是约伯记中最震撼的信仰宣告。"גֹּאֵל"（go'el）是"救赎者"或"至近亲属"——在旧约中，"גֹּאֵל"有特定的法律含义：指有义务为亲属赎回产业、报仇血恨、或娶寡嫂的至近亲属（参利25:25；民35:19；得3:12）。
  • 末了必站立在地上（וְאַחֲרוֹן עַל־עָפָר יָקוּם / ve-acharon 'al-'afar yaqum）：他必在末后站立在地上——"אַחֲרוֹן"（acharon）是"末后的"、"最后的"。约伯宣告他的救赎者最终必在地上"站立"（קוּם / qum），这可能暗示复活的盼望。
  • 我这皮肉灭绝之后（וְאַחַר עֹרִי נִקְּפוּ־זֹאת / ve-achar ori niqqufu-zot）：在我这皮肉被击碎之后——约伯知道自己即将死去。
  • 我必在肉体之外得见神（וּמִבְּשָׂרִי אֶחֱזֶה אֱלוֹהַּ / u-mi-besari 'echezeh Eloah）：从我的肉体中我必见神——此句含义有争议。可能指"在我的肉体中（即复活的身体）我必见神"，或"离开我的肉体（即死后）我必见神"。
  • 我自己要见他，亲眼要看他（אֲנִי אֶחֱזֶנּוּ וְעֵינַי רָאוּ / ani 'echennu ve-einai ra'u）：我自己要见他，我的眼睛要看见他——不是通过三友的中介，而是亲眼看见。
  • 并不像外人（וְלֹא־זָר / ve-lo zar）：不像外人——约伯与神的关系不是陌生人的关系，而是亲密的。
  • 我的心肠在我里面消灭了（כָּלוּ כִלְיוֹתַי בְּחֵקִי / kalu kilotai be-cheqi）：我的心肠在我里面消融——约伯在极度渴望中感到自己快要消融。
  • 这三节经文是约伯记中最伟大的信仰时刻。约伯在极端苦难中、在死亡面前、在三友的指控中，宣告了一位活着的救赎主。他的宣告包含了几个关键要素：（1）救赎主是活着的；（2）他必在末后站立在地上；（3）约伯必在肉体中见神；（4）这是亲眼的看见，不是外人的关系。

19:28-29

你们若说：我们逼迫他要何等地重呢？惹事的根乃在乎你们。你们就当惧怕刀剑，因为忿怒惹动刀剑的刑罚，使你们知道有报应。

  • 惹事的根乃在乎你们（אִם־תֹּאמְרוּ מַה־נִּרְדָּף־לוֹ / im-tom'ru mah-nirdaf-lo）：惹事的根在你们——约伯警告三友：你们的逼迫将招致神的审判。
  • 你们就当惧怕刀剑：你们应当惧怕刀剑——暗示三友将因他们的行为受到惩罚。

琐法的第二次发言（20章）

20:1-11

拿玛人琐法回答说："我心中急躁，所以我的思念叫我回答。我已听见那羞辱我、责备我的话，我的悟性叫我回答。你岂不知亘古以来，自从人生在地，恶人夸胜是暂时的，不敬虔人的喜乐不过转眼之间吗？他的尊荣虽达到天上，头虽顶到云中，他终必灭亡，像自己的粪一样；素来见他的人要说：'他在哪里呢？'他必飞去如梦，不再寻见，速被赶去，如夜间的异象。亲眼见过他的，必不再见他；他的本处也再见不着他。他的儿女要求穷人的恩，他的手要赔还不义之财。他的骨头虽然有青年之力，却要和他一同躺卧在尘土中。"

  • 我心中急躁（לָכֵן חֶרְבְּנִי עֲנָנַנִי / lakhern cherb'uni 'ananuni）：我的心回应我——琐法声称他不得不发言。
  • 恶人夸胜是暂时的（שִׂמְחַת נָבָל עֲדֵי־רָגַע / simchat naval 'adei-raga'）：恶人的喜乐不过转眼之间——"נָבָל"（naval）是愚昧人、不敬虔的人。
  • 他的尊荣虽达到天上（אִם־יַעֲלֶה לַשָּׁמַיִם שִׂיאוֹ / im-ya'aleh la-shamayim si'o）：即使他的高傲达到天上——暗示恶人的骄傲终将被神打碎。
  • 他终必灭亡，像自己的粪一样（מִנִּבְלָתוֹ יֹאבֵדוּ / mi-nivlato yo'vedu）：从他的腐烂中灭亡——暗示恶人的毁灭如同粪便被丢弃。
  • 他必飞去如梦（חֲלוֹם יָעוּף וְלֹא יִמְצָאֵהוּ / chalom ya'uf ve-lo yimtsa'ehu）：如梦飞逝，不再被找到——恶人的存在如梦般短暂。
  • 他的儿女要求穷人的恩：他的儿女要向穷人乞讨——暗示恶人的后代将失去一切。
  • 他的骨头虽然有青年之力：即使他的骨头有青春的力量——也要躺卧在尘土中。
  • 琐法的第二次发言与比勒达的第一次发言（18章）主题相同：恶人的命运。但琐法的描述更加生动和恶毒。

20:12-29

他口内虽以恶为甘甜，藏在舌头底下，舍不得吐出来，看为宝贵；所以他的食物在肚腹里却要变质，在他里面成为虺蛇的毒。他吞了财宝，还要吐出；神要从他腹中掏出来。他必吸虺蛇的毒气，蝮蛇的舌头也必杀他。他不得再看见江河、流蜜与奶油的溪。他劳碌得来的要赔还，不得吞下；所得财物的欢乐也不得享受。因为他欺压贫穷人，夺取他们所造的；他强夺非自己所盖的房屋。他因贪而无厌，所喜悦的连一样也不能保存。他吞了财宝，还要吐出……恶人经营，得来容易，神却要从他手中取去。神要向他射箭，并不留情。他恨不得逃脱神的手。人要向他拍掌，在他所到之处嗤笑他。

  • 他口内虽以恶为甘甜（אִם־תַּמְתִּיק בְּפִיו רָעָה / im-tamtik be-fiv ra'ah）：即使他口中的恶如蜜甘甜——暗示恶人以犯罪为乐。
  • 藏在舌头底下（יַכְחִדֶנָּה תַּחַת לְשׁוֹנוֹ / yakhidenah tachat leshono）：藏在舌头底下——暗示恶人隐藏自己的罪。
  • 虺蛇的毒（חֲמַת צִפְעֹנִי / chamat tsif'oni）：毒蛇的毒——恶人所吞的"食物"最终变成毒药。
  • 神要从他腹中掏出来（יָרֹעַ אֵל בְּמֵעָיו / yaro'a' el be-me'av）：神要从他腹中取出他所吞的——暗示神要审判恶人。
  • 他不得再看见江河、流蜜与奶油的溪：他不再享受河流和蜜奶——暗示他失去一切祝福。
  • 神要向他射箭：神向他射箭——与约伯在6:4中描述的"全能者的箭"相呼应。
  • 琐法用恶人的命运来威胁约伯——他的每一个细节都在暗示约伯就是这个恶人。

约伯的回应（21章）

21:1-6

约伯回答说："你们要细听我的言语，就算是你们的安慰我。请宽容我，我又要说话。说了以后，任凭你们嗤笑吧。我岂是向人诉冤？为何不焦急呢？你们要看着我而惊奇，用手捂口。我每逢思想，心就惊惶，浑身战兢。"

  • 就算是你们的安慰我（וְאַחֲר כֵּן דַּבֵּרוּ / ve-achar ken dabberu）：约伯请求三友听他说话——这本身就是对他的"安慰"。
  • 我岂是向人诉冤（הֶאָנֹכִי לָאָדָם שִׂיחִי / he-anokhi la-adam sichi）：我不是向人诉冤——约伯的申诉对象是神，不是人。
  • 你们要看着我而惊奇：你们看着我惊奇——暗示约伯的处境使三友也感到震惊。
  • 我每逢思想，心就惊惶：我每逢思想就惊惶——约伯的思考不是平静的，而是充满恐惧。

21:7-16

恶人为何存活，享大寿数，势力强盛呢？他们眼见儿孙和他们一同坚立。他们的家宅平安无惧，神的杖也不加在他们身上。他们的公牛孳生而不断绝，母牛下犊而不掉胎。他们打发小孩子出去，多如羊群，他们的儿女踊跃跳舞。他们随着琴鼓歌唱，又因箫声欢喜。他们度日诸事亨通，转眼下入阴间。他们对神说：离开我们吧！我们不愿晓得你的道。全能者是谁，何必事奉他呢？求告他有什么益处呢？看哪，他们亨通不在乎自己，恶人所谋定的离我好远。

  • 恶人为何存活，享大寿数（מַדּוּעַ רְשָׁעִים יִחְיוּ / maddu'a resha'im yichyu）：恶人为什么活着？——约伯提出了一个简单而致命的问题。
  • 势力强盛呢（עָתְקוּ גַם־גָּבְרוּ חָיִל / 'atqu gam-gavru chayil）：他们年老而且强盛——与报应神学的"恶人必衰"矛盾。
  • 他们眼见儿孙和他们一同坚立：恶人的后代也兴旺——与比勒达在18:19中"他的儿女灭亡"的预言矛盾。
  • 他们的家宅平安无惧：恶人的家宅平安——没有恐惧。
  • 神的杖也不加在他们身上（וְשֵׁבֶט אֱלוֹהַּ עֲלֵיהֶם / ve-sheveth Eloah 'aleihem）：神的杖也不在他们身上——暗示神没有管教他们。
  • 他们的公牛孳生而不断绝：牲畜兴旺——这是神赐福的标志（参申28:4）。
  • 他们打发小孩子出去，多如羊群：儿女多如羊群——大家族是祝福的象征。
  • 他们随着琴鼓歌唱：享受音乐和欢乐。
  • 他们度日诸事亨通（יְבַלּוּ בַטּוֹב יְמֵיהֶם / yevalu ba-tov yemeihem）：他们平安度过日子——一切顺利。
  • 转眼下入阴间（וּבְרֶגַע שְׁאוֹל יֵחָתּוּ / u-verega' She'ol yechattu）：但在转眼之间就下到阴间——暗示恶人的死亡是突然的、安详的。
  • 他们对神说：离开我们吧（וַיֹּאמְרוּ לָאֵל סוּר מִמֶּנּוּ / va-yomru la-El sur mimmennu）：恶人对神说"离开我们"——他们不承认神的权柄。
  • 全能者是谁，何必事奉他（ו�מַה־שַׁדַּי כִּי־נַעַבְדֶנּוּ / u-mah-Shaddai ki-na'avdennu）：全能者是谁？——恶人否认神的权柄。
  • 看哪，他们亨通不在乎自己（הֵן לֹא בְיָדָם טוּבָם / hen lo be-yadam tuvam）：看哪，他们的亨通不在自己手中——约伯在此做了一个微妙的转折：恶人的亨通不是因为他们好，而是因为神允许。
  • 恶人所谋定的离我好远（עֲצַת רְשָׁעִים רָחֲקָה מִמֶּנִּי / 'atsat resha'im ra chaqah mimeni）：恶人的计谋离我好远——约伯在此否认自己与恶人同谋。
  • 约伯用恶人亨通的事实来反驳三友的报应神学。他不是在赞美恶人，而是在指出一个不容回避的事实：报应神学与现实不符。

21:17-26

恶人的灯何尝熄灭？患难何尝临到他们呢？神何尝发怒，向他们分散忿怒的暴风呢？你们说：神为恶人的儿女积蓄罪孽。不如说：神报应本人，使他自己知道。愿他亲眼看见自己败亡，亲自饮全能者的忿怒。他的岁月既尽，他还有什么喜乐呢？有人心里痛苦，终身享福，至终也不知神降祸。有人至死心中痛苦，终身未尝福乐的滋味。他们一样躺卧在尘土中，都被虫子遮盖。

  • 恶人的灯何尝熄灭（נֵר רְשָׁעִים יִדְעָךְ / ner resha'im yida'）：约伯引用比勒达在18:5中的话，然后反驳它——恶人的灯并没有熄灭！
  • 患难何尝临到他们呢（הֲלֹא־צָרָה כָּמוֹהוּ / ha-lo tsarah kamohu）：患难并没有临到他们——约伯用事实反驳比勒达的预言。
  • 神何尝发怒：神没有向他们发怒——暗示神似乎在容忍恶人。
  • 你们说：神为恶人的儿女积蓄罪孽：三友曾说神惩罚恶人的儿女（参比勒达在8:4中指控约伯的儿女犯罪）。约伯反驳：不如让神直接报应本人。
  • 愿他亲眼看见自己败亡：约伯希望恶人亲眼看见自己的毁灭——但事实是恶人往往安详地死去。
  • 有人心里痛苦，终身享福（חֶלְקַת אָדָם מִחוּץ / chelqat adam mi-chuts）：有人终身享福却不知祸患——这是约伯对报应神学最致命的反驳。
  • 有人至死心中痛苦，终身未尝福乐的滋味：有人至死痛苦——义人（如约伯）反而一生受苦。
  • 他们一样躺卧在尘土中（יַחְדָּו עַל־עָפָר יִשְׁכָּבוּ / yachdav 'al-'afar yishkavu）：他们一样躺卧在尘土中——义人和恶人最终的结局相同。
  • 这段话是约伯对报应神学最系统、最致命的反驳。他用事实——恶人亨通、义人受苦——来证明三友的神学是错误的。

21:27-34

我知道你们的意思，并诬害我的计谋。你们说：霸者的房屋在哪里？恶人住过的帐棚在哪里？你们岂没有问过路的人吗？我不知道他们所引的证据吗？就是恶人在患难的日子得存留，在发怒的日子得逃脱。他所行的，有谁当面给他说明？他所作的，有谁报应他呢？然而他被抬到茔地，有人守护他的坟墓。他要以谷中的土为甘甜，在他以前去的无数，在他以后去的也无数。你们对我说的话全是虚空，全是无益的。你们对我答的都是诡诈。

  • 我知道你们的意思（יָדַעְתִּי מַחְשְׁבֹתֵיכֶם / yada'ti machshevotekhem）：我知道你们的想法——约伯不是无知的。
  • 并诬害我的计谋（וְתַזְמוּת עָלַי מִרְמָוֹת / ve-tazmut 'alai mirmavot）：你们用诡计害我——约伯控诉三友的"辩护"实际上是"诬害"。
  • 霸者的房屋在哪里（שְׁאִלְנָא עבְרֵי דָרֶךְ / she'ilna 'ovrei darekh）：约伯引用三友的论点——恶人的房屋在哪里？然后用事实反驳：恶人的房屋仍然存在。
  • 恶人在患难的日子得存留：恶人在患难中得存留——与三友"恶人必灭亡"的预言矛盾。
  • 然而他被抬到茔地（וְהוּא לְקִבְרוֹת יוּבָל / ve-hu le-qivrotyuval）：他被抬到坟墓——暗示恶人的葬礼是隆重的。
  • 有人守护他的坟墓：有人守护他的坟墓——暗示恶人死后仍然受到尊重。
  • 在他以前去的无数，在他以后去的也无数：在他之前和之后去的无数——暗示恶人的死亡和其他人一样，没有特殊的惩罚。
  • 你们对我说的话全是虚空（וְאֵיךְ תְּנַחֲמֻנִי הָבֶל / ve-'ekh tenachamuni havev）：你们的安慰全是虚空——约伯最终否认三友的安慰有任何价值。
  • 约伯以对三友的彻底否定结束第二轮对话。他们的安慰是虚空，他们的论点是诡诈。

神学要点总结

一、报应神学的困境

约伯在21章中用恶人亨通的事实对报应神学进行了最系统、最致命的反驳。恶人存活、强盛、儿女兴旺、平安死去（21:7-16），而义人（如约伯）反而一生受苦（21:25-26）。这个事实使三友的神学面临无法回避的困境——他们的理论与现实不符。

二、救赎主的信仰

约伯在19:25-27中宣告"我知道我的救赎主活着"，这是约伯记中最伟大的信仰时刻。约伯的宣告包含了几个关键要素：

1. 救赎主是活着的（חָי / chai）：不是死去的传说，而是活着的现实。 2. 他必在末后站立在地上：在最后的时刻，救赎者必在地上"站立"（קוּם / qum）。 3. 约伯必在肉体中见神：约伯必亲眼看见神，不是外人的关系。 4. 这是在皮肉灭绝之后：这个盼望超越了死亡。

这个宣告在旧约中极为罕见——它预示了后来复活信仰的发展（参但12:2；赛26:19），并在新约中得到了完全的回应。

三、苦难中的孤独

约伯在19:13-20中描述了他的极端孤独——家人、朋友、仆人、甚至妻子都远离他。这种孤独是苦难中最深刻的部分：不仅是身体的痛苦，更是关系的断裂。约伯的孤独与十字架上的耶稣形成呼应——"我的朋友，我的亲人都离弃我"（参诗22:11；太26:56）。

四、安慰的虚空

约伯在21:34中宣告三友的安慰"全是虚空"（הָבֶל / havev）。这个词与传道书的"虚空"（הֶבֶל / hevel）同源。约伯用这个词来否定三友的神学——他们的安慰不仅没有帮助，反而加重了他的痛苦。

五、在绝望中坚持信仰

约伯在19:25-27中的宣告展现了信仰最深刻的形式：在极端苦难中、在死亡面前、在三友的指控中，他仍然宣告一位活着的救赎主。这不是盲目的乐观，而是一种在绝望中仍然抓住神的信心。

对当年读者的意义

复活信仰的萌芽

19:25-27在旧约中极为罕见。旧约的主流死亡观是：人死后下到阴间（שְׁאוֹל），义人和恶人都去那里，没有区别（传9:2-3）。约伯的宣告打破了这个框架——他相信在皮肉灭绝之后，他必在肉体中见神。这个宣告预示了后来犹太教中复活信仰的发展（但12:2），并在基督教中得到了完全的回应（林前15章）。

义人受苦的事实

约伯在21章中的质疑对古代读者是一个挑战。报应神学是以色列智慧传统的核心——义人得福，恶人遭祸（申28章；箴10:3）。约伯的观察——恶人亨通、义人受苦——挑战了这个传统。约伯记的作者没有给出简单的答案，而是让读者面对这个困境。

至近亲属的角色

约伯用"גֹּאֵל"（go'el，救赎者、至近亲属）来描述天上的那位。在古代以色列社会中，"גֹּאֵל"有特定的法律角色：他必须为亲属赎回失去的产业（利25:25），为被杀的亲属报仇（民35:19），或娶死去兄弟的寡妻（得3:12）。约伯的宣告暗示：天上有他的"至近亲属"，这位亲属将为他伸张正义。

深化反思与应用

关键词：גֹּאֵל（go'el，救赎者、至近亲属）

"גֹּאֵל"在旧约中有丰富的含义：（1）至近亲属，有义务为家族成员伸张正义；（2）救赎者，赎回失去的产业或自由；（3）报仇者，为被杀的亲属追讨血债。约伯用这个词来描述天上的那位——他不是抽象的"神"，而是约伯的"至近亲属"，将为他赎回失去的一切、为他伸张正义。在新约中，耶稣基督正是这个"גֹּאֵל"的最终实现——他成为人的"至近亲属"，通过十字架赎回了人类失去的一切。

古代近东背景：义人受苦的文学

约伯在21章中的质疑在古代近东文学中有广泛对应。苏美尔文献《一个受苦者的独白》和巴比伦文献《巴比伦的约伯》都面对类似的问题——义人为什么受苦？但这些文献最终都以传统的报应神学为结论。约伯记的独特之处在于：它拒绝给出简单的答案，让问题悬而未决。

当代应用

1. 相信活着的救赎主：约伯在19:25中的宣告是每个信徒在苦难中的盼望。当我们面对无法理解的苦难时，我们可以像约伯一样宣告：我知道我的救赎主活着。耶稣基督是我们的"גֹּאֵל"——我们的至近亲属、我们的救赎者。 2. 承认报应神学的局限：约伯在21章中的质疑提醒我们：现实比"正确神学"更复杂。恶人亨通、义人受苦的事实不容回避。面对这个困境，我们不需要给出简单的答案，而是可以像约伯一样诚实地面对。 3. 陪伴孤独的人：约伯在19:13-20中的孤独提醒我们：苦难中最深刻的伤害往往是关系的断裂。面对孤独的人，最重要的不是"正确的神学"，而是陪伴和同理心。

默想引导

默想19:25-27中约伯的话："我知道我的救赎主活着，末了必站立在地上。我这皮肉灭绝之后，我必在肉体之外得见神。"当你面对无法理解的苦难时，你是否能像约伯一样宣告：我知道我的救赎主活着？约伯在三千年前呼求的那位"גֹּאֵל"，在耶稣基督里已经来到——他是我们的至近亲属，他赎回了我们失去的一切。

下文预告

第二轮对话以约伯和三友之间最激烈的交锋结束。在第三轮对话中，争论将进一步激化。以利法将第三次发言（22章），这次他不再只是暗示约伯有罪，而是捏造罪名来指控约伯。约伯将在回应中发出更强烈的呼喊——他渴望在神面前申诉（23章），质疑神为什么不惩罚恶人（24章）。第三轮对话将以三友论证的彻底失败和约伯信仰的进一步深化结束。
