约伯记解经：第一轮对话——比勒达与约伯

经文范围

约伯记 8:1-10:22

本段包含约伯记第一轮对话中第二位发言者——书亚人比勒达的发言（8章）与约伯的回应（9-10章）。比勒达代表的是比以利法更直接、更不容置疑的传统报应观：神不会偏离公义，约伯的儿女必然是因犯罪而死，约伯若纯全就必蒙神恢复。约伯在回应中开始更深入地质疑神的公义——不是否认神的存在，而是质疑神是否在按照公义行事。

核心神学命题

1. 传统报应观坚持神的绝对公义——比勒达认为神不可能偏离公义，约伯的儿女死于罪的后果，约伯若回转向神就必蒙恢复。这一立场在圣经中有其根基（申28章；箴言），但面对约伯的处境时却显得过于简化。 2. 人在神的审判面前无法自辩——约伯在回应中承认神的伟大和人的渺小，但也指出这一现实使义人受苦的问题更加尖锐——如果人无法在神面前自辩，那么"无辜"如何证明？ 3. 苦难中对神的质疑可以达到信仰的最深处——约伯在9-10章中的呼喊不是弃绝神，而是在信仰的极限处追问神的本性——他是慈爱的还是残忍的？他看顾人还是忽略人？

上文回顾

在第一轮对话的前半部分（4-7章），以利法从异象和经验出发，论证了苦难来自罪和神的管教（4-5章）。约伯回应时控诉三友如同干涸的溪水（6:14-17），向神发出深层的呼喊——"人算什么，你竟看他为大？"（7:17）。约伯坚持自己没有弃绝神的话语（6:10），但也宣告他厌弃性命（7:16），请求神赦免他不知名的过犯（7:21）。

序幕已经证实：约伯的苦难不是因为犯罪，而是天上法庭的试炼（1-2章）。三友和约伯都不知道这一点，这一"戏剧性反讽"将继续驱动对话的冲突。

主题概述

比勒达的发言（8章）和约伯的回应（9-10章）形成了第一轮对话的第二组辩论。比勒达的立场比以利法更直接、更不留余地——他直接指控约伯的儿女因犯罪而死（8:4），并以历史和传统的权威为依据，要求约伯回转。

约伯的回应则达到了新的深度：他不再只是表达个人的痛苦（如第3章和第7章），而是开始系统地质疑神的公义。他承认神的伟大（9:2-10），但质疑这种伟大是否意味着公平（9:11-24）；他承认自己的短暂（9:25-35），但质疑神为何要审判一个注定消逝如影的人（10:18-22）。

比勒达的发言可分为两个部分：

  • 8:1-7：指控约伯，指出神不会偏离公义，约伯的儿女因犯罪而死。
  • 8:8-22：以古人的智慧传统和自然界的比喻，论证恶人必衰、义人必兴。

约伯的回应同样可分为两个部分：

  • 9:1-35：承认神的大能，但质疑人在神面前无法自辩的困境。
  • 10:1-22：直接向神发出呼喊，质问神为何创造他然后又毁灭他。

逐句解经

比勒达的发言（8章）

8:1-2

书亚人比勒达回答说："这些话你要说到几时？口中的言语如狂风要到几时呢？"

  • 书亚人比勒达（בִּלְדַּד הַשּׁוּחִי / Bildad ha-Shuchi）："书亚"是亚伯拉罕和基土拉的后裔所居之地（创25:2），位于幼发拉底河中游地区。"比勒达"可能意为"巴力是亲爱的"或"争辩之子"。
  • 这些话你要说到几时（עַד־אָנָה תְּמַלֵּל־אֵלֶּה / ad-annah temallel-eleh）：比勒达对约伯的长篇哀歌感到不耐烦。"עַד־אָנָה"是"到什么时候"——语气尖锐。
  • 口中的言语如狂风（וְרוּחַ כַּבִּיר אִמְרֵי־פִיךָ / ve-ru'ach kavvir imrei-fikha）：你的言语如同暴风——充满力量但没有方向，是破坏性的。比勒达从一开始就否认了约伯发言的正当性。

8:3-4

神岂能偏离公平？全能者岂能偏离公义？你的儿女若得罪了他，他就任凭他们落在自己的过犯里。

  • 神岂能偏离公平（הֲיַטֶּה־אֵל מִשְׁפָּט / ha-yatteh-El mishpat）：这是比勒达论证的基石——神不可能偏离公义。"מִשְׁפָּט"（mishpat）是公正、审判。
  • 全能者岂能偏离公义（וְאִם־שַׁדַּי יְעַוֵּת־צֶדֶק / ve-im-Shaddai ye'avvet-tsedeq）："צֶדֶק"（tsedeq）是公义、正义。比勒达的逻辑是：神是公义的，因此所有苦难都是公义的审判——包括约伯儿女的死。
  • 你的儿女若得罪了他（אִם־בָּנֶיךָ חָטְאוּ לוֹ / im-banekha chata'u lo）：比勒达直接指控约伯的儿女犯了罪——这是序幕中完全没有提及的。比勒达从结果（儿女的死）推导原因（犯罪），这是典型的报应神学推论。
  • 他就任凭他们落在自己的过犯里（וַיְשַׁלְּחֵם בְּיַד־פִּשְׁעָם / va-yeshallem be-yad fish'am）：神将他们交给自己过犯的权势——意思是他们的死是自己犯罪的后果。
  • 此句对约伯而言是极其残忍的——比勒达不仅否认约伯的义，还指控他死去的儿女是罪人。序幕中我们知道约伯的儿女是在筵宴中死于狂风（1:18-19），并非因犯罪受罚。

8:5-7

你若殷勤地寻求神，向全能者恳求；你若清洁正直，他必为你起来，使你公义的居所兴旺。你起初虽然微小，终久必甚发达。

  • 你若殷勤地寻求神（אִם־אַתָּה תְשַׁחֵר אֶל־אֵל / im-attah teschakher el-El）："שָׁחַר"（shachar）是早起寻求、恳切求告。比勒达的建议是：如果你真的清洁正直，就恳切寻求神，他必恢复你。
  • 他必为你起来（אִם־זַךְ וְיָשָׁר אָתָּה כִּי־עַתָּה יָעִיר עָלֶיךָ / im-zakh ve-yashar attah ki-'attah ya'ir 'alekha）：如果你清洁正直，神会为你"醒起"——采取行动帮助你。
  • 你起初虽然微小（אִם־מְעָט וְאַחֲרִיתְךָ מְאֹד יִשְׂגֶּה / im-me'at ve-acharitkha me'od yisgeh）：即使开始时微小，最终也会大大兴旺。比勒达在此给出了"悔改-蒙福"的应许。
  • 比勒达的论点总结：神是公义的→你的儿女死了→他们犯了罪→如果你回转→神必恢复你。这个逻辑看似合理，但它的前提（苦难一定是犯罪的后果）在约伯的处境中不成立。

8:8-10

请你考问前代，留意列祖所查究的事。我们不过从昨日才有，一无所知，我们在世的日子好像影儿。

  • 考问前代（כִּי־שְׁאָל־נָא לְדֹר רִישׁוֹן / ki-she'al-na le-dor rishon）：比勒达诉诸传统权威——询问前代的智慧。这是古代近东智慧文学常见的手法。
  • 我们不过从昨日才有（כִּי־תְמוֹל אֲנַחְנוּ וְלֹא נֵדָע / ki-temol anachnu ve-lo neda'）：我们不过是昨天才出生的——暗示人的一生短暂如影，不足以质疑古人的智慧。
  • 比勒达在此建立了一个权威基础：古人的智慧传统证明了他的论点。约伯若质疑这些，就是质疑整个智慧传统。

8:11-19

蒲草没有泥岂能发长？芦荻没有水岂能生发？尚青的时候，还没有割下来，就比百草先枯。凡忘记神的人，其指望也是如此；不信靠神的人，他的依靠也如蛛网。他虽倚靠他的房屋，房屋却不能存立；他虽抓住房屋，房屋却不能存留。他在日光之下发青，蔓子爬满了园子。他的根盘绕石堆，扎入石地。他若从本地被拔出，那地就不认识他，说："我没有见过你。"看哪，他欢喜自己的道路，却从尘土中长出另一个人来。

  • 蒲草没有泥岂能发长（הֲיִגְאֶה־גֹּמֶא בְּלֹא בִצָּה / ha-yig'eh-gome' be-lo bitsah）：蒲草需要沼泽的泥才能生长——同样，不信靠神的人没有根基。
  • 芦荻没有水岂能生发（יִשְׂגֶּה־אָחוּ בְלֹא מָיִם / yisgeh-achu be-lo mayim）：芦苇需要水才能茂盛。
  • 凡忘记神的人，其指望也是如此（כֵּן אָרְחוֹת כָּל־מְנַחֵי אֵל / ken orchot kol-menachei El）：忘记神的人的"指望"如同没有根的植物。
  • 不信靠神的人，他的依靠也如蛛网（בֵּיתוֹ עַכָּבִישׁ / beito 'akkavish）：不信靠神的人的房屋如同蜘蛛网——看似有结构，但一碰就碎。
  • 他的根盘绕石堆（עַל־גַּל יְסַבֵּב שָׁרָשָׁיו / al-gal yesabbev sharashav）：恶人看似有根，但根只是盘绕在石头上——没有真正的土壤。
  • 他若从本地被拔出：一旦被拔出，连那地都不认识他——恶人的存在不留痕迹。
  • 比勒达用一系列自然界的比喻论证：不信靠神的人看似兴旺，但最终必如无根的植物般消亡。这暗示约伯的处境正是如此——他看似曾经兴旺，但现在被拔出，说明他没有真正的根基。

8:20-22

神必不丢弃完全人，也不扶助邪恶人的手。他还要以喜笑充满你的口，以欢呼充满你的嘴唇。恨你的人要披戴惭愧，恶人的帐棚必归于无有。

  • 神必不丢弃完全人（הִנֵּה אֵל לֹא־יִמְאַס־תָּם / hinneh El lo-yim'as-tam）：神不会弃绝完全人。比勒达在此暗含了一个尖锐的逻辑：如果约伯被弃绝了，那他就不完全。
  • 以喜笑充满你的口（עַד־יְמַלֵּה שְׂחוֹק פִּיךָ / ad-yesakhekha sechoq pikha）：如果你回转，神会让你重新喜笑——失去的一切都会恢复。
  • 恨你的人要披戴惭愧：你的仇敌将蒙羞。
  • 这是比勒达的"福音"：悔改→蒙福；坚持己见→继续受苦。在一般情况下这个原则是对的，但在约伯的处境中，它预设了一个错误的前提。

约伯的回应（9-10章）

9:1-2

约伯回答说："我真知道是这样。但人在神面前怎能成为义呢？"

  • 我真知道是这样（אָמֵן יָדַעְתִּי כִּי־כֵן / amen yada'ti ki-ken）：约伯同意比勒达的一个前提——神确实不会偏离公义（8:3）。但他在同意的同时提出了一个更深刻的问题。
  • 人在神面前怎能成为义呢？（מַה־יִּצְדַּק אֱנוֹשׁ עִם־אֵל / mah-yitsdaq enosh im-El）：如果神是绝对公义的，人在他面前怎么能称义？这不是认罪，而是一个认识论的问题——人没有资格在神面前为自己辩护。与以利法4:17的"人岂能比神公义"相呼应，但语气不同：以利法的结论是"所以人不应该质疑"，约伯的结论是"所以人无法自辩"。

9:3-4

人若愿意与他争辩，千中之一也不能回答。他心里有智慧，且大有能力。谁向神刚硬而得亨通呢？

  • 千中之一也不能回答（אִם־יַחְפֹּץ לָרִיב עִמּוֹ לֹא־יַעֲנֶנּוּ אַחַת מִנִּי־אָלֶף / im-yachpotz la-riv imo lo-ya'ennenu achat mini-alaf）：如果人想和神争辩，千分之一的问题也答不了。这不是在赞美神的智慧，而是在哀叹人面对神的无助。
  • 他心里有智慧，且大有能力（חֲכַם לֵבָב וְאַמִּיץ כֹּחַ / chakham levav ve-'ammits ko'ach）：神有智慧和力量——但他是否用公义使用这些？
  • 谁向神刚硬而得亨通（מִי הִקְשָׁה אֵלָיו וַיִּשְׁלָם / mi hiqshah elav va-yishlam）：谁向神硬着颈项还能亨通？暗示与神对抗是没有出路的。

9:5-10

他发怒把山翻倒挪移，山并不知觉。他使地震动，离其本位，地的柱子就摇撼。他吩咐日头不出来，又封闭众星。他独自铺张诸天，步行在海浪之上。他造北斗、参星、昴星和南方的众星。他行大事不可测度，行奇事不可胜数。

  • 他发怒把山翻倒（מַעְתִּיק הָרִים וְלֹא יָדָעוּ / ma'tiq harim ve-lo yada'u）：神在愤怒中移动山岭，而山并不知道——暗示神的力量超越自然界。
  • 他使地震动（מַרְגִּיז אֶרֶץ מִמְּקוֹמָהּ / margi' eretz mi-meqomah）：地震、日月遮蔽——这些都是创造主权的展示。
  • 步行在海浪之上（דֹּרֵךְ עַל־בָּמֳתֵי־יָם / dorekh al-bamotey-yam）：神在海浪之上行走——这与耶稣在水上行走（太14:25）形成有趣的呼应。
  • 北斗、参星、昴星（עָשָׁה עָשׁ כְּסִיל וְכִימָה / 'asah 'Ash Kesil ve-Khima'）：北极星座、猎户座、昴星团——神创造了天上的星宿。
  • 约伯在此承认了以利法和比勒达都强调的真理：神确实大有能力、行大事不可测度。但约伯的结论不是"所以人应该谦卑"，而是"所以人无法在神面前自辩"。

9:11-12

他从我旁边经过，我却看不见；他在我面前行走，我倒不知觉。他夺取，谁能阻挡？谁敢问他"你做什么"？

  • 他从我旁边经过，我却看不见（הֵן יַעֲבֹר עָלַי וְלֹא אֶרְאֶה / hen ya'avor 'alai ve-lo er'eh）：神从约伯身边经过，约伯却看不见——暗示神的行动对人而言是隐秘的。
  • 谁敢问他"你做什么"（מִי־יֹאמַר אֵלָיו מַה־תַּעֲשֶׂה / mi-yomar elav mah-ta'aseh）：谁敢质问神的作为？这个问题充满了张力——约伯在质疑神，但他同时知道这种质疑在逻辑上是无望的。

9:13-20

神必不收回他的怒气，扶助拉哈伯的屈身在其下。何况我怎能回答他，怎能选择言语与他辩论呢？我虽有义，也不回答他，只要向那审判我的恳求。我若呼吁，他应允我，我仍不信他真听我的声音。他用暴风折断我，无故加增我的损伤。我就是喘一口气都不容，使我饱受苦恼。若论力量，他真有能力；若论审判，谁能传我出庭呢？我虽有义，自己的口要定我为有罪；我虽完全，他必定我为弯曲。

  • 扶助拉哈伯的屈身（תַּחַתּוֹ שָׁחוּ עֹזְרֵי רָהַב / tachtav shachu 'ozrei Rahav）："רָהַב"（Rahav）是古代神话中的海怪（参赛30:7；诗87:4），代表混沌力量。即使那些帮助混沌力量的，也在神面前屈身——暗示无人能抵挡神。
  • 何况我怎能回答他（אַף כִּי־אָנֹכִי אֶעֱנֶנּוּ / af ki-anokhi 'e'ennennu）：如果连宇宙的力量都在神面前屈身，我一个脆弱的人怎能回答他？
  • 我虽有义，也不回答他（אֲשֶׁר־אַצְדִּיק דְּבָרַי לֹא אֶעֱנֶנּוּ / asher-'atsdiq devarai lo 'e'ennennu）：即使我是对的，我也不回答——因为在神面前，"对"和"错"的标准不是人的标准。
  • 他用暴风折断我（אֲשֶׁר בְּשֵׂעָף יְשׁוּפֵנִי / asher be-se'af yesufenni）：神用暴风（שֵׂעָף / se'af）打击我——"无故地"（חִנָּם / hinnam，与1:9撒但的用词相同）。
  • 我虽有义，自己的口要定我为有罪（אִם־צָדַקְתִּי פִּי יַרְשִׁיעֵנִי / im-tsadaqti pi yarshi'eni）：即使我是对的，我自己的口也要定我为有罪——因为在神面前，任何辩护都显得无力。这不是认罪，而是对"在神面前自辩"之不可能的哀叹。
  • 我虽完全，他必定我为弯曲（תָּם אָנֹכִי וַיַּעְקְשֵׁנִי / tam anokhi va-ya'qeshenni）：即使我是完全的，神也要使我弯曲——约伯在此指控神的"不公"。这不是约伯真的说神不公，而是他从自己的经验出发，感到神似乎在使义人受苦。
  • 这段经文是约伯记中最深刻的神学表述之一：如果人无法在神面前自辩，如果神使义人受苦，那么人与神之间的关系如何维持？

9:21-24

我虽完全，我却不顾自己，我厌恶我的性命。善恶都是一样，所以我说：完全人和恶人都是一样灭绝。若忽然遭杀害之祸，他必戏笑无辜人的绝望。世界交在恶人手中，蒙蔽审判官的脸；若不是他，是谁呢？

  • 我虽完全，我却不顾自己（תָּם אָנֹכִי לֹא־אֵדַע נַפְשִׁי / tam anokhi lo-'eda' nafshi）：我是完全的，但我不在乎自己的"完全"——因为既然完全人和恶人一样灭绝，"完全"有什么价值？
  • 善恶都是一样（אֶחָד הוּא עַל־כֵּן אָמַרְתִּי תָּם וְרָשָׁע הוּא מְכַלֶּה / echad hu al-ken amarti tam ve-rasha' hu mekalleh）：善恶的结果一样——这是约伯最尖锐的质疑：如果义人和恶人一样灭亡，那么义有什么意义？
  • 世界交在恶人手中（אֶרֶץ נִתְּנָה בְיַד־רָשָׁע / eretz nittenah be-yad-rasha'）：世界被交在恶人手中——审判官被蒙蔽，正义被扭曲。
  • 若不是他，是谁呢？（פְּנֵי שֹׁפְטֶיהָ יְכַסֶּה אִם־לֹא אֵפוֹ מִי־הוּא / penei shofteha yekhasseh im-lo efou mi-hu）：如果不是神，是谁蒙蔽了审判？约伯在此将质疑直接指向神——如果神掌管一切，那么世上的不公最终也是神的责任。

9:25-31

我的日子比跑信的更快，急速过去，不见福乐。我的日子过去如快船，如急落抓食的鹰。我若说"我要忘记我的哀情，我要除去愁容，仍然畅快"，我就惧怕我一切的痛苦，我知道你不以我为无罪。我必被你定为有罪，我何必徒然劳苦呢？我若用雪水洗身，用手碱洁净自己，你还要把我扔在坑里，连我的衣服都厌恶我。

  • 比跑信的更快（אִם־קַלּוּ מִמֶּנִּי יְמֵי־רָץ / im-qallu mimeni yemei-rats）：日子比奔跑的信使还快——生命在痛苦中飞速流逝。
  • 如快船（חָדְשׁוּ מִנִּי־אֶזֶרֶת / chadshu mini-'ezeret）：如同快船滑过水面——不留痕迹。
  • 如急落抓食的鹰（חָשׁוּ כְנֶשֶׁר יָטוּשׂ עֲלֵי־אֹכֶל / chashu ka-nesher yatus 'alei-okhel）：如同俯冲捕食的鹰——快速而无情。
  • 我知道你不以我为无罪（יָדַעְתִּי כִּי לֹא תְנַקֵּנִי / yada'ti ki lo tenaqeni）：约伯痛苦地意识到——无论他做什么，神都不会宣告他无罪。这不是说约伯承认自己有罪，而是说在神面前，人永远无法证明自己的"无罪"。
  • 我若用雪水洗身（אִם־הִתְרָחַצְתִּי בְמֵי־שֶׁלֶג / im-hitrachatsi be-me-shelgeg）：即使我用最洁净的水洗身——在象征意义上做最彻底的自洁——神还是要把我扔在坑里。
  • 连我的衣服都厌恶我（וַתִּגְעַל בַּשַּׁחַת לְבֻשִׁי / va-tig'al ba-shachat levushi）：连我的衣服都厌恶我——暗示约伯的污秽是如此彻底，连无生命的物体都感到厌弃。

9:32-35

他本不像我是人，使我可以回答他，又使我们可以同听审判。我们中间没有听讼的人可以向我们两造按手。愿他把杖离开我，不使惊惶威吓我；我就说话，也不惧怕他，现在我却不是那样。

  • 他本不像我是人（לֹא־אִישׁ כָּמוֹנִי אֲעֶנֶנּוּ / lo-ish kamoni 'e'ennennu）：他不是像我一样的人，使我能够回答他——这是约伯最深的哀叹之一。人与神之间存在本体论的鸿沟——神不是人，人无法像与另一个人辩论那样与神辩论。
  • 我们中间没有听讼的人（אֵין בֵּינֵינוּ מוֹכִיחַ / ein beineinu mokiach）：在神和约伯之间没有中间人、没有仲裁者。这个呼喊预示了约伯在9:33中渴望的"中保"（מֹכִיחַ / mokiach，听讼者、辩护者）。
  • 愿他把杖离开我（יָסֵר מֵעָלַי שִׁבְטוֹ / yaser me'alai shivto）：愿神的审判之杖离开我——让我能够不惧怕地说话。
  • 现在我却不是那样（כִּי־כֵן לֹא אָנֹכִי עִמָּדִי / ki-ken lo anokhi immadi）：但现在我不是那样——我现在被恐惧和痛苦所压制，无法不惧怕地站立。

10:1-3

我厌烦我的性命，必由着自己述说我的哀情，因心里苦恼，我要说话。对神说：不要定我有罪，要指示我，你为何与我争辩。你手所造的，你又欺压，又藐视，却光照恶人的计谋，你以为美吗？

  • 厌烦我的性命（נַקְּטָה נַפְשִׁי בְחַיָּי / naqtah nafshi be-chayyai）："קָטַן"（qatan）在此作"厌倦"——约伯对自己的生命感到厌倦。
  • 必由着自己述说（אֶעְזְבָה עָלַי שִׂיחִי / e'ezvah 'alay sichi）：我要放任自己说出我的哀情——不再约束自己的言语。
  • 不要定我有罪（אַל־תַּרְשִׁיעֵנִי / al-tarshi'eni）：不要先定我有罪——让我知道我犯了什么罪。
  • 你手所造的，你又欺压（הֲטוֹב לְךָ כִּי־תַעֲשֹׁק כִּי־תִּמְאַס יְגִיעַ כַּפֶּיךָ / ha-tov lekha ki-ta'asoq ki-tim'as yegia' kapekha）：你亲手所造的，你又欺压和藐视？这是约伯最尖锐的质问之一——创造者为什么要毁灭他自己的创造物？

10:4-7

你岂有肉眼吗？你岂能看见人所看见的吗？你的日子岂像人的日子？你的年岁岂像人的年岁？你追问我的罪孽，寻察我的罪过。其实你知道我没有罪过，也没有能救我脱离你手的。

  • 你岂有肉眼（הֲעֵינֵי בָשָׂר לָךְ / ha-einei basar lakh）：你有血肉之眼吗——你是否用人的有限视角看事物？
  • 你的日子岂像人的日子：你的日子不像人的日子——暗示神的时间维度与人完全不同。
  • 你知道我没有罪过（וְיָדַעְתָּ כִּי־לֹא רָשָׁעְתִּי / ve-yada'ta ki-lo rasha'ti）：约伯在此做出了一个大胆的宣告：你知道我是无罪的。这不是约伯否认自己有任何过犯，而是说他没有犯过能导致如此巨大苦难的罪。
  • 也没有能救我脱离你手的：没有人能救我脱离神的手——这呼应了9:12中"谁能阻挡"的主题。

10:8-12

你的手创造我，造就我的四肢百体，你却要毁灭我。求你记念，你造我如抟泥一般，你还要使我归于尘土吗？你不是把我倒出来好像奶，使我凝结如同奶饼吗？你以皮和肉为衣给我穿上，用骨与筋把我联络。你将生命和慈爱赐给我，你也眷顾保全我的心灵。

  • 你的手创造我（יָדֶיךָ עָצְבוּנִי וַיַּעֲשׂוּנִי / yadekha 'atsevuni va-ya'asuni）：你的手塑造了我——"עָצַב"（'atsav）是塑造、制作（参创2:7用泥造人）。
  • 你还要使我归于尘土吗？：造物主为什么要毁灭他亲手制造的？
  • 把我倒出来好像奶（הֲלֹא כֶחָלָב תִּתִּכֵנִי / ha-lo ke-chalab tittenikheni）：你把我倒出来如同奶——暗示胎儿在子宫中的形成如同凝结的奶。
  • 用骨与筋把我联络（עֶצֶם וּגְדִים תְּסֹבְכֵנִי / etsem u-gedim tesovkheni）：用骨头和筋腱把我编织在一起。
  • 你将生命和慈爱赐给我（חַיִּים וָחֶסֶד עָשִׂיתָ עִמָּדִי / chayyim va-chesed 'asita immadi）：你赐给我生命和"חֶסֶד"（慈爱）——这是约伯承认神曾经的恩典。
  • 眷顾保全我的心灵（וּפְקֻדָּתְךָ שָׁמְרָה רוּחִי / u-fqudatkha shamerah ruchi）：你的看顾保守了我的灵。
  • 约伯在此用创造神学来质问神：你既然创造了我、赐给我生命和慈爱，为什么要毁灭我？创造和毁灭之间的矛盾是约伯最深的困惑。

10:13-17

然而你待我的这些事，早已藏在你心里，我知道你久有此意。我若犯罪，你就察看我，并不赦免我的罪孽。我若行恶，便有了祸；我若为义，也不敢抬头，正是满心羞愧，饱受我的烦恼。我若昂首自夸，你就追捕我如狮子，又在我身上彰显你的奇能。你重立见证攻击我，向我加增恼怒，如更换军兵攻击我。

  • 早已藏在你心里（וְאֵלֶּה טָמַנְתָּ בִלְבָבֶךָ / ve-eleh tamanta bilvavekha）：这些事早已藏在你心里——暗示神在创造约伯时就已经计划了他的苦难。
  • 我知道你久有此意（יָדַעְתִּי כִּי־זֹאת עִמָּךְ / yada'ti ki-zot immakh）：我知道这是你一直以来的意图——约伯质疑神是否从一开始就有毁灭他的计划。
  • 我若犯罪，你就察看我：如果我犯罪，你就不赦免——但如果我行义呢？结果也是"不敢抬头"——无论善恶，结果都一样。
  • 追捕我如狮子（וְעַל־תַּנִּין תִּתְגַּבָּר / ve'al tannin titgabbar）：你追捕我如同狮子（或海怪）——暗示神的攻击如同猛兽的追捕。
  • 重立见证攻击我（תְּחַדֵּשׁ עֵדֶיךָ בִּי / techaddes 'eddekha bi）：你不断更新对我的控告——不断有新的"证人"出现指控约伯。

10:18-22

你为何使我出母胎呢？不如我当时气绝，无人得见我。这样，就如没有我一般，一出母胎就被送入坟墓。我的日子不是甚少吗？求你停手宽容我，叫我在往而不返之先——就是往黑暗和死荫之地以先——可以稍得畅快。那地是幽暗，是死荫混沌之地，那里光好像幽暗。

  • 你为何使我出母胎（וְלָמָּה מִבֶּטֶן הוֹצֵאתָנִי / ve-lammah mi-betev hotse'tani）：呼应3:11和3:20——为什么让我出生？
  • 不如我当时气绝（מִי־יִתֵּן כִּי הֹיִיתִי וְעַתָּה לֹא אֶהְיֶה / mi-yitten ki hoyiti ve-'attah lo 'ehyeh）：当时就不存在该多好。
  • 一出母胎就被送入坟墓（מִבֶּטֶן לְקֶבֶר אוּבָל / mi-betev le-kever uval'）：从子宫直接到坟墓——跳过人生。
  • 我的日子不是甚少吗？（הֲלֹא־מְעַט יָמַי יֶחְדָּל / ha-lo-me'at yamai yechdal）：我的日子已经够少了——求你停手，让我在死前稍得喘息。
  • 往而不返之先（קֹדֶם אֵלֵךְ וְלֹא אָשׁוּב / qodem elekh ve-lo ashuv）：在我去那不归之地以前。
  • 黑暗和死荫之地（אֶרֶץ חֹשֶׁךְ וְצַלְמָוֶת / eretz choshekh ve-tsalmavet）：黑暗和死荫之地——阴间（שְׁאוֹל）。
  • 那地是幽暗，是死荫混沌之地（אֶרֶץ עֵיפָתָה כְמוֹ אֹפֶל / eretz 'eifatah kemo ofel）：黑暗如同深渊——"עֵיפָתָה"指极度的黑暗。
  • 光好像幽暗（וְלֹא־סְדָרִים וַתֹּפַע כַּמֹּפֶל / ve-lo-sedarim va-tofa' ka-mofel）：即使有光也如同黑暗——阴间是一个没有秩序、没有光明的地方。
  • 约伯的哀歌在此达到最后的高潮：他不仅渴望死亡（如3:11-19），更在死前向神恳求——在去那不归之地以前，让我稍得喘息。这不是弃绝生命，而是在生命无法承受之重下发出的最后呼声。

神学要点总结

一、神的公义与人的困境

比勒达坚持神的绝对公义（8:3），约伯也同意这一点（9:2）。然而，约伯从这个前提导出了一个痛苦的结论：如果神是绝对公义的，而人在神面前永远无法称义（9:2-3），那么"公义"对人而言意味着什么？如果善恶的结果一样（9:22），那么义有什么价值？这不是约伯否认神的公义，而是他质疑人在面对绝对公义时的无力。

二、创造与毁灭的张力

约伯在10:8-12中用创造神学质问神：你用慈爱创造了我，为什么又要毁灭我？这个问题触及了创造论的核心：如果神是良善的，为什么他的创造物要受苦？如果神是全能的，为什么他不阻止苦难？这些问题在整本圣经中回响（参诗89:46；赛45:7；罗9:20-21）。

三、中保的渴望

约伯在9:33中渴望一位"听讼者"（מֹכִיחַ），能在神和人之间按手。这个渴望在新约中得到了回应——耶稣基督作为神与人之间的中保（提前2:5；来8:6; 9:15; 12:24）。约伯在不知情中呼求的，正是后来启示所成就的。

四、苦难的无解性

约伯在9-10章中的回应表明：苦难的问题在人类理性的范围内是无解的。人无法在神面前自辩（9:3），人无法理解神的道路（9:10-12），人无法逃脱神的手（10:7）。这不是约伯的投降，而是对人类认知极限的诚实承认。

五、信仰的极限处

约伯在10:18-22中的最后呼喊达到了信仰表达的最深处：他没有弃绝神（他仍然向神说话），但他表达了对存在的彻底厌倦。这种在信仰极限处的真实表达，为后世的信徒在苦难中提供了语言和空间。

对当年读者的意义

传统智慧的坚持与挑战

比勒达的发言代表了以色列智慧传统中最核心的信念：神是公义的，义人必蒙福，恶人必遭祸。这一信念在申命记（28章）、箴言（10:3; 11:19）和许多诗篇中有坚实基础。然而，约伯的处境挑战了这一传统——不是因为它完全错误，而是因为它不能涵盖所有情况。对原初读者而言，约伯记是智慧传统的自我批判——在坚持核心信仰的同时，承认其局限性。

阴间观的演变

约伯在10:21-22中对阴间的描述（黑暗、死荫、混沌）反映了旧约时期以色列人对死后世界的理解。这与后来犹太教和基督教关于天堂和地狱的教导有所不同。旧约中的阴间是一个幽暗、无力的地方（赛38:18；诗6:5），义人和恶人都去那里。这种"来世观"使得约伯的处境更加绝望——不仅此生受苦，死后也没有安慰。

深化反思与应用

关键词：מֹכִיחַ（mokiach，听讼者、辩护者）

"מֹכִיחַ"在9:33中出现，意为"听讼者"、"辩护者"、"仲裁者"。约伯渴望在神和人之间有这样一位中保——他能在两者之间按手，消除本体论的鸿沟。在旧约中，天使有时担任这个角色（参亚3:1-5），但约伯渴望的是更根本的解决。新约宣告耶稣基督正是这样一位中保——"在神和人中间，只有一位中保，就是降世为人的基督耶稣"（提前2:5）。约伯在三千年前的渴望，在十字架上得到了回应。

古代近东背景：创造者与被造物的关系

约伯对创造者的质疑（10:3-12）在古代近东文学中有广泛对应。巴比伦创世史诗《埃努玛·埃利什》中，马尔杜克用被杀的提亚马特的身体创造世界——创造本身就伴随着暴力。约伯记的独特之处在于：它不接受创造中暴力的正当性，而是质问创造者为何要用慈爱创造（10:12）然后又毁灭。

当代应用

1. 传统智慧的价值与局限：比勒达的立场（神是公义的，苦难是神的管教）在大多数情况下是对的，但它不能涵盖所有苦难的处境。面对无法解释的苦难时，不要急于给出"神学答案"——有时候承认"我不知道"比给出错误的答案更诚实。 2. 承认人的有限性：约伯在9:2-3中的承认——人在神面前无法自辩——是对人类认知极限的诚实。这不是反智主义，而是认识到：有些问题超出了人类理性的范畴。信仰不是对所有问题都有答案，而是在没有答案时仍然信靠。 3. 创造与存在的意义：约伯在10:8-12中的质疑触及了存在主义的核心问题：如果生命是被赐予的，创造者是否有权收回？这个问题在安乐死、自杀等现代伦理讨论中仍然具有深刻的相关性。

默想引导

默想9:33中约伯对中保的渴望。在你和神之间，是否感到存在无法跨越的鸿沟？约伯在三千年前呼求的那位"听讼者"，在耶稣基督里已经来到。他是"神与人之间的中保"（提前2:5），他曾在人性中经历了约伯所经历的一切——被神"离弃"（太27:46），在黑暗和死荫中呼喊。在基督里，约伯的渴望得到了回应。

下文预告

约伯的回应远未结束。在第一轮对话的最后一组中，琐法将发言（11章），以更尖锐的语气指控约伯隐藏的罪。约伯随后将发出更长的回应（12-14章），其中包含对三友的嘲讽、对神主权的更深思考、以及对复活的隐约盼望（14:14）。第一轮对话将在约伯的独白中结束，为第二轮更激烈的辩论铺路。
