第十二章：以利户的第三、第四次讲话（34:1-37:24）

经文范围

约伯记 34:1-37:24，涵盖：

  • 以利户的第三次讲话：神的绝对公义（34:1-37）
  • 以利户的第四次讲话：神的不可测度与自然启示（35:1-37:24）

核心神学命题

1. 神的绝对公义：以利户宣告神不可能行不义，因为审判全地的主岂不行公义吗？（34:12） 2. 神的主权与人的责任：神有权柄施行审判，但人也有责任回应神的管教 3. 苦难的教育意义（深化）：以利户进一步强调苦难的目的是使人归正，使人听神的话（36:10-12） 4. 神的不可测度：以利户从神在自然中的作为出发，论证神的智慧超越人的理解（37:5, 14-16） 5. 为神从旋风中说话铺路：以利户的第四次讲话以对神在雷电、暴风、降雪中的作为的描述结束，直接引出神从旋风中说话（38:1）

上文回顾

在32-33章中，以利户完成了他的第一、第二次讲话。他引入了两个核心神学贡献：苦难的教育意义（33:19-30）和中保的概念（33:23-24）。以利户批评三友未能为神的公义做出有效的辩护，也回应了约伯「神在哪里？」的追问——神在梦中、在异象中、在苦难中向人说话。

以利户的前两次讲话主要回应了约伯的具体指控，而第三、第四次讲话则上升到更高的神学层面：从神的公义（34章），到神的超越性（35-37章）。这两次讲话将以利户的神学推向顶峰，也为神从旋风中说话做了最后的铺垫。

主题概述

以利户的第三、第四次讲话是他神学论述的高潮部分：

  • 34章（第三次讲话）：以利户从神的绝对公义出发，回应约伯对神公义的质疑。他宣告「审判全地的主岂不行公义吗？」（34:12），并论证神不可能行不义
  • 35章（第三次讲话续）：以利户回应约伯「敬虔无益」的论点，指出人的罪不会影响神，人的义也不会加添神什么
  • 36章（第四次讲话）：以利户进一步深化苦难的教育意义，强调神藉苦难使人归正，并开始描述神在自然中的作为
  • 37章（第四次讲话续）：以利户以对神在雷电、暴风、降雪中的作为的壮丽描述结束他的发言，将读者的注意力引向神的不可测度的智慧

第一部分：以利户的第三次讲话——神的绝对公义（34:1-37）

34:1-4 — 以利户呼吁众人聆听

「以利户又说：智慧人哪，你们要听我的话；有知识的人哪，你们要侧耳听我说。」

  • 希伯来文：「智慧人」（`חֲכָמִים`，`chakhamim`）；「有知识的人」（`יֹדְעֵי בִינָה`，`yod'ei vinah`）
  • 修辞功能：以利户不再只对约伯发言，而是向所有在场的人呼吁。这表明他即将讨论的是超越个人争论的普世神学命题
  • 关键经文（34:3-4）：「因为耳朵试验话语，好像上膛尝食物。我们当选择何为是，彼此知道何为善。」——以利户呼吁理性的判断，而非情感的偏见

34:5-9 — 以利户引述约伯的话

「约伯曾说：『我是公义，神夺去我的理。我虽有理，还算为说谎的；我虽无过，受的伤还不能医治。』」

  • 希伯来文：「理」（`מִשְׁפָּט`，`mishpat`）与27:2相同的词；「公义」（`צַדִּיק`，`tsaddiq`）
  • 论证策略：以利户首先准确引述约伯的话（34:5-6），然后再进行反驳。这延续了他在33:8-11使用的策略
  • 约伯的真实立场：约伯从未声称自己绝对无罪，他声称的是自己没有犯导致苦难的具体罪行。以利户在此略微扭曲了约伯的立场

34:10-15 — 神不可能行不义

「所以你们明理的人要听我的话。神断不至行恶，全能者断不至作孽。」

  • 希伯来文：「断不至」（`חָלִילָה`，`chalilah`）是强烈的否定表达，意为「绝对不可能！」；「行恶」（`רֶשַׁע`，`resha'`）；「作孽」（`עַוְלָה`，`avlah`）
  • 神学核心：这是以利户第三次讲话的核心宣言——神的绝对公义。神不可能行不义，因为公义是神本性的一部分
  • 关键经文（34:12）：「神决不至行恶，全能者决不至作孽。他必按人所做的报应人，使各人照所行的得报。」——以利户在此重新确认了因果报应的原则，但他的理解比三友更深
  • 神学深度：以利户的论点是：如果神是不公义的，那么整个宇宙秩序就会崩溃（34:13-15）。因此，约伯对神公义的质疑在逻辑上是不可能的

34:16-30 — 神的审判是公正的

「他使君王蒙羞被废，使绑绑的放松。」

  • 希伯来文：「蒙羞」（`חָלַף`，`chalaf`）意为「改变、更替」；「放松」（`אָסַר`，`'asar`）的反义
  • 神学分析：以利户从神对君王的审判出发，论证神的审判是公正的。神不偏待人，无论是君王还是平民，都按其行为受审
  • 关键经文（34:21-22）：「神注目观看人的道路，看明人的脚步。没有黑暗、阴翳能给作孽的藏身。」——以利户宣告神的全知——没有任何罪能逃避神的眼目
  • 与三友的区别：以利户与三友都坚持因果报应，但以利户的论证更系统、更深入。他不是简单地指控约伯有罪，而是从神的本性出发论证神不可能不公

34:31-37 — 约伯应该悔改

「人对神说：我受了责罚，不再犯罪。我所看不明的，求你指教我；我若作了孽，必不再作。」

  • 希伯来文：「责罚」（`נָשָׂא`，`nasa'）意为「承担、背负」；「指教」（`יָרָה`，`yarah`）意为「教导」
  • 以利户的劝告：以利户在此直接劝告约伯悔改。他认为约伯应该接受苦难的管教，承认自己的有限，而不是继续与神争辩
  • 叙事功能：以利户的劝告虽然有道理，但他仍然没有完全理解约伯的处境。约伯的苦难不是因为他犯了罪，而是因为神允许的试炼

第二部分：以利户继续回应约伯的质疑（35:1-16）

35:1-8 — 敬虔不会加添神什么

「以利户又说：你以为有理，或以为你的公义胜于神的公义，才说：『这与我有什么益处？我不犯罪比犯罪有什么好处呢？』」

  • 希伯来文：「益处」（`תּוֹעֶלֶת`，`to'elet`）意为「好处、利益」
  • 约伯的真实立场：约伯确实表达过类似的情绪（参9:21-22, 21:7-15），但以利户在此将约伯的情绪性表达简化为一个逻辑命题
  • 神学深度（35:6-8）：「你若犯罪，能使神受何害呢？你的过犯加增，能使神受何损呢？你若是公义，还能加增他什么呢？他从你手里还接受什么呢？」——以利户在此提出了一个深刻的神学洞见：人的行为不会影响神的本质。这挑战了「敬虔是为了利益」的功利主义信仰观
  • 与罗马书11:35的关联：保罗同样宣告「谁是先给了他，使他后来偿还呢？」

35:9-16 — 人因受欺压而呼求

「人因多受欺压就哀求，因受能者的辖制便求救。」

  • 希伯来文：「欺压」（`עֹשֶׁק`，`'osheq`）；「辖制」（`זְרוֹעַ`，`zeroa`，字面意为「膀臂」）
  • 神学分析：以利户承认人在苦难中会呼求，但他指出人往往是在绝望中呼求，而不是出于对神的真诚敬畏
  • 关键经文（35:14）：「何况你说，你不得见他；你的案件在他面前，你等候他吧！」——以利户在此回应约伯「神隐藏」的抱怨，劝约伯耐心等候

第三部分：以利户的第四次讲话——苦难的教育意义与神的超越性（36:1-37:24）

36:1-12 — 苦难的教育意义（深化）

「你再容我片时，我就指示你，因我还有话为神说。」

  • 希伯来文：「容我片时」（`כַּמְעַט`，`kam'a`）意为「稍等片刻」；「指示」（`יָדַע`，`yada'`）意为「使知道」
  • 关键经文（36:5-7）：「神有大能，并不藐视人，他的智慧甚广。他不保护恶人的性命，却为困苦人伸冤。他不离弃义人，却叫他们和君王同坐宝座，永远被高举。」——以利户在此提出了一个与三友不同的观点：神不藐视人，而是眷顾人
  • 核心经文（36:8-12）：「他们若被锁链捆住，被苦难的绳索缠住，他就把他们的作为和过犯指示他们，叫他们知道有骄傲的行动。他也开通他们的耳朵得受教训，吩咐他们离开罪孽转回。他们若听从事奉他，就必度日亨通，历年福乐。若不听从，就要被刀杀灭，无知无识而死。」
  • 神学核心：这是以利户苦难神学的最完整表达。苦难的教育过程是：（1）被锁链捆住 → （2）神指示他们的过犯 → （3）开通耳朵 → （4）吩咐离开罪孽 → （5）听从则亨通，不听从则灭亡
  • 与以利户前两次讲话的关联：33:19-30提出了苦难的教育功能，36:8-12将其深化为一个完整的神学体系

36:13-21 — 对约伯的警告

「那心中不敬虔的人积蓄怒气；神捆绑他们，他们竟不求救。」

  • 希伯来文：「积蓄怒气」（`חָנַף`，`chanaf`）意为「不敬虔、伪善」；「求救」（`שָׁוַע`，`shava'`）是本书的核心动词
  • 以利户的警告：以利户警告约伯不要走不敬虔之人的路——他们积蓄怒气，拒绝神的管教，最终走向灭亡
  • 关键经文（36:16-17）：「神也必引你出离患难，进入宽阔不狭窄之地；摆在你席上的必满有肥甘。但你满有审判和恶行的争论，审判和刑罚抓住你。」——以利户在此给约伯两条路：接受管教则得释放，拒绝管教则受刑罚

36:22-33 — 神在自然中的作为

「神行事有高大的能力，教训人的有谁像他呢？」

  • 希伯来文：「高大的能力」（`גָּדְלוֹ`，`godlo`）意为「他的伟大」；「教训」（`יָסַר`，`yasar`）与33:19相同的词
  • 文学转折：以利户从苦难的神学转向自然的神学。他开始描述神在自然中的作为，为37章的壮丽描述做铺垫
  • 关键经文（36:26）：「神为大，我们不能全知；他的年数不能测度。」——这是以利户第四次讲话的核心宣言：神的不可测度

37:1-13 — 神在雷电暴风中的作为

「因此我心战兢，从原处移动。听啊，神轰轰的声音，是他口中所发的响声。」

  • 希伯来文：「战兢」（`סָעַר`，`sa'ar`）意为「震动、暴风」；「轰轰」（`רָעַם`，`ra'am`）意为「雷鸣」
  • 文学结构：37章以一系列壮丽的自然现象来描述神的作为——雷电（37:1-5）、暴风（37:6-10）、云彩（37:11-13）、冰雪（37:6, 10）
  • 神学意义：以利户在此不是在做自然神学，而是在为神从旋风中说话做铺垫。自然现象是神的声音的外在表现
  • 关键经文（37:5）：「神发出奇妙的雷声，他行大事，我们不能测透。」——以利户再次强调神的不可测度

37:14-20 — 以利户呼吁约伯思想神的奇妙

「约伯啊，你要留心听，要站立思想神奇妙的作为。」

  • 希伯来文：「留心听」（`עָמַד`，`amad`）意为「站立」；「思想」（`הִתְבּוֹנֵן`，`hitbonen`）意为「深思」
  • 关键经文（37:16）：「你晓得神行这些事吗？他如何使云彩浮游，他奇妙的作为你晓得吗？」——以利户在此挑战约伯：你能否解释云彩的浮游？如果连自然现象都无法解释，怎能理解神的作为？
  • 与38-41章的关联：以利户的问题直接预示了神从旋风中将要提出的问题——「我立大地根基的时候，你在哪里呢？」（38:4）

37:21-24 — 以利户发言的结束

「现在有云遮蔽，人不得见穹苍的光亮；但风吹过，天又发亮。金光出于北方，在神那里有可怕的威严。论到全能者，我们不能测度；他大有能力，有公平和大义，必不苦待人。」

  • 希伯来文：「遮蔽」（`לֹא יֵרָאֶה`，`lo yera'eh`）直译为「不能被看见」；「威严」（`נוֹרָא`，`nora'）意为「可畏的」
  • 神学高潮：以利户以三个关键宣告结束他的发言：（1）神有「可怕的威严」；（2）神的智慧「不能测度」；（3）神「必不苦待人」
  • 叙事功能：以利户的发言以对神在暴风中的描述结束，直接引出神从旋风中说话（38:1）。以利户说「风吹过，天又发亮」，这与神「从旋风中回答」形成完美的叙事衔接

神学要点总结

1. 神的绝对公义

以利户在34章宣告神不可能行不义——「审判全地的主岂不行公义吗？」（34:12）。这一宣言回应了约伯对神公义的质疑，但也引发了新的问题：如果神是绝对公义的，为何义人受苦？

2. 苦难教育论的深化

以利户在36:8-12将他的苦难教育论深化为一个完整的神学体系。苦难的目的是使人归正，开通人的耳朵，使人听神的话。这一神学洞见超越了三友的因果论。

3. 神的不可测度

以利户在36:26和37:5两次宣告「神为大，我们不能全知」。这是以利户第四次讲话的核心主题——神的智慧超越人的理解。

4. 自然作为神的声音

以利户在37章描述了神在雷电、暴风、降雪中的作为。自然现象不是随机的，而是神的声音的外在表现。

5. 为神从旋风中说话铺路

以利户的第四次讲话以对神在暴风中的描述结束，直接引出神从旋风中说话。以利户说「风吹过，天又发亮」（37:21），这与神「从旋风中回答」（38:1）形成完美的叙事衔接。

6. 基督论预表：以利户描述的救赎主预表基督

以利户在36章中对神的描述——那位不藐视人、为困苦人伸冤、不离弃义人的神（36:5-7）——深刻地预表了基督的救赎工作。

一、以利户描述神"不保护恶人的性命，却为困苦人伸冤"（36:6）。这预表了基督的使命——"主的灵在我身上，因为他用膏膏我，叫我传福音给贫穷的人，差遣我报告被掳的得释放，瞎眼的得看见，叫那受压制的得自由"（路4:18）。基督来到世界，不是为义人，乃是为罪人（可2:17）。

二、以利户宣告神"不离弃义人，却叫他们和君王同坐宝座，永远被高举"（36:7）。这预表了基督的升高——"他本有神的形像，不以自己与神同等为强夺的，反倒虚己，取了奴仆的形像……所以神将他升为至高，又赐给他那超乎万名之上的名"（腓2:6-9）。基督经过十字架的苦难后，被高举坐在父神的右边（来1:3）。

三、以利户描述的苦难教育过程（36:8-12）——被捆绑、神指示过犯、开通耳朵、离开罪孽——预表了基督救赎工作的过程。人被罪捆绑（罗3:10-12），基督来"叫被掳的得释放"（路4:18）；人需要被指示罪的真相，基督是"世界的光"（约8:12）；人需要开通耳朵听神的话，基督是那位真正的教师（太23:10）；人需要离开罪孽，基督赐下圣灵使人"从罪里得了释放"（罗6:18）。

四、以利户在37:21-24中以"金光出于北方，在神那里有可怕的威严"结束他的发言。"金光出于北方"（זָהָב מִצָּפוֹן）在古代近东文化中可能指向神圣的荣耀显现。这预表了基督再来时的荣耀——"人子要在他父的荣耀里，同着众使者降临"（太16:27）。以利户说"神必不苦待人"（37:24），这指向基督来不是要定世人的罪，乃是要叫世人因他得救（约3:17）。

对当年读者的意义

历史处境中的理解

以利户的发言为约伯记的古代读者提供了一个新的苦难神学框架。在「因果报应」的文化中，苦难往往被理解为对罪的惩罚。以利户的「苦难教育论」挑战了这一观念——苦难可能是神的管教，目的是使人归正，而不是惩罚人。

以利户在约伯记神学中的位置

以利户的发言在约伯记的神学发展中起着承上启下的作用：

  • 对三友的超越：以利户超越了三友的因果论，提出了苦难的教育意义
  • 对约伯的回应：以利户回应了约伯「神隐藏」的追问，指出神在多种方式中向人说话
  • 为神的发言做铺垫：以利户的第四次讲话直接引出神从旋风中说话

古代近东背景

以利户的自然神学在古代近东文学中有广泛的平行文本。美索不达米亚的暴风雨之神阿达德（Adad）同样以雷电和暴风彰显能力。约伯记的独特之处在于：自然现象不是神本身，而是神的声音的外在表现。

当代应用

1. 苦难的再诠释：以利户的「苦难教育论」鼓励我们在苦难中寻找神的旨意，而不是简单地将其归因于罪 2. 神的不可测度：以利户的「神为大，我们不能全知」提醒我们保持谦卑——我们无法完全理解神的作为 3. 自然作为启示：以利户的自然神学提醒我们，神不仅在圣经中启示自己，也在自然中向人说话 4. 耐心等候：以利户劝约伯「等候神」（35:14），这一劝告对任何在苦难中等候神的人都有意义

默想引导

  • 以利户说「神断不至行恶」。你是否曾在苦难中质疑神的公义？这一宣告如何回应你的疑问？
  • 以利户描述了苦难的教育过程：被捆绑 → 神指示过犯 → 开通耳朵 → 离开罪孽。你是否曾在苦难中经历这样的过程？
  • 以利户说「神为大，我们不能全知」。你如何在不理解的情况下仍然信靠神？
  • 以利户的第四次讲话以对神在暴风中的描述结束。自然现象如何帮助你认识神的作为？

下文预告

以利户的发言结束后，神从旋风中说话（38:1）。神不再通过人类的中介发言，而是直接向约伯显现。神的第一次讲话（38-39章）从创造的奥秘出发，挑战约伯对宇宙秩序的理解；第二次讲话（40-41章）以河马和鳄鱼为例，展示神对受造世界的绝对主权。约伯的回应（42:1-6）将标志着整个书卷的神学高潮。
