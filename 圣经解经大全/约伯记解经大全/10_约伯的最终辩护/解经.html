第十章：约伯的最终辩护（29:1-31:40）

经文范围

约伯记 29:1-31:40，涵盖：

  • 约伯追忆往昔的荣光（29:1-25）
  • 约伯哀叹今日的屈辱（30:1-31）
  • 约伯的誓言——旧约最庄严的自我辩护（31:1-40）

核心神学命题

1. 约伯对往昔的追忆：约伯曾蒙神眷顾、受人尊敬、施行公义，如今却遭受彻底的逆转 2. 约伯对今日的哀叹：约伯不仅承受身体的痛苦，更承受社会的羞辱，甚至被最卑微的人藐视 3. 约伯的誓言：约伯以31章的庄严誓言向神发出挑战，逐条审查自己的行为，证明自己无辜 4. 苦难中的自我辩护：约伯的辩护不是出于骄傲，而是出于对公义的执着追求 5. 为神从旋风中说话铺路：约伯的最终辩护将他的案件推到神面前，为神的回应创造了叙事条件

上文回顾

在28章的智慧颂中，约伯宣告「敬畏耶和华就是智慧」，超越了三友的因果论智慧观。然而，约伯并没有因此放弃自己的诉求——他仍然渴望与神对话，仍然坚持自己的无辜。

29-31章是约伯的最终独白，也是他在神从旋风中说话之前的最后一次发言。这三章构成了约伯记诗歌部分的最高潮，也是旧约中最深刻的自我反思文本之一。约伯从追忆（29章）、到哀叹（30章）、到誓言（31章），完成了他对自己人生的全面审视。

主题概述

约伯的最终辩护分为三个部分：

  • 29章：约伯追忆往昔的荣光——他蒙神眷顾、受人尊敬、施行公义，是一个理想的古代近东族长形象
  • 30章：约伯哀叹今日的屈辱——他不仅承受身体的痛苦，更承受社会的彻底逆转，甚至被最卑微的人藐视
  • 31章：约伯发出旧约最庄严的誓言——他逐条审查自己的行为，从色情（31:1-4）、到欺诈（31:5-8）、到奸淫（31:9-12）、到虐待仆人（31:13-15）、到漠视穷人（31:16-23）、到贪财（31:24-25）、到拜偶像（31:26-28）、到幸灾乐祸（31:29-30）、到不款待客旅（31:31-32）、到隐藏罪恶（31:33-34），最后以一个挑战结束——约伯愿意将所有指控写下来，签上自己的名字，交给神审判

第一部分：约伯追忆往昔的荣光（29:1-25）

29:1-6 — 约伯与神同行的日子

「约伯又接着说：『惟愿我的景况如从前的月份，如神保守我的日子。』」

  • 希伯来文：「景况」（`כְּיַרְחֵי`，`keyarchei`）直译为「如月份」，指往昔的时光；「保守」（`שָׁמַר`，`shamar`）意为「看顾、保护」
  • 神学分析：约伯怀念的不仅是物质的丰富，更是与神的亲密关系——「那时他的灯照在我头上，我借他的光行过黑暗」（29:3）。这是一个深刻的神学洞见：真正的祝福不是物质，而是与神同在
  • 关键经文（29:5）：「全能者仍与我同在，我的儿女都环绕我。」——约伯将神的同在与家庭的完整并列，两者都是祝福的核心

29:7-17 — 约伯的社会地位

「我出到城门，在街上设立座位。」

  • 希伯来文：「城门」（`שַׁעַר`，`sha'ar`）是古代近东城市的公共空间，法庭和商业活动都在此进行
  • 古代近东背景：在城门口设座是族长或法官的特权，约伯在此描述的是一个理想的古代近东领袖形象——他不仅富有，更有社会影响力
  • 关键经文（29:12-13）：「因我拯救哀求的困苦人和无人帮助的孤儿。将要灭亡的为我祝福；我使寡妇心中欢乐。」——约伯的社会角色不仅是审判官，更是弱势群体的保护者
  • 与箴言31:8-9的关联：约伯的形象与箴言31章的理想君王形象相似——为不能自辩的人开口

29:18-25 — 约伯的期望

「我便说：我必死在家中，必增添我的日子如尘沙。」

  • 希伯来文：「死在家中」（`גָּוַע בְּקִנִּי`，`gava' beqinni`）直译为「在我的巢中死去」，是一个温暖的意象
  • 神学深度：约伯的期望并非出于骄傲，而是出于对神恩典的感恩。他期望在神的保守中安然度过一生
  • 关键经文（29:24-25）：「我向他们微笑，他们不自信；我脸上光华，他们不敢发忧。我为他们选择道路，又坐首位。」——约伯的社会权威建立在公义和智慧之上

第二部分：约伯哀叹今日的屈辱（30:1-31）

30:1-8 — 被最卑微的人藐视

「但如今，比我年少的人戏笑我；其人之父我曾藐视，不肯安在看守我羊群的狗中。」

  • 希伯来文：「戏笑」（`שָׂחַק`，`sachaq`）意为「嘲笑、戏弄」；「藐视」（`מָאַס`，`ma'as`）意为「拒绝、轻看」
  • 社会背景：约伯被社会最底层的人藐视——这些人的父辈甚至不配做他羊群的看守。这种社会逆转不仅是物质的，更是尊严的
  • 关键经文（30:9）：「现在这些人以我为歌曲，以我为笑谈。」——约伯的苦难成了别人的娱乐

30:10-19 — 身体的痛苦

「他们厌恶我，躲在旁边站着，不住地吐唾沫在我脸上。」

  • 希伯来文：「厌恶」（`תָּעַב`，`ta'av`）意为「憎恶、可憎」；「吐唾沫」（`יָרַק`，`yaraq`）在古代近东是极深的羞辱
  • 关键经文（30:16-17）：「现在我心极其悲伤，困苦的日子将我抓住。夜间我里面的骨头刺我，疼痛不止，好像啃我。」——约伯的身体痛苦与精神痛苦交织
  • 神学深度：约伯在此描述的不仅是身体的疾病，更是灵魂的破碎——他感受到神的「残忍」（30:21）

30:20-31 — 约伯向神呼求

「主啊，我呼求你，你不应允我；我站起来，你就定睛看我。」

  • 希伯来文：「呼求」（`שָׁוַע`，`shava'`）是恳切的呼喊；「定睛看」（`תָּמַר`，`tamar`）在此意为「严厉地注视」
  • 神学危机：约伯在此表达了他的核心困惑——神不仅不回应他的呼求，反而以严厉的目光注视他。这是约伯最深的痛苦：不是疾病，不是贫穷，而是神的沉默
  • 关键经文（30:23）：「我知道要使我临到死地，到那为众生所定的阴宅。」——约伯已经预期死亡的来临

第三部分：约伯的誓言——旧约最庄严的自我辩护（31:1-40）

31:1-4 — 色情之罪

「我与眼睛立约，怎能恋恋瞻望处女呢？」

  • 希伯来文：「立约」（`כָּרַת בְּרִית`，`karat berit`）是正式的契约；「恋恋瞻望」（`הִתְבּוֹנֵן`，`hitbonen`）意为「凝视、注目」
  • 古代近东背景：在古代近东，眼睛被视为欲望的入口。约伯与眼睛立约，是一种极端的自我约束
  • 神学深度：约伯的誓言从色情开始，这不是偶然的——色情之罪是最隐秘的罪，约伯以此证明他的纯洁从内到外

31:5-8 — 欺诈之罪

「我若与虚谎同行，我的脚若追随诡诈。」

  • 希伯来文：「虚谎」（`שָׁוְא`，`shav'`）意为「虚假、虚空」；「诡诈」（`מִרְמָה`，`mirmah`）意为「欺骗」
  • 修辞结构：约伯使用了「条件句+咒诅」的结构——「我若做了X，就愿Y临到我」。这是一种法庭上的宣誓形式

31:9-12 — 奸淫之罪

「我若受迷惑，向邻舍起淫心，在别人的妻子旁等候。」

  • 希伯来文：「起淫心」（`פָּתָה`，`patah`）意为「被引诱、迷惑」
  • 社会背景：奸淫在古代近东不仅是道德罪，更是对社会秩序的破坏——它侵犯了另一个家庭的完整性
  • 关键经文（31:11-12）：「因为这是大罪，是审判官当罚的罪孽。这本是焚烧到灭绝的火，必拔除我所有的家产。」——约伯承认奸淫是极其严重的罪

31:13-15 — 虐待仆人之罪

「我的仆婢与我争辩的时候，我若藐视不听他们的情节。」

  • 希伯来文：「藐视」（`מָאַס`，`ma'as`）与30:1同一个词，形成讽刺的呼应——约伯曾被藐视，但他从未藐视他人
  • 神学深度：约伯在此展示了古代近东极为罕见的伦理意识——即使是仆人，也有权向主人申诉
  • 关键经文（31:15）：「造我在腹中的，不也造他吗？将他与我抟在腹中的，岂不是一位吗？」——约伯从创造论的角度论证人的平等

31:16-23 — 漠视穷人之罪

「我若不容贫寒人得其所愿，或叫寡妇眼中失望。」

  • 希伯来文：「不容」（`מָנַע`，`mana'`）意为「阻止、扣留」
  • 关键经文（31:19-20）：「我若见人因无衣死亡，或见穷乏人身无遮盖；我若不使他因我羊的毛得暖，为我祝福。」——约伯的具体关怀延伸到最基本的生存需求
  • 与马太福音25:35-40的关联：约伯的关怀与耶稣的教导完全一致——对穷人的关怀就是对神的敬拜

31:24-25 — 贪财之罪

「我若以黄金为指望，对精金说：你是我的倚靠。」

  • 希伯来文：「指望」（`מִבְטָח`，`mivtach`）意为「信赖、依靠」
  • 神学深度：约伯在此区分了「拥有财富」和「倚靠财富」。约伯是富有的，但他从未将财富当作神

31:26-28 — 拜偶像之罪

「我若见太阳发光，明月行在空中，心就暗暗被引诱，口便亲手。」

  • 希伯来文：「暗暗被引诱」（`פְּתִי`，`peti`）意为「被引诱、迷惑」；「口便亲手」（`נָשַׁק לְיָד`，`nashaq leyad`）是拜日月的仪式
  • 古代近东背景：拜日月是古代近东普遍的宗教实践（参申4:19），约伯以此证明自己没有参与任何形式的偶像崇拜
  • 神学意义：约伯从色情到拜偶像，覆盖了人类罪恶的整个光谱——从最私密到最公开的罪

31:29-30 — 幸灾乐祸之罪

「我若见恨我的遭报就欢喜，见他遭灾便高兴。」

  • 希伯来文：「遭报」（`רָעָה`，`ra'ah`）意为「灾祸」；「高兴」（`גִּיל`，`gil`）意为「欢喜、快乐」
  • 神学深度：约伯在此审查的是一种极为隐蔽的罪——对仇敌的幸灾乐祸。这是一种「合法的恶」，因为仇敌确实遭了报应

31:31-32 — 不款待客旅之罪

「我若容我帐棚内的人说：何不有人以约伯的肉为饱足呢？」

  • 希伯来文：「帐棚内的人」（`אַנְשֵׁי אָהֳלִי`，`anshei 'oholi`）指家人和仆人
  • 古代近东背景：款待客旅是古代近东最神圣的义务之一（参创18章亚伯拉罕款待天使）

31:33-34 — 隐藏罪恶之罪

「我若像亚当遮掩我的过犯，将罪孽藏在怀中。」

  • 希伯来文：「亚当」（`אָדָם`，`'adam`）在此可能指「人类」（通用名词），也可能特指亚当
  • 神学深度：约伯在此与亚当形成对比——亚当在伊甸园中隐藏罪（创3:8），约伯则公开接受审查

31:35-40 — 约伯的最终挑战

「惟愿有一位肯听我！看哪，在这里有我所画的押，愿全能者回答我。」

  • 希伯来文：「听」（`שָׁמַע`，`shama'`）是约伯全书的核心诉求；「画押」（`תָּו`，`tav`）是希伯来字母的最后一个，象征签名
  • 神学高潮：约伯在此将他的案件呈交到神面前。他愿意将自己的誓言写成文书，签上名字，交给神审判
  • 关键经文（31:35-37）：「惟愿有一位肯听我！看哪，在这里有我所画的押，愿全能者回答我！愿那与我争辩的写下状词！我必将它带到肩上，又绑在头上为冠冕。」——约伯将控告书当作冠冕，这是一种极大的信心表达
  • 叙事功能：约伯的誓言将他的案件推到了神面前，为神从旋风中说话创造了叙事条件

神学要点总结

1. 约伯的品格一致性

约伯在29章追忆的往昔品格，与他在31章誓言中审查的行为完全一致。试炼并未摧毁他的道德品质。

2. 苦难中的自我辩护

约伯的自我辩护不是出于骄傲，而是出于对公义的执着追求。他愿意接受最严格的审查，证明自己的清白。

3. 约伯的社会关怀

约伯的誓言不仅审查了个人的道德行为，更审查了社会关怀——他对穷人、寡妇、孤儿的照顾。真正的敬虔不仅是个人的纯洁，更是社会的责任。

4. 约伯对神的诉求

约伯的最终挑战（31:35）将他的案件推到神面前。他不寻求三友的认同，只寻求神的回应。

5. 约伯与亚当的对比

约伯在31:33-34与亚当形成对比——亚当隐藏罪，约伯公开接受审查。约伯在某种意义上是「新亚当」，他经历了试炼却没有堕落。

对当年读者的意义

历史处境中的理解

约伯记的古代读者生活在一个「因果报应」的文化中。约伯的誓言挑战了这一文化——一个完全正直的人仍然可能遭受苦难。这对于那些在苦难中困惑的读者来说，是一种极大的安慰：苦难不是对罪的惩罚，而可能有更深层的意义。

约伯誓言的伦理价值

31章的誓言是古代近东最全面的伦理审查清单之一。它涵盖了色情、欺诈、奸淫、虐待、漠视穷人、贪财、拜偶像、幸灾乐祸、不款待客旅、隐藏罪恶等多个方面。这一清单对任何时代的读者都有伦理指导意义。

古代近东背景：与古代近东法庭程序的对比

约伯在29-31章中的最终辩护，其形式和结构与古代近东的法庭程序有着深刻的关联。

古代近东的法庭制度

在古代近东（尤其是美索不达米亚和古代以色列），城门口（`שַׁעַר`，sha'ar）是法庭审判的主要场所。约伯追忆自己"出到城门，在街上设立座位"（29:7），正是描述他作为审判官的身份。汉谟拉比法典（约公元前1754年）和中亚述法典（约公元前1076年）都规定了法庭程序：原告陈述、被告答辩、证人作证、法官判决。

约伯在31章中使用的"条件句+咒诅"形式（"我若做了X，就愿Y临到我"）与古代近东的法庭宣誓程序高度相似。在美索不达米亚的法律传统中，当事人可以通过"神圣的诅咒"（divine oath）来证明自己的清白——如果他们说谎，愿神明降罚。约伯的31章誓言正是这种传统的延伸，但其规模和系统性远超已知的任何古代近东法律文本。

《汉谟拉比法典》与约伯的自我辩护

《汉谟拉比法典》（约公元前1754年）包含了对各类犯罪的惩罚规定，其中多条与约伯在31章中审查的罪行相对应：

  • 偷盗和欺诈（31:5-8）：汉谟拉比法典第5-12条规定了对盗窃和虚假指控的惩罚。约伯以"我若与虚谎同行"起誓，表明他自愿接受比法律更严格的标准。
  • 奸淫（31:9-12）：法典第129条规定："若自由民之妻被捉与另一男子同寝，则应将此二人缚而投之于河。"约伯将奸淫视为"焚烧到灭绝的火"（31:12），与法典的严厉态度一致。
  • 虐待仆人（31:13-15）：法典第226-227条规定了对非法对待奴隶的惩罚。约伯更进一步，从创造论的角度论证主人与仆人的平等（31:15）——这超越了法典的法律框架，进入了神学领域。

古代近东的"无罪宣告"传统

在古代近东文学中，有一种特殊的"无罪宣告"（negative confession）传统，即在神明面前逐条声明自己没有犯过某类罪。最著名的例子是古埃及《亡灵书》第125章中死者在四十二位神明面前的自我辩护——死者必须逐条声明自己没有犯过特定的罪行（"我没有偷窃"、"我没有杀人"、"我没有说谎"等），才能通过审判进入来世。

约伯在31章中的誓言与这种"无罪宣告"传统有明显的平行关系。两者都是在神圣权威面前逐条审查自己的行为，两者都涵盖了广泛的道德领域（从个人行为到社会责任），两者都以接受审判者的最终裁决为结束。然而，约伯的辩护远比埃及的无罪宣告更加系统和全面——他不仅声明自己"没有做"某些事，还解释了他"为什么没有做"，并以创造神学（31:15）和对穷人的关怀（31:16-23）为基础。

约伯的"法律诉状"与古代近东的法律文书

约伯在31:35-37中要求将他的辩词"写下来"、"画押"、"带到肩上、绑在头上为冠冕"。这与古代近东的法律文书实践相呼应。在美索不达米亚，重要的法律判决会被刻在泥板上，当事人和证人都要"画押"（通常是印章）。汉谟拉比法典本身就是刻在石柱上，顶部是太阳神沙玛什（Shamash）授予汉谟拉比法律权柄的浮雕。

约伯的"画押"（`תָּו`，tav，希伯来字母表的最后一个字母）具有深刻的象征意义——他愿意将自己的全部人生记录下来，签上最终的名字，交给神审判。这不是草率的请求，而是经过深思熟虑的法律行动。在古代近东的语境中，这种"书面诉状"是对法庭程序的尊重——约伯不是在随意抱怨，而是在正式地提出诉讼。

与苏美尔"争论文学"的对比

苏美尔文学中有一类特殊的"争论文学"（disputation literature），如《夏与冬的争论》、《锄头与犁的争论》、《银子与铜的争论》等。这些文本通常以两个拟人化的事物争论谁更有价值开始，最后由一个权威做出裁决。约伯记在结构上与这种传统有相似之处——约伯与三友争论"义人是否应当受苦"，而最终的裁决者是神。

但约伯的最终辩护（29-31章）超越了苏美尔争论文学的范围。约伯不是在争论一个抽象的命题，而是在法庭上为自己的一生辩护。他的誓言涵盖了从色情（31:1-4）到拜偶像（31:26-28）的全部道德领域，这是已知古代近东法律文书中最全面的自我审查清单。

当代应用

1. 自我审查的勇气：约伯的誓言鼓励我们定期审查自己的行为——不是为了自义，而是为了保持对罪的警觉 2. 苦难中的坚持：约伯在极端痛苦中仍然持守自己的纯正，这为我们在困难时期提供了榜样 3. 社会关怀的深度：约伯的誓言不仅审查个人道德，更审查社会关怀。真正的敬虔必须延伸到对弱势群体的关怀 4. 向神申诉的自由：约伯的挑战（31:35）证明向神申诉不是不敬，而是一种信心的表达

默想引导

  • 你是否愿意像约伯一样，接受对自己行为的全面审查？你最害怕审查哪个方面？
  • 约伯在苦难中仍然持守纯正。你是否曾在苦难中想要放弃自己的原则？
  • 约伯将控告书当作冠冕（31:35-37）。你如何理解这种「以苦难为荣耀」的态度？
  • 约伯的社会关怀延伸到穷人、寡妇、孤儿。你的关怀是否也延伸到社会的边缘群体？

约伯的最终辩护结束后，辩论陷入了僵局。三友已经无话可说，约伯也已经将他的案件呈交到神面前。此时，一个年轻的声音——以利户——将从32章开始介入。以利户不同于三友，他强调苦难的教育意义，为神从旋风中说话做了重要的铺垫。
