大卫的哀歌 解经

一、诗篇背景

  • 经文范围：诗篇 3, 4, 5, 6, 7, 13, 22, 25, 28, 31, 35, 38, 39, 40, 42, 43, 51, 54, 55, 56, 57, 59, 61, 63, 64, 69, 70, 71, 86, 88, 102, 109, 130, 140, 141, 142, 143
  • 作者：大卫（绝大多数），诗88为可拉后裔希幔所写，诗102为困苦人的祷告
  • 历史背景：大卫一生经历无数苦难——逃避扫罗、逃避押沙龙、犯罪后的悔改、疾病、敌人的攻击
  • 写作目的：哀歌教导我们在苦难中如何向神呼求，从哀叹转向赞美

二、经文概览

大卫的哀歌共37篇，是诗篇中数量最多的一类。它们表达了大卫在各种苦难中向神的真实呼求——不是绝望的哭泣，而是信心的呼求。

哀歌的结构特征

1. 呼求（Invocation）：向神呼求，求神垂听 2. 申诉（Complaint）：描述苦难和困境 3. 信靠（Confidence）：表达对神的信靠 4. 祈求（Petition）：具体的祈求 5. 赞美（Praise）：转向赞美（不是每首哀歌都有）

三、逐节解经

1. 逃避扫罗的哀歌

诗篇3篇 — 早晨的祷告

"耶和华啊，我的敌人何其加增！"（3:1）

大卫逃避押沙龙时所写。虽四面受敌，大卫仍宣告："但你耶和华是我四围的盾牌，是我的荣耀，叫我抬起头来"（3:3）。他安然入睡，因神保佑他。

诗篇4篇 — 晚间的祷告

"你们应当畏惧，不可犯罪。"（4:4）

与诗3配对的晚间祷告。大卫劝勉众人要敬畏神。结尾宣告："我必安然躺下睡觉，因为独有你耶和华使我安然居住"（4:8）。

诗篇7篇 — 求神申冤

"耶和华我的神啊，我投靠你，求你救我脱离一切追赶我的人。"（7:1）

大卫因便雅悯人古益的话而写的哀歌。大卫求神按他的公义判断。

诗篇57篇 — 在翅膀下避难

"神啊，求你怜悯我，怜悯我！因为我的心投靠你。"（57:1）

大卫逃避扫罗时在洞中所写。虽在危险中，大卫仍宣告："神啊，我心坚定，我心坚定！我要唱诗，我要歌颂！"（57:7）。

诗篇142篇 — 在洞中的祷告

"我发声哀告耶和华，发声恳求耶和华。"（142:1）

大卫在洞中的祷告。"你是我在患难中的避难所"（142:5）。

2. 逃避押沙龙的哀歌

诗篇55篇 — 被朋友出卖

"原来不是仇敌辱骂我，若是仇敌，还可忍耐。"（55:12）

大卫被亲密朋友亚希多弗出卖时的哀歌。"你要把你的重担卸给耶和华，他必抚养你"（55:22），这是信心的宣告。

诗篇63篇 — 渴慕神

"神啊，你是我的神！我要切切地寻求你。"（63:1）

大卫在犹大旷野所写。"因你的慈爱比生命更好，我的嘴唇要颂赞你"（63:3）。

3. 犯罪后的悔改

诗篇6篇 — 病中的忏悔

"耶和华啊，求你不要在怒中责备我。"（6:1）

七篇忏悔诗之一。大卫在病中向神呼求。

诗篇51篇 — 大卫的忏悔

"神啊，求你按你的慈爱怜恤我。"（51:1）

最伟大的忏悔诗。大卫与拔示巴犯罪后所写。"求你为我造清洁的心，使我里面重新有正直的灵"（51:10）。

诗篇32篇 — 罪得赦免的福气

"得赦免其过、遮盖其罪的，这人是有福的。"（32:1）

忏悔诗。保罗在罗马书中引用（罗4:7-8）。

4. 疾病中的哀歌

诗篇13篇 — 信心的坚持

"耶和华啊，你忘记我要到几时呢？"（13:1）

简短的哀歌，从"要到几时"的追问，转向"但我倚靠你的慈爱"（13:5）的信心。

诗篇22篇 — 弥赛亚的受苦

"我的神，我的神，为什么离弃我？"（22:1）

最伟大的弥赛亚诗篇。预言基督在十字架上的受苦。

诗篇38篇 — 病中的忏悔

"耶和华啊，求你不要在怒中责备我。"（38:1）

大卫在病中的忏悔。"我的罪孽高过我的头，如同重担叫我担当不起"（38:4）。

诗篇39篇 — 人生的短暂

"耶和华啊，求你叫我晓得我的身世。"（39:4）

大卫在病中思想人生的短暂。"世人行动实系幻影，他们忙乱，真是枉然"（39:6）。

5. 被敌人攻击的哀歌

诗篇25篇 — 求神引导

"耶和华啊，我的心仰望你。"（25:1）

大卫的哀歌，求神引导他走正路。"求你将你的道指示我，将你的路教训我"（25:4）。

诗篇28篇 — 求神应允

"耶和华啊，我要求告你；我的磐石啊，不要向我缄默。"（28:1）

求神垂听祷告。"耶和华是我的力量，是我的盾牌"（28:7）。

诗篇31篇 — 投靠神

"耶和华啊，我投靠你，求你使我永不羞愧。"（31:1）

大卫在患难中投靠神。"我将我的灵魂交在你手里"（31:5），耶稣在十字架上引用此话（路23:46）。

诗篇35篇 — 求神申冤

"耶和华啊，与我相争的，求你与他们相争。"（35:1）

咒诅诗。求神为大卫申冤，惩罚他的仇敌。

诗篇64篇 — 求神保护

"神啊，我哀叹的时候，求你听我的声音。"（64:1）

求神保护脱离恶人的谋害。

诗篇69篇 — 大卫的哀歌

"神啊，求你救我！因为众水要淹没我。"（69:1）

大卫的哀歌。"他们拿苦胆给我当食物；我渴了，他们拿醋给我喝"（69:21），预言基督在十字架上的经历。

诗篇109篇 — 咒诅诗

"我所赞美的人，反而攻击我。"（109:4）

咒诅诗。大卫被敌人攻击，求神施行公义审判。

6. 其他哀歌

诗篇42-43篇 — 渴慕神

"神啊，我的心切慕你，如鹿切慕溪水。"（42:1）

可拉后裔的诗篇，表达对神的深切渴慕。

诗篇88篇 — 最黑暗的哀歌

"耶和华拯救我的神啊，我昼夜在你面前呼吁。"（88:1）

诗篇中最黑暗的哀歌。全诗以哀叹结束，没有转向赞美。

诗篇102篇 — 困苦人的祷告

"耶和华啊，求你听我的祷告。"（102:1）

困苦人的哀歌。"我如同旷野的鹈鹕"（102:6）。

诗篇130篇 — 从深处求告

"耶和华啊，我从深处向你求告。"（130:1）

忏悔诗。"主耶和华啊，你若究察罪孽，谁能站得住呢？但在你有赦免之恩"（130:3-4）。

诗篇143篇 — 求神引导

"耶和华啊，求你听我的祷告。"（143:1）

忏悔诗。"求你指教我遵行你的旨意"（143:10）。

四、神学主题

1. 真实的苦难：大卫不掩饰自己的痛苦，向神真实倾诉 2. 信心的呼求：哀歌不是绝望的哭泣，而是信心的呼求 3. 从哀叹到赞美：大多数哀歌从哀叹开始，以赞美结束 4. 神的信实：在苦难中经历神的信实 5. 神的公义：求神施行公义审判 6. 认罪与赦免：在神面前真诚认罪，经历赦免

五、新约关联

  • 诗6:1 → 罗9:13（神的主权）
  • 诗22:1 → 太27:46（十字架的呼喊）
  • 诗31:5 → 路23:46（耶稣的最后话语）
  • 诗32:1-2 → 罗4:7-8（因信称义）
  • 诗41:9 → 约13:18（犹大的出卖）
  • 诗69:21-22 → 约19:28-29（十字架的经历）

六、实际应用

1. 在苦难中向神呼求：学习大卫向神真实倾诉 2. 信靠神的信实：在困难中信靠神的保护 3. 真实认罪：在神面前真诚认罪，经历赦免 4. 从哀叹转向赞美：让信心带领我们从哀叹走向赞美 5. 等候神的拯救：忍耐等候神的时间

七、祷告与默想

耶和华我的神，我在急难中求告你，你就应允我。求你不要记念我的罪孽，按你的慈爱怜恤我。求你为我造清洁的心，使我里面重新有正直的灵。我要投靠在你翅膀的荫下，直到灾害过去。奉耶稣基督的名祷告，阿们。

"耶和华靠近伤心的人，拯救灵性痛悔的人。"（诗篇 34:18）
