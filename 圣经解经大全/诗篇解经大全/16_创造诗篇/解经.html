创造诗篇 解经

一、诗篇背景

  • 经文范围：诗篇 8, 19, 33, 65, 104, 136, 147, 148
  • 作者：大卫（诗8, 19, 33）、可拉后裔（诗65）、未注明作者（诗104, 136, 147, 148）
  • 历史背景：创造诗篇赞美神创造天地万物的大能，与创世记1-2章相呼应
  • 写作目的：教导信徒认识神的创造大能，在受造界中看见神的荣耀

二、经文概览

创造诗篇共8篇，涵盖了神创造的各个方面：天地万物、日月星辰、山川河流、动植物、人类。

三、逐节解经

1. 诗篇8篇 — 人的尊贵

"人算什么，你竟顾念他？"（8:4）

背景：赞美神的伟大和人的尊贵。

逐节分析：

  • 8:1-2 婴孩的能力："你因敌人的缘故，从婴孩和吃奶的口中建立了能力，使仇敌和报仇的闭口无言。"

  • 8:3-5 创造与人："我观看你指头所造的天，并你所陈设的月亮星宿，便说：人算什么，你竟顾念他？世人算什么，你眷顾他？你叫他比天使微小一点，并赐他荣耀尊贵为冠冕。"

  • 8:6-8 管理万物："你派他管理你手所造的，使万物，就是一切的羊牛、田野的兽、空中的鸟、海里的鱼，凡经行海道的，都服在他的脚下。"

神学意义：人虽渺小，神却赐他荣耀尊贵为冠冕，派他管理万物。

新约应验：希伯来书引用此诗（来2:6-8），说明基督道成肉身，成为人的代表。

2. 诗篇19篇 — 创造与律法的启示

"诸天述说神的荣耀，穹苍传扬他的手段。"（19:1）

背景：赞美创造和律法的双重启示。

逐节分析：

  • 19:1-6 创造的启示："诸天述说神的荣耀，穹苍传扬他的手段。这日到那日发出言语，这夜到那夜传出知识。无言无语，也无声音可听。它的量带通遍天下，它的言语传到地极。"

  • 19:7-11 律法的完美："耶和华的律法全备，能苏醒人心；耶和华的法度确定，能使愚人有智慧；耶和华的训词正直，能快活人的心；耶和华的命令清洁，能明亮人的眼目；耶和华的道理洁净，存到永远；耶和华的典章真实，全然公义。"

  • 19:12-14 认罪与祷告："谁能知道自己的错失呢？愿你赦免我隐而未现的过错。求你拦阻仆人不犯任意妄为的罪，不容这罪辖制我。耶和华我的磐石，我的救赎主啊，愿我口中的言语，心里的意念，在你面前蒙悦纳。"

神学意义：神通过创造和律法两方面启示自己。

3. 诗篇33篇 — 赞美创造主

"诸天借耶和华的命而造，万象借他口中的气而成。"（33:6）

背景：赞美神的创造和护理。

逐节分析：

  • 33:1-3 赞美："义人哪，你们应当靠耶和华欢乐。正直人的赞美是合宜的。你们当弹琴称谢耶和华，用十弦瑟歌颂他。当向他唱新歌，弹得巧妙，声音洪亮。"

  • 33:4-7 创造："因为耶和华的言语正直，凡他所做的尽都诚实。他喜爱仁义公平，遍地满了耶和华的慈爱。诸天借耶和华的命而造，万象借他口中的气而成。"

  • 33:8-12 列邦的回应："列国啊，应当敬畏耶和华！世上的居民哪，都当畏惧他！"

  • 33:13-19 神的眷顾："耶和华从天上观看，他看见一切的世人。"

  • 33:20-22 信靠："我们的心向来等候耶和华，他是我们的帮助，我们的盾牌。"

神学意义：神创造万物，也眷顾万有。

4. 诗篇65篇 — 赞美神的供应

"你以恩典为年岁的冠冕，你的路径都滴下脂油。"（65:11）

背景：可拉后裔的诗篇，赞美神的供应。

逐节分析：

  • 65:1-4 赞美："神啊，锡安的人都等候赞美你。所许的愿也要向你偿还。你听了祷告。"

  • 65:5-8 神的大能："拯救我们的神啊，你必以威严秉公义应允我们。你是一切地极和海上远处的人所倚靠的。"

  • 65:9-13 丰收："神啊，你眷顾地，降下透雨，使地大得肥美。你以恩典为年岁的冠冕，你的路径都滴下脂油。滴在旷野的草场上。小山以欢乐束腰，草场以羊群为衣，谷中也长满了五谷。这一切都欢呼歌唱。"

神学意义：神不仅创造万物，也持续供应万物。

5. 诗篇104篇 — 创造的赞美

"我的心哪，你要称颂耶和华！耶和华我的神啊，你为至大。"（104:1）

背景：赞美神的创造，与创世记1章相对应。

逐节分析：

  • 104:1-4 光与天空："你以尊荣威严为衣服，披上亮光，如披外袍。你铺张穹苍如铺幔子。"

  • 104:5-9 陆地与海洋："你将大地立在根基上，使地永不动摇。你用深水遮盖地面，犹如衣裳。"

  • 104:10-18 植物与动物："他使泉源涌在山谷，流在山间。野地的走兽在那里喝。"

  • 104:19-23 日月昼夜："你安置月亮管理黑夜，太阳自知沉落。"

  • 104:24-30 海洋与生物："耶和华啊，你所造的何其多！都是你用智慧造成的，遍地满了你的丰富。"

  • 104:31-35 赞美："愿耶和华的荣耀存到永远，愿耶和华喜悦自己所造的。"

神学意义：神创造万物，也持续护理万物。

6. 诗篇136篇 — 他的慈爱永远长存

"你们要称谢耶和华，因他本为善，他的慈爱永远长存。"（136:1）

背景：著名的感恩诗，每节都以"他的慈爱永远长存"结束。

逐节分析：

  • 136:1-3 赞美："你们要称谢耶和华，因他本为善，他的慈爱永远长存。你们要称谢万神之神，因他的慈爱永远长存。你们要称谢万主之主，因他的慈爱永远长存。"

  • 136:4-9 创造："称谢独行大奇事的，因他的慈爱永远长存。称谢用智慧造天的，因他的慈爱永远长存。称谢铺地在水以上的，因他的慈爱永远长存。称谢造成大光的，因他的慈爱永远长存。他造日头管白昼，因他的慈爱永远长存。他造月亮星宿管黑夜，因他的慈爱永远长存。"

  • 136:10-22 历史："称谢那击杀埃及人之长子的，因他的慈爱永远长存。"

  • 136:23-26 感恩："他顾念我们在卑微的地步，因他的慈爱永远长存。他救拔我们脱离敌人，因他的慈爱永远长存。"

神学意义：神的创造和历史中的作为，都彰显他的慈爱。

7. 诗篇147篇 — 赞美神的慈爱

"你们要赞美耶和华！因歌颂我们的神为善为美。"（147:1）

背景：赞美神的慈爱和大能。

逐节分析：

  • 147:1-6 神的眷顾："耶和华扶持被压下的，将恶人倾覆于地。"

  • 147:7-11 创造与护理："他用云遮天，为地降雨，使草生长在山上。他赐食给走兽和啼叫的小乌鸦。"

  • 147:12-14 锡安的赞美："耶路撒冷啊，你要颂赞耶和华！锡安哪，你要赞美你的神！"

  • 147:15-18 他的命令："他发命在地，他的话颁行最快。"

  • 147:19-20 他的话："他将他的道指示雅各，将他的律例典章指示以色列。"

神学意义：神不仅创造万物，也赐下他的话语。

8. 诗篇148篇 — 万有赞美耶和华

"你们要从天上赞美耶和华！"（148:1）

背景：邀请天上地上一切受造之物赞美神。

逐节分析：

  • 148:1-6 天上的赞美："你们要从天上赞美耶和华，在高处赞美他。他的众使者都要赞美他，他的诸军都要赞美他。日头月亮，你们要赞美他。放光的星宿，你们要赞美他。天上的天和天上的水，你们都要赞美他。"

  • 148:7-12 地上的赞美："大山和小山，结果的树木和一切香柏树，野兽和一切牲畜，昆虫和飞鸟，世上的君王和万民，首领和世上一切审判官，少年人和处女，老年人和孩童，你们都要赞美耶和华。"

  • 148:13-14 赞美的原因："愿这些都赞美耶和华的名！因为他荣耀的名独当赞美。他的威严高过天地。"

  • 148:14 他的百姓："他将他百姓的角高举，因此他一切圣民以色列人，就是与他相近的百姓，都赞美他。你们要赞美耶和华！"

神学意义：天上地上一切受造之物都要赞美创造主。

四、神学主题

1. 神的创造大能：神创造天地万物 2. 神的护理：神持续供应和维系万物 3. 人的尊贵：人虽渺小，神却赐他荣耀尊贵 4. 创造的启示：诸天述说神的荣耀 5. 万有的赞美：一切受造之物都要赞美创造主

五、新约关联

  • 诗8:4-6 → 来2:6-8（基督的尊贵）
  • 诗19:1 → 罗1:20（神的永能和神性）
  • 诗33:6 → 约1:3（万物是借着他造的）
  • 诗104:24 → 西1:16（万有都是靠他造的）
  • 诗136:6 → 西1:17（万有也靠他而立）
  • 诗147:4 → 赛40:26（数点星宿）

六、实际应用

1. 在自然中看见神：在受造界中看见神的荣耀 2. 敬畏创造主：认识神的创造大能，生发敬畏 3. 管理万物的责任：作为神的管家，善待受造界 4. 赞美创造主：在受造界中赞美创造主 5. 信靠供应者：信靠神的持续供应

七、祷告与默想

耶和华我的神，你为至大，你以尊荣威严为衣服。诸天述说你的荣耀，穹苍传扬你的手段。我观看你指头所造的天，便说：人算什么，你竟顾念我？你赐我荣耀尊贵为冠冕，派我管理你手所造的。愿你荣耀存到永远。奉耶稣基督的名祷告，阿们。

"诸天述说神的荣耀，穹苍传扬他的手段。"（诗篇 19:1）
