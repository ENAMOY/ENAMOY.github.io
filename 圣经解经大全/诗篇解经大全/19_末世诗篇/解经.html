末世诗篇 解经

一、诗篇背景

  • 经文范围：诗篇 2, 46, 47, 48, 50, 72, 93, 96, 97, 98, 99, 110, 145, 149, 150
  • 作者：大卫（诗2, 50, 145）、可拉后裔（诗46, 47, 48）、亚萨（诗50）、所罗门（诗72）、未注明作者（诗93, 96-99, 110, 149, 150）
  • 历史背景：末世诗篇（Eschatological Psalms）是关于末世和神国度的诗篇，预言神最终的审判和国度的实现
  • 写作目的：教导信徒盼望神的国度，认识末世的真理

二、经文概览

末世诗篇共15篇，涵盖了末世的各个方面：神的国度、神的审判、万民的归向、最终的赞美。

末世的主题

三、逐节解经

1. 诗篇2篇 — 弥赛亚的国度

"你是我的儿子，我今日生你。"（2:7）

背景：第一篇弥赛亚诗篇，预言弥赛亚的国度。

逐节分析：

  • 2:1-3 列邦的反抗："外邦为什么争闹？万民为什么谋算虚妄的事？"

  • 2:4-6 神的嗤笑："那坐在天上的必发笑，主必嗤笑他们。"

  • 2:7-9 受膏者的宣告："你是我的儿子，我今日生你。你求我，我就将列国赐你为基业。"

  • 2:10-12 劝勉："你们世上的君王当存畏惧侍奉耶和华。当以嘴亲子。"

新约应验：基督的国度将在末世完全实现。

2. 诗篇46篇 — 神的国度的平安

"你们要休息，要知道我是神。"（46:10）

背景：可拉后裔的诗篇，预言神的国度的平安。

逐节分析：

  • 46:1-3 不害怕："神是我们的避难所，是我们的力量。"

  • 46:4-5 神的城："有一道河，这河的分汊使神的城欢喜。"

  • 46:8-11 神的作为："你们来看耶和华的作为。你们要休息，要知道我是神。"

神学意义：在末世，神的国度将带来真正的平安。

3. 诗篇47篇 — 神作王

"你们当向神欢呼，当用向雅各的神发声欢乐。"（47:1）

背景：可拉后裔的诗篇，宣告神作王。

逐节分析：

  • 47:1-4 神的王权："耶和华至高者是可畏的，他是治理全地的大君王。"

  • 47:5-7 赞美："神上升，有喊声相送；耶和华上升，有角声相送。"

  • 47:8-9 万民归向："神作王治理万邦，神坐在他的圣宝座上。"

神学意义：在末世，神将作王治理万邦。

4. 诗篇48篇 — 锡安的荣耀

"耶和华本为大，在我们神的城中，在他的圣山上，该受大赞美。"（48:1）

背景：可拉后裔的诗篇，赞美锡安的荣耀。

逐节分析：

  • 48:1-3 锡安的美丽："锡安山，大君王的城，在北面居高华美。"

  • 48:4-8 神的保护："众王会合，一同经过。他们见了这城，就惊奇丧胆。"

  • 48:9-11 神的慈爱："神啊，我们在你的殿中想念你的慈爱。"

  • 48:12-14 锡安的荣耀："你们当周游锡安，四围旋绕。"

神学意义：在末世，锡安将成为万民归向的中心。

5. 诗篇50篇 — 神的审判

"大能者神耶和华，已经发言招呼天下。"（50:1）

背景：亚萨的诗篇，宣告神的审判。

逐节分析：

  • 50:1-4 神的显现："大能者神耶和华，已经发言招呼天下，从日出之地到日落之处。"

  • 50:5-6 神的审判："招聚我的圣民到我这里来，就是那些用祭物与我立约的人。"

  • 50:7-15 真正的祭物："我并不因你的祭物责备你，你的燔祭常在我面前。"

  • 50:16-21 恶人的罪："恶人，你怎可述说我的律例，口中提到我的约呢？"

  • 50:22-23 感恩的祭："凡以感谢献上为祭的便是荣耀我。"

神学意义：在末世，神将施行审判，按公义审判世界。

6. 诗篇72篇 — 弥赛亚的国度

"他的名要存到永远。"（72:17）

背景：所罗门的诗篇，描写弥赛亚的国度。

逐节分析：

  • 72:1-4 公义的统治："他要按公义审判你的民。"

  • 72:5-7 万民敬畏："太阳还存，月亮还在，人要敬畏你，直到万代。"

  • 72:8-11 广大的国度："他要执掌权柄，从这海直到那海。"

  • 72:12-14 拯救贫穷："穷乏人呼求的时候，他要搭救。"

  • 72:15-17 永远的名："他的名要存到永远，要留传如日之久。"

神学意义：在末世，弥赛亚的国度将完全实现。

7. 诗篇93篇 — 耶和华作王

"耶和华作王，他以威严为衣穿上。"（93:1）

背景：宣告神的王权。

逐节分析：

  • 93:1-2 神的王权："耶和华作王，他以威严为衣穿上。"

  • 93:3-4 神的大能："耶和华在高处大有能力，胜过诸水的响声。"

  • 93:5 神的法度："耶和华啊，你的法度最确定。"

神学意义：在末世，耶和华将作王直到永远。

8. 诗篇96-99篇 — 新歌与审判

诗篇96篇 — 新歌

"你们要向耶和华唱新歌！"（96:1）

邀请全地向耶和华唱新歌。"他要来审判世界，他要按公义审判世界"（96:13）。

诗篇97篇 — 耶和华作王

"耶和华作王，愿地快乐！"（97:1）

赞美神的王权。"诸天表明他的公义，万民看见他的荣耀"（97:6）。

诗篇98篇 — 新歌

"你们要向耶和华唱新歌！因为他行过奇妙的事。"（98:1）

赞美神的拯救。"他要来审判世界"（98:9）。

诗篇99篇 — 耶和华作王

"耶和华作王，万民当战抖！"（99:1）

赞美神的圣洁。"耶和华在锡安为大"（99:2）。

9. 诗篇110篇 — 弥赛亚的王权

"耶和华对我主说：你坐在我的右边。"（110:1）

背景：最重要的弥赛亚诗篇。

逐节分析：

  • 110:1 在神右边："耶和华对我主说：你坐在我的右边，等我使你仇敌作你的脚凳。"

  • 110:2-3 永远的王权："耶和华必使你从锡安伸出能力的杖来。"

  • 110:4 祭司职分："你是照着麦基洗德的等次永远为祭司。"

  • 110:5-7 得胜："在你右边的主，当他发怒的日子，必打伤列王。"

神学意义：在末世，基督将完全实现他的王权。

10. 诗篇145篇 — 神的国度

"我的神、我的王啊，我要尊崇你。"（145:1）

背景：大卫的赞美诗。

逐节分析：

  • 145:1-7 赞美："耶和华本为大，该受大赞美，其大无法测度。"

  • 145:8-13 神的国度："耶和华有恩惠，有怜悯，不轻易发怒，大有慈爱。你的国是永远的国，你执掌的权柄存到万代。"

  • 145:14-21 神的眷顾："耶和华扶持一切被压下的人。"

神学意义：神的国度是永远的国度，他的权柄存到万代。

11. 诗篇149-150篇 — 最终的赞美

诗篇149篇 — 新歌

"你们要向耶和华唱新歌！"（149:1）

赞美诗。"愿以色列因造他的主欢喜"（149:2）。

诗篇150篇 — 你们要赞美耶和华

"你们要赞美耶和华！"（150:1）

诗篇的最终高潮。六次呼唤"赞美耶和华"。"凡有气息的都要赞美耶和华！你们要赞美耶和华！"（150:6）。

神学意义：在末世，万有都要赞美耶和华。

四、神学主题

1. 神的国度：神的国度是永远的国度 2. 神的审判：神将按公义审判世界 3. 弥赛亚的王权：基督将完全实现他的王权 4. 万民的归向：万民将归向神 5. 最终的赞美：万有都要赞美耶和华

五、新约关联

  • 诗2:7 → 徒13:33（基督为神的儿子）
  • 诗47:8 → 启11:15（基督作王）
  • 诗72:8 → 启11:15（基督的国度）
  • 诗96:13 → 太25:31-32（最后的审判）
  • 诗110:1 → 来1:13（基督在神右边）
  • 诗145:13 → 启11:15（永远的国度）
  • 诗150:6 → 启5:13（万有的赞美）

六、实际应用

1. 盼望神的国度：在地上盼望神的国度 2. 信靠神的审判：相信神必按公义审判世界 3. 敬拜基督为王：基督是万王之王 4. 传扬福音：万民将归向神 5. 赞美的人生：以赞美为生命的高峰

七、祷告与默想

耶和华我的神，你的国度是永远的国度，你执掌的权柄存到万代。求你帮助我在地上盼望你的国度，信靠你的审判。求你帮助我敬拜基督为王，传扬你的福音。凡有气息的都要赞美耶和华！你们要赞美耶和华！奉耶稣基督的名祷告，阿们。

"耶和华作王，愿地快乐！"（诗篇 97:1）
