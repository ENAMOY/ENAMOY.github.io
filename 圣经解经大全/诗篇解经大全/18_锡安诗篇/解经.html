锡安诗篇 解经

一、诗篇背景

  • 经文范围：诗篇 46, 48, 76, 84, 87, 122, 125, 132
  • 作者：可拉后裔（诗46, 48, 84, 87）、亚萨（诗76）、大卫（诗122, 125, 132）
  • 历史背景：锡安诗篇是关于耶路撒冷和圣殿的诗篇，锡安是耶路撒冷的山名，是神同在的象征
  • 写作目的：教导信徒认识圣殿的意义，渴慕神的同在

二、经文概览

锡安诗篇共8篇，涵盖了关于锡安的各个方面：神的保护、圣殿的荣耀、朝圣的喜乐、永远的安息。

锡安的主题

主题诗篇
神的保护46, 48, 76, 125
圣殿的荣耀48, 84, 87

永远的安息132

三、逐节解经

1. 诗篇46篇 — 神是我们的避难所

"神是我们的避难所，是我们的力量。"（46:1）

背景：可拉后裔的诗篇，赞美神在锡安的保护。

逐节分析：

  • 46:1-3 不害怕："神是我们的避难所，是我们的力量，是我们在患难中随时的帮助。所以地虽改变，山虽摇动到海心，其中的水虽砰訇翻腾，山虽因海涨而战抖，我们也不害怕。"

  • 46:4-5 神的城："有一道河，这河的分汊使神的城欢喜，这城就是至高者居住的圣所。神在其中，城必不动摇。"

  • 46:6-7 万邦的喧嚷："万邦喧嚷，列国动摇。他发声，地便熔化。万军之耶和华与我们同在。"

  • 46:8-11 你们要休息："你们来看耶和华的作为。你们要休息，要知道我是神。我必在外邦中被尊崇，在遍地上也被尊崇。"

神学意义：神在锡安中保护他的百姓，即使天地动摇，我们也不害怕。

2. 诗篇48篇 — 锡安的荣耀

"耶和华本为大，在我们神的城中，在他的圣山上，该受大赞美。"（48:1）

背景：可拉后裔的诗篇，赞美锡安的荣耀。

逐节分析：

  • 48:1-3 锡安的美丽："耶和华本为大，在我们神的城中，在他的圣山上，该受大赞美。锡安山，大君王的城，在北面居高华美，为全地所喜悦。"

  • 48:4-8 神的保护："众王会合，一同经过。他们见了这城，就惊奇丧胆，急忙逃跑。"

  • 48:9-11 神的慈爱："神啊，我们在你的殿中想念你的慈爱。神啊，你受的赞美正与你的名相称，直到地极。"

  • 48:12-14 锡安的荣耀："你们当周游锡安，四围旋绕，数点城楼，细看他的外郭，察看他的宫殿。"

神学意义：锡安是神的城，彰显神的荣耀和慈爱。

3. 诗篇76篇 — 神在锡安得胜

"神在犹大为人所认识，在以色列名为大。"（76:1）

背景：亚萨的诗篇，赞美神在锡安的得胜。

逐节分析：

  • 76:1-3 神在锡安："神在犹大为人所认识，在以色列名为大。在撒冷有他的帐幕，在锡安有他的居所。"

  • 76:4-6 神的威严："你比猎食的山峰更有荣华、更有能力。心中勇敢的人都被抢夺。"

  • 76:7-9 神的审判："惟独你是可畏的。你怒气一发，谁能在你面前站得住呢？"

  • 76:10-12 神的得胜："人的忿怒要成全你的荣美，人的余怒你要禁止。"

神学意义：神在锡安彰显他的威严和得胜。

4. 诗篇84篇 — 渴慕神的院宇

"万军之耶和华啊，你的居所何等可爱！"（84:1）

背景：可拉后裔的诗篇，表达对圣殿的渴慕。

逐节分析：

  • 84:1-4 渴慕："万军之耶和华啊，你的居所何等可爱！我羡慕渴想耶和华的院宇，我的心肠、我的肉体向永生神呼吁。万军之耶和华，我的王、我的神啊，在你祭坛那里，麻雀为自己找着房屋，燕子为自己找着抱雏之窝。"

  • 84:5-7 朝圣的旅程："靠你有力量、心中想往锡安大道的，这人便为有福。他们经过流泪谷，叫这谷变为泉源之地，并有秋雨之福盖满了全谷。他们行走，力上加力，各人到锡安朝见神。"

  • 84:8-12 神的恩惠："耶和华万军之神啊，求你听我的祷告。神啊，求你留心垂听。"

神学意义：渴慕神的同在，如同鹿切慕溪水。

5. 诗篇87篇 — 锡安的荣耀

"耶和华所爱的门，他爱锡安的门。"（87:2）

背景：可拉后裔的诗篇，赞美锡安的荣耀。

逐节分析：

  • 87:1-3 锡安的地位："耶和华所立的根基在圣山上。他爱锡安的门，胜于爱雅各一切的住处。神的城啊，有荣耀的事乃指着你说的。"

  • 87:4-6 万民归向："论到拉哈伯和巴比伦，他们是在认识我的人之中的。看哪，非利士和推罗并古实人，这人那人生在那里。"

  • 87:7 歌唱："歌唱的、跳舞的都要说：我的泉源都在你里面。"

神学意义：锡安是万民归向的地方，是神同在的象征。

6. 诗篇122篇 — 为耶路撒冷求平安

"人对我说：我们往耶和华的殿去，我就欢喜。"（122:1）

背景：大卫的诗，朝圣者的诗歌。

逐节分析：

  • 122:1-2 朝圣的喜乐："人对我说：我们往耶和华的殿去，我就欢喜。耶路撒冷啊，我们的脚站在你的门内。"

  • 122:3-5 耶路撒冷的美丽："耶路撒冷被建造如同联络整齐的一座城。众支派，就是耶和华的支派，上那里去，按以色列的常例称赞耶和华的名。"

  • 122:6-9 为耶路撒冷求平安："你们要为耶路撒冷求平安。耶路撒冷啊，爱你的人必然兴旺。愿你城中平安，愿你宫内兴旺。因我弟兄和同伴的缘故，我要说：愿平安在你中间。因耶和华我们神殿的缘故，我要为你求福。"

神学意义：为圣城求平安，是信徒的责任。

7. 诗篇125篇 — 信靠的保护

"倚靠耶和华的人好像锡安山，永不动摇。"（125:1）

背景：上行之诗。

逐节分析：

  • 125:1-2 信靠的稳固："倚靠耶和华的人好像锡安山，永不动摇。众山怎样围绕耶路撒冷，耶和华也照样围绕他的百姓，从今时直到永远。"

  • 125:3-5 公义的保护："恶人的杖不常落在义人的份上。耶和华啊，求你善待那些为善和心里正直的人。"

神学意义：倚靠神的人永不动摇，因为神围绕保护他们。

8. 诗篇132篇 — 记念大卫

"耶和华啊，求你记念大卫所受的一切苦难。"（132:1）

背景：君王诗篇，记念大卫将约柜安放在锡安。

逐节分析：

  • 132:1-5 大卫的誓愿："耶和华啊，求你记念大卫所受的一切苦难。他怎样向耶和华起誓，向雅各的大能者许愿。"

  • 132:6-9 约柜的安放："我们听说约柜在以法他，我们在基列耶琳就寻见了。我们要进他的居所，在他脚凳前下拜。"

  • 132:10-12 神的应许："求你为你仆人大卫的缘故，不要厌弃你的受膏者。耶和华向大卫凭诚实起了誓，必不反复，说：我要使你所生的坐在你的宝座上。"

  • 132:13-18 锡安的福气："耶和华拣选了锡安，愿意当作自己的居所。这是我永远安息之所，我要住在这里。"

神学意义：神拣选锡安作为他永远的居所，应许大卫的后裔永远作王。

四、神学主题

1. 神的同在：锡安是神同在的象征 2. 神的保护：神在锡安保护他的百姓 3. 圣殿的荣耀：圣殿彰显神的荣耀 4. 朝圣的喜乐：渴慕神的同在 5. 永远的安息：锡安是永远安息之所

五、新约关联

  • 诗46:10 → 可4:39（平静风浪）
  • 诗48:1-2 → 来12:22（锡安山）
  • 诗84:7 → 林后4:16-17（力上加力）
  • 诗87:5 → 弗2:19-22（神的家）
  • 诗122:6 → 罗10:1（为以色列求平安）
  • 诗132:11 → 路1:32-33（基督的王位）

六、实际应用

1. 渴慕神的同在：渴慕在圣殿中与神相遇 2. 为教会求平安：为神的教会求平安 3. 信靠神的保护：倚靠神永不动摇 4. 记念神的应许：相信神必成就他的应许 5. 朝圣的人生：我们的人生如同朝圣旅程

七、祷告与默想

耶和华我的神，你的居所何等可爱！我羡慕渴想耶和华的院宇。求你帮助我渴慕你的同在，如同鹿切慕溪水。愿我城中平安，愿我宫内兴旺。因耶和华我们神殿的缘故，我要为你求福。奉耶稣基督的名祷告，阿们。
