卷五：诗篇107-150篇 解经

一、诗篇背景

  • 经文范围：诗篇 107:1 - 150:6
  • 作者：大卫（诗108-110, 122, 124, 131, 133, 138-145）、所罗门（诗127）、未注明作者（诗107, 111-121, 123, 125-126, 128-130, 132, 134-137, 146-150）
  • 历史背景：本卷是诗篇最长的一卷，可能在被掳归回后编辑成册，包含朝圣诗篇（120-134）和哈利路亚诗篇（146-150）
  • 写作目的：如同申命记是摩西五经的总结，卷五是诗篇的总结和高峰

二、经文概览

卷五包含44篇诗篇，是诗篇中最长的一卷。核心主题是感恩与赞美、神的话语以及哈利路亚的颂赞。本卷以感恩开始，以哈利路亚结束。

结构大纲

三、逐节解经

1. 感恩之歌（诗107篇）

诗篇107篇 — 从各方拯救

"你们要称谢耶和华，因他本为善，他的慈爱永远长存。"（107:1）

感恩诗篇。描述神从四方拯救他的百姓：从旷野（107:4-9）、从监牢（107:10-16）、从疾病（107:17-22）、从风浪（107:23-32）。四次重复"但愿人因耶和华的慈爱和他向人所行的奇事都称赞他"（107:8, 15, 21, 31）。

2. 大卫的诗（诗108-110篇）

诗篇108篇 — 神啊，我心坚定

"神啊，我心坚定，我口要唱诗歌颂。"（108:1）

大卫的感恩诗，由诗57:7-11和诗60:5-12合并而成。

诗篇109篇 — 咒诅诗

"我所赞美的人，反而攻击我。"（109:4）

咒诅诗。大卫被敌人攻击，求神施行公义审判。犹大用三十块钱出卖耶稣的预言可能与此诗有关（109:8）。

诗篇110篇 — 弥赛亚的王权

"耶和华对我主说：你坐在我的右边。"（110:1）

最重要的弥赛亚诗篇之一。耶稣引用此诗证明他是大卫的主（太22:44）。"你是照着麦基洗德的等次永远为祭司"（110:4），新约希伯来书详细阐述基督照此等次为大祭司（来5:6）。

3. 赞美与智慧（诗111-119篇）

诗篇111篇 — 赞美神的作为

"你们要赞美耶和华！我要在正直人的大会中，并公会中，一心称谢耶和华。"（111:1）

字母诗。赞美神的作为和信实。"耶和华的作为本为大"（111:2）。"敬畏耶和华是智慧的开端"（111:10）。

诗篇112篇 — 义人的福气

"敬畏耶和华，甚喜爱他命令的，这人便为有福。"（112:1）

与诗111配对的字母诗。描绘义人的福气。"他施舍钱财，周济贫穷"（112:9）。"恶人看见便恼恨"（112:10）。

诗篇113篇 — 赞美耶和华的名

"你们要赞美耶和华！耶和华的仆人哪，你们要赞美，赞美耶和华的名。"（113:1）

哈利路亚诗篇。赞美神的尊荣和谦卑。"他从灰尘里抬举贫寒人，从粪堆中提拔穷乏人"（113:7）。

诗篇114篇 — 出埃及的神迹

"以色列出了埃及，雅各家离开说异言之民。"（114:1）

赞美出埃及的神迹。红海分开，约旦河倒流，大地在主面前震动。

诗篇115篇 — 荣耀归于神

"耶和华啊，荣耀不要归于我们，不要归于我们，要因你的慈爱和诚实归在你的名下。"（115:1）

对比真神与偶像。"造他的要和他一样，凡靠他的也要如此"（115:8）。"耶和华啊，他是我们的帮助，是我们的盾牌"（115:9-11）。

诗篇116篇 — 感恩的诗

"我爱耶和华，因为他听了我的声音和我的恳求。"（116:1）

感恩诗。"在耶和华眼中，看圣民之死极为宝贵"（116:15）。"我要以感谢为祭献给你"（116:17）。

诗篇117篇 — 万国赞美耶和华

"万国啊，你们都当赞美耶和华！万民哪，你们都当颂赞他！"（117:1）

诗篇中最短的一篇，也是圣经中最短的一章。保罗引用此诗说明神的救恩临到万民（罗15:11）。

诗篇118篇 — 感恩与弥赛亚

"你们要称谢耶和华，因他本为善，他的慈爱永远长存。"（118:1）

感恩诗篇，也是弥赛亚诗篇。"匠人所弃的石头已成了房角的头块石头"（118:22），耶稣引用此诗指自己（太21:42）。"奉耶和华名来的是应当称颂的"（118:26），这是耶稣进耶路撒冷时群众的欢呼。

诗篇119篇 — 赞美神的律法

"行为完全、遵行耶和华律法的，这人便为有福。"（119:1）

诗篇中最长的一篇，共176节，是圣经中最长的一章。每一段都以希伯来字母开头，共22段（22个字母），每段8节。全诗赞美神的律法，使用律法的同义词（律法、法度、训词、律例、命令、典章、话、道）共176次。

"你的话是我脚前的灯，是我路上的光"（119:105），是最著名的经文之一。

4. 上行之诗（诗120-134篇）

诗篇120篇 — 求神拯救

"我在急难中求告耶和华，他就应允我。"（120:1）

朝圣旅程的开始。诗人远离家乡，住在米设和基达之间。

诗篇121篇 — 我的帮助从何而来

"我要向山举目，我的帮助从何而来？我的帮助从造天地的耶和华而来。"（121:1-2）

最受人喜爱的上行之诗。"保护你的是耶和华，耶和华在你右边荫庇你"（121:5）。"耶和华要保护你，免受一切的灾害。他要保护你的性命"（121:7）。

诗篇122篇 — 为耶路撒冷求平安

"人对我说：我们往耶和华的殿去，我就欢喜。"（122:1）

大卫的诗。朝圣者到达耶路撒冷的喜乐。"你们要为耶路撒冷求平安"（122:6）。

诗篇123篇 — 仰望神

"坐在天上的主啊，我向你举目。"（123:1）

仰望神的怜悯。"仆人的眼睛怎样望主人的手，使女的眼睛怎样望主母的手，我们的眼睛也照样望耶和华我们的神"（123:2）。

诗篇124篇 — 耶和华帮助我们

"若不是耶和华帮助我们，以色列当说，若不是耶和华帮助我们。"（124:1）

大卫的诗。回顾神的帮助。"我们好像雀鸟，从捕鸟人的网罗里逃脱"（124:7）。

诗篇125篇 — 信靠的保护

"倚靠耶和华的人好像锡安山，永不动摇。"（125:1）

信靠诗篇。"众山怎样围绕耶路撒冷，耶和华也照样围绕他的百姓"（125:2）。

诗篇126篇 — 梦中欢喜

"当耶和华将那些被掳的带回锡安的时候，我们好像做梦的人。"（126:1）

归回的喜乐。"那带种流泪出去的，必要欢欢乐乐地带禾捆回来"（126:5-6）。

诗篇127篇 — 神所建造的

"若不是耶和华建造房屋，建造的人就枉然劳力。"（127:1）

所罗门的智慧诗。"儿女是耶和华所赐的产业"（127:3）。

诗篇128篇 — 敬畏耶和华的福气

"凡敬畏耶和华、遵行他道的人便为有福。"（128:1）

描绘义人家庭的福气。

诗篇129篇 — 从苦难中拯救

"从我幼年以来，敌人屡次苦害我。"（129:1）

回顾以色列的苦难和神的拯救。

诗篇130篇 — 从深处求告

"耶和华啊，我从深处向你求告。"（130:1）

忏悔诗。"主耶和华啊，你若究察罪孽，谁能站得住呢？但在你有赦免之恩"（130:3-4）。"我的心等候主，胜于守夜的等候天亮"（130:6）。

诗篇131篇 — 谦卑安稳

"耶和华啊，我的心不狂傲，我的眼不高大。"（131:1）

大卫的谦卑诗。"我的心平稳安静，好像断过奶的孩子在他母亲怀中"（131:2）。

诗篇132篇 — 记念大卫

"耶和华啊，求你记念大卫所受的一切苦难。"（132:1）

君王诗篇。记念大卫将约柜安放在锡安的誓愿。"这是我永远安息之所，我要住在这里"（132:14）。

诗篇133篇 — 弟兄和睦同居

"看哪，弟兄和睦同居，是何等地善，何等地美！"（133:1）

大卫的智慧诗。以贵重的油和黑门的甘露比喻弟兄和睦的福气。

诗篇134篇 — 夜间的赞美

"耶和华的仆人，夜间站在耶和华殿中的，你们当称颂耶和华。"（134:1）

上行之诗的结束。夜间在圣殿中的侍者被邀请赞美神。

5. 赞美与信靠（诗135-145篇）

诗篇135篇 — 赞美耶和华的名

"你们要赞美耶和华！你们要赞美耶和华的名。"（135:1）

赞美诗。"耶和华所喜悦的，必记念"（135:14）。对比真神与偶像。

诗篇136篇 — 他的慈爱永远长存

"你们要称谢耶和华，因他本为善，他的慈爱永远长存。"（136:1）

著名的感恩诗。每节都以"他的慈爱永远长存"结束，共26次。回顾神的创造和历史中的作为。

诗篇137篇 — 在巴比伦河边

"我们曾在巴比伦的河边坐下，一追想锡安就哭了。"（137:1）

被掳的哀歌。"耶路撒冷啊，我若忘记你，情愿我的右手忘记技巧"（137:5）。

诗篇138篇 — 感谢神

"我要一心称谢你，在诸神面前歌颂你。"（138:1）

大卫的感恩诗。"耶和华虽高，仍看顾低微的人"（138:6）。

诗篇139篇 — 神的全知

"耶和华啊，你已经鉴察我，认识我。"（139:1）

大卫的诗篇，赞美神的全知全能。"我坐下，我起来，你都晓得"（139:2）。"我在暗中受造，在地的深处被联络"（139:15）。"我未成形的体质，你的眼早已看见了"（139:16）。

诗篇140篇 — 求神保护

"耶和华啊，求你拯救我脱离凶恶的人。"（140:1）

大卫的哀歌。求神保护脱离恶人的谋害。

诗篇141篇 — 晚间的祷告

"耶和华啊，我曾求告你，求你快快临到我这里。"（141:1）

大卫的晚祷。"愿我的祷告如香陈列在你面前"（141:2）。

诗篇142篇 — 在洞中的祷告

"我发声哀告耶和华，发声恳求耶和华。"（142:1）

大卫在洞中的祷告。"你是我在患难中的避难所"（142:5）。

诗篇143篇 — 求神引导

"耶和华啊，求你听我的祷告，留心听我的恳求。"（143:1）

忏悔诗。"求你指教我遵行你的旨意，因你是我的神"（143:10）。

诗篇144篇 — 神是磐石

"耶和华我的磐石，是应当称颂的。"（144:1）

大卫的诗。赞美神在争战中的帮助。"耶和华啊，人算什么，你竟认识他？世人算什么，你竟顾念他？"（144:3）。

诗篇145篇 — 赞美神的国度

"我的神、我的王啊，我要尊崇你，我要永永远远称颂你的名。"（145:1）

大卫的赞美诗。"耶和华本为大，该受大赞美，其大无法测度"（145:3）。"耶和华在他一切所行的，无不公义；在他一切所做的，都有慈爱"（145:17）。

6. 哈利路亚诗篇（诗146-150篇）

诗篇146篇 — 你们要赞美耶和华

"你们要赞美耶和华！我的心哪，你要赞美耶和华。"（146:1）

哈利路亚诗篇。"你们不要倚靠君王，不要倚靠世人，他一点不能帮助"（146:3）。"耶和华扶持被压下的人"（146:8）。

诗篇147篇 — 赞美神的慈爱

"你们要赞美耶和华！因歌颂我们的神为善为美，赞美的话是合宜的。"（147:1）

赞美神的慈爱和大能。"他数点星宿的数目，一一称他的名"（147:4）。"耶和华喜爱敬畏他和盼望他慈爱的人"（147:11）。

诗篇148篇 — 万有赞美耶和华

"你们要从天上赞美耶和华！在高处赞美他。"（148:1）

邀请天上地上一切受造之物赞美神。"大山和小山，结果的树木和一切香柏树"（148:9）。

诗篇149篇 — 新歌

"你们要向耶和华唱新歌！"（149:1）

赞美诗。"愿以色列因造他的主欢喜"（149:2）。

诗篇150篇 — 你们要赞美耶和华

"你们要赞美耶和华！在神的圣所赞美他！"（150:1）

诗篇的最终高潮。六次呼唤"赞美耶和华"。"凡有气息的都要赞美耶和华！你们要赞美耶和华！"（150:6），这是整本诗篇的完美结束。

四、神学主题

1. 感恩与赞美：对神恩典的回应 2. 神的话语：律法的宝贵和完美 3. 朝圣的旅程：从地上到天上的旅程 4. 弥赛亚的盼望：基督的王权和祭司职分 5. 哈利路亚的颂赞：最终的赞美与敬拜

五、新约关联

  • 诗110:1 → 太22:44（基督在神右边）
  • 诗110:4 → 来5:6（基督为大祭司）
  • 诗118:22 → 太21:42（房角石）
  • 诗118:26 → 太21:9（和散那）
  • 诗117:1 → 罗15:11（万国赞美）

六、实际应用

1. 感恩的生活：像诗136篇一样，数算神的恩典 2. 渴慕神的话语：像诗119篇一样，爱慕神的律法 3. 信靠神的保护：像诗121篇一样，仰望造天地的主 4. 弟兄和睦：像诗133篇一样，追求合一 5. 赞美的人生：像诗150篇一样，以赞美为生命的高峰

七、祷告与默想

耶和华我的神，你的话是我脚前的灯，是我路上的光。我要一心称谢你，在诸神面前歌颂你。我的心哪，你要称颂耶和华，凡在我里面的，也要称颂他的圣名。凡有气息的都要赞美耶和华！你们要赞美耶和华！奉耶稣基督的名祷告，阿们。

"凡有气息的都要赞美耶和华！你们要赞美耶和华！"（诗篇 150:6）
