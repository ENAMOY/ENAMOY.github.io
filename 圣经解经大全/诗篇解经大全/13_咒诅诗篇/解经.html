咒诅诗篇 解经

一、诗篇背景

  • 经文范围：诗篇 35, 58, 69, 83, 109, 137
  • 作者：大卫（诗35, 69, 109）、亚萨（诗58, 83）、未注明作者（诗137）
  • 历史背景：咒诅诗篇（Imprecatory Psalms）是诗篇中最令人困惑的一类，表达了对仇敌的咒诅和对神公义审判的呼求
  • 写作目的：这些诗篇教导我们在面对不公义时，如何向神呼求公义审判，同时理解它们在新约中的转化

二、经文概览

咒诅诗篇共6篇，表达了对仇敌的咒诅和对神公义审判的呼求。这些诗篇需要在神的公义和慈爱的背景下理解。

咒诅诗篇的特点

特点说明
呼求公义不是私人报复，而是呼求神施行公义
信心的表现相信神是公义的审判者
先知性的预言神对恶人的审判
集体性的代表整个受压迫的群体

三、逐节解经

1. 诗篇35篇 — 求神申冤

"耶和华啊，与我相争的，求你与他们相争。"（35:1）

背景：大卫被敌人攻击时的哀歌。

逐节分析：

  • 35:1-3 求神争战："耶和华啊，与我相争的，求你与他们相争。与我相战的，求你与他们相战。"

  • 35:4-8 愿仇敌蒙羞："愿那寻索我命的，蒙羞受辱。愿那谋害我的，退后羞愧。"

  • 35:11-16 以恶报善："无故的恶人聚集攻击我，他们向我以恶报善，使我凄凉。"

  • 35:17-28 求神拯救："耶和华啊，你看着不理要到几时呢？"

神学意义：大卫不是寻求私人报复，而是将审判交给神。这是信心的表现——相信神是公义的审判者。

2. 诗篇58篇 — 审判不公的审判官

"世人哪，你们默然不语，真合公义吗？施行审判，岂按正直吗？"（58:1）

背景：亚萨的诗篇，针对不公义的审判官。

逐节分析：

  • 58:1-2 控告不公义："世人哪，你们默然不语，真合公义吗？施行审判，岂按正直吗？不然，你们是心中作恶，你们在地上秤出你们手所行的强暴。"

  • 58:3-5 恶人的本性："恶人一出母胎，就与神疏远；一离母腹，便走错路，说谎话。"

  • 58:6-9 求神审判："神啊，求你敲碎他们口中的牙；耶和华啊，求你敲掉少壮狮子的大牙。"

神学意义：针对不公义的掌权者，呼求神施行公义审判。

3. 诗篇69篇 — 被仇敌围困

"神啊，求你救我！因为众水要淹没我。"（69:1）

背景：大卫的哀歌，表达极度的苦难。

逐节分析：

  • 69:1-4 苦难的描述："我陷在深淤泥中，没有立脚之地；我到了深水中，大水漫过我身。"

  • 69:7-12 为神的殿热心："因我为你的殿心里焦急，如同火烧。"

  • 69:13-18 求神拯救："耶和华啊，求你凭你的慈爱应允我。"

  • 69:19-28 对仇敌的咒诅："愿他们的筵席变为网罗，变为机槛。"

  • 69:29-36 赞美："我要以诗歌赞美神的名。"

新约应验：

  • 69:9 → 约2:17（为神的殿焦急）
  • 69:21 → 约19:28-29（苦胆和醋）
  • 69:25 → 徒1:20（犹大的职分）

4. 诗篇83篇 — 求神击败仇敌

"神啊，求你不要静默！"（83:1）

背景：亚萨的诗篇，针对攻击以色列的仇敌。

逐节分析：

  • 83:1-2 求神不要静默："神啊，求你不要静默！神啊，求你不要闭口，也不要不作声！"

  • 83:3-8 仇敌的阴谋："因为你的仇敌喧嚷，恨你的抬起头来。他们同谋奸诈，要害你的百姓。"

  • 83:9-17 求神审判："求你待他们如待米甸，如在基顺河待西西拉和耶宾一样。"

  • 83:18 愿人认识神："使他们知道，惟独你名为耶和华的，是全地以上的至高者。"

神学意义：呼求神保护他的百姓，击败仇敌。最终目的是让人认识神。

5. 诗篇109篇 — 大卫的咒诅诗

"我所赞美的人，反而攻击我。"（109:4）

背景：大卫被敌人攻击时的哀歌，是诗篇中最严厉的咒诅诗。

逐节分析：

  • 109:1-5 以恶报善："我所赞美的人，反而攻击我。他们以恨我代替爱我。"

  • 109:6-15 对仇敌的咒诅："愿你派一个恶人辖制他，派一个对头站在他右边。"

  • 109:16-20 原因："因为他不想施恩，却逼迫困苦穷乏的和伤心的人。"

  • 109:21-31 求神拯救："耶和华我的神啊，求你帮助我。"

  • 109:30-31 赞美："我要用口极力称谢耶和华，我要在众人中间赞美他。"

新约关联：

  • 109:8 → 徒1:20（犹大的职分）

6. 诗篇137篇 — 在巴比伦河边

"耶路撒冷啊，我若忘记你，情愿我的右手忘记技巧。"（137:5）

背景：被掳到巴比伦后，以色列人的哀歌。

逐节分析：

  • 137:1-4 被掳的哀伤："我们曾在巴比伦的河边坐下，一追想锡安就哭了。我们把琴挂在那里的柳树上。"

  • 137:5-6 不忘记耶路撒冷："耶路撒冷啊，我若忘记你，情愿我的右手忘记技巧。我若不记念你，若不看耶路撒冷过于我所最喜乐的，情愿我的舌头贴于上膛。"

  • 137:7-9 对仇敌的咒诅："耶路撒冷遭难的日子，以东人说：拆毁，拆毁，直拆到根基！耶和华啊，求你记念这仇。将要被灭的巴比伦城啊，报复你像你待我们的，那人便为有福。拿你的婴孩摔在磐石上的，那人便为有福。"

神学意义：这是被掳群体的哀歌，表达了对仇敌的愤怒和对神公义审判的呼求。需要理解被掳的极度痛苦背景。

四、神学主题

1. 神的公义：神是公义的审判者，必按公义审判世界 2. 信心的表现：将审判交给神，不自己报仇 3. 集体的呼求：代表受压迫的群体向神呼求 4. 先知性的预言：预言神对恶人的审判 5. 公义与慈爱的平衡：神既有公义，也有慈爱

五、新约的转化

咒诅诗篇在新约中得到了转化：

1. 耶稣的教导："要爱你们的仇敌，为那逼迫你们的祷告"（太5:44） 2. 保罗的教导："不要自己伸冤，宁可让步，听凭主怒"（罗12:19） 3. 彼得的榜样：耶稣在十字架上为仇敌祷告（路23:34） 4. 启示录的应验：圣徒在天上呼求神的公义审判（启6:10）

理解咒诅诗篇的关键：

  • 这不是私人报复，而是呼求神施行公义
  • 这是信心的表现——相信神是公义的审判者
  • 这是代表受压迫的群体向神呼求
  • 在新约中，我们学习为仇敌祷告，将审判交给神

六、实际应用

1. 将审判交给神：不自己报仇，相信神是公义的审判者 2. 为仇敌祷告：学习耶稣的榜样，为逼迫我们的人祷告 3. 呼求公义：为受压迫的人呼求神的公义 4. 信靠神的时间：忍耐等候神的时间 5. 在苦难中信靠：在困难中信靠神的保护

七、祷告与默想

耶和华我的神，你是公义的审判者，必按你的公义审判世界。求你帮助我不自己报仇，将一切审判交给你。求你帮助我为仇敌祷告，像耶稣在十字架上所做的。求你施行公义，拯救受压迫的人。奉耶稣基督的名祷告，阿们。

"亲爱的弟兄，不要自己伸冤，宁可让步，听凭主怒。因为经上记着：主说：伸冤在我，我必报应。"（罗马书 12:19）
