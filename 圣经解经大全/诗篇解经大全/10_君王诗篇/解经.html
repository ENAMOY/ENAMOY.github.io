君王诗篇 解经

一、诗篇背景

  • 经文范围：诗篇 2, 18, 20, 21, 45, 72, 89, 101, 110, 132, 144
  • 作者：大卫（诗2, 18, 20, 21, 89, 101, 110, 132, 144）、可拉后裔（诗45）、所罗门（诗72）
  • 历史背景：君王诗篇与以色列的君王制度密切相关，既描写地上的君王，也指向天上的弥赛亚君王
  • 写作目的：这些诗篇教导我们认识王权的本质，以及弥赛亚君王的完全应验

二、经文概览

君王诗篇共11篇，涵盖了君王的各个方面：登基、争战、得胜、公义统治、永远的王权。

三、逐节解经

1. 登基诗篇

诗篇2篇 — 受膏者的登基

"你是我的儿子，我今日生你。"（2:7）

第一篇君王诗篇。列邦争闹，但神已经立他的君在锡安。受膏者宣告："你求我，我就将列国赐你为基业"（2:8）。

诗篇72篇 — 所罗门的登基

"神啊，求你将判断的权柄赐给王。"（72:1）

所罗门的诗篇，描写理想的君王。"他要按公义审判你的民，按公平审判你的困苦人"（72:2）。"他的名要存到永远"（72:17）。

诗篇110篇 — 弥赛亚的登基

"耶和华对我主说：你坐在我的右边。"（110:1）

最重要的君王诗篇。预言弥赛亚坐在神右边，照麦基洗德的等次为祭司。

2. 争战诗篇

诗篇18篇 — 大卫的得胜之歌

"耶和华我的力量啊，我爱你！"（18:1）

大卫在脱离一切仇敌和扫罗之手后所写。"耶和华是我的岩石，我的山寨，我的救主"（18:2）。描述神在争战中的帮助。

诗篇20篇 — 争战前的祷告

"愿耶和华在你遭难的日子应允你。"（20:1）

争战前的祷告。"有人靠车，有人靠马，但我们要提到耶和华我们神的名"（20:7）。

诗篇144篇 — 神是磐石

"耶和华我的磐石，是应当称颂的。"（144:1）

大卫的诗。赞美神在争战中的帮助。"耶和华啊，人算什么，你竟认识他？世人算什么，你竟顾念他？"（144:3）。

3. 得胜诗篇

诗篇21篇 — 王的得胜

"耶和华啊，王必因你的能力欢喜。"（21:1）

得胜的感恩诗。"他向你求寿，你赐给他，就是日子长久，直到永远"（21:4）。

诗篇45篇 — 王的婚礼

"你比世人更美。"（45:2）

弥赛亚诗篇。描写君王的婚礼。"神啊，你的宝座是永永远远的"（45:6）。

4. 公义统治

诗篇101篇 — 王的榜样

"我要歌唱慈爱和公平。"（101:1）

大卫的诗篇，描绘理想君王的品格。"我要用智慧行完全的道"（101:2）。

5. 弥赛亚预言

诗篇89篇 — 神与大卫的约

"我与我所拣选的人立了约。"（89:3）

以探的智慧诗。回顾神与大卫所立的永远之约。

6. 约的纪念

诗篇132篇 — 记念大卫

"耶和华啊，求你记念大卫所受的一切苦难。"（132:1）

君王诗篇。记念大卫将约柜安放在锡安的誓愿。"这是我永远安息之所"（132:14）。

四、神学主题

1. 王权的来源：王权来自神的设立 2. 公义的统治：君王必须按公义治理 3. 神的约：神与大卫所立的永远之约 4. 弥赛亚的王权：基督永远的王权 5. 争战与得胜：神帮助君王争战得胜

五、新约关联

  • 诗2:7 → 徒13:33; 来1:5（基督为神的儿子）
  • 诗45:6-7 → 来1:8-9（基督的王权）
  • 诗72:8 → 启11:15（基督的国度）
  • 诗89:3-4 → 路1:32-33（基督的王位）
  • 诗110:1 → 太22:44（基督在神右边）
  • 诗110:4 → 来5:6（基督为大祭司）

六、实际应用

1. 认识基督为王：基督是万王之王，万主之主 2. 顺服神的权柄：在一切事上顺服神的权柄 3. 公义的生活：像理想君王一样追求公义 4. 信靠神的保护：在争战中信靠神的帮助 5. 盼望基督的国度：仰望基督永远的国度

七、祷告与默想

耶和华我的神，感谢你设立君王治理你的百姓。感谢你差遣基督作我们的君王，他按公义审判世界，他的国度存到永远。求帮助我们顺服基督的权柄，活出公义的生活。奉耶稣基督的名祷告，阿们。

"耶和华对我主说：你坐在我的右边，等我使你仇敌作你的脚凳。"（诗篇 110:1）
