卷二：诗篇42-72篇 解经

一、诗篇背景

  • 经文范围：诗篇 42:1 - 72:20
  • 作者：可拉后裔（诗42, 44-49, 84-85, 87-88）、大卫（诗51-65, 68-70）、所罗门（诗72）
  • 历史背景：涵盖大卫王朝时期，包括大卫的逃亡、犯罪、悔改以及王国的建立
  • 写作目的：与卷一相对应，如同出埃及记对应创世记，本卷强调从困苦到拯救的主题

二、经文概览

卷二包含31篇诗篇，核心主题是从困苦到拯救、国家的救恩。本卷与卷一相比，更多使用"以罗欣"（Elohim，神）这一神名，诗歌风格更加集体化和国家化。

结构大纲

段落诗篇主题
渴慕神42-49可拉后裔的诗篇
大卫的哀歌51-65个人与国家的呼求
赞美与感恩66-70集体的敬拜
所罗门的诗72弥赛亚的国度

三、逐节解经

1. 渴慕神（诗42-49篇）

诗篇42篇 — 渴慕神

"神啊，我的心切慕你，如鹿切慕溪水。"（42:1）

可拉后裔的诗篇，表达对神的深切渴慕。诗人在外邦地远离圣殿，心中忧闷，但他仍对神说："我昼夜以眼泪当饮食"（42:3）。这首诗教导我们，在灵性低谷中仍要渴慕神。

诗篇44篇 — 国家的哀歌

"神啊，你在古时列祖眼前所行的事，我们亲耳听见了。"（44:1）

国家哀歌。诗人回顾神在过去所行的拯救，却困惑于当前的苦难。"你叫我们当作被杀的羊，散在列邦中"（44:11），但诗人仍信靠神。

诗篇45篇 — 王的婚礼

"你比世人更美，在你嘴里满有恩惠。"（45:2）

弥赛亚诗篇。描写君王的婚礼，预表基督与教会的联合。"神啊，你的宝座是永永远远的"（45:6），新约引用此诗指基督（来1:8）。

诗篇46篇 — 神是我们的避难所

"神是我们的避难所，是我们的力量，是我们在患难中随时的帮助。"（46:1）

信靠诗篇。即使大地改变、山摇动到海心，我们仍不害怕。"你们要休息，要知道我是神"（46:10），这是信心最深的宣告。

诗篇48篇 — 锡安的荣耀

"耶和华本为大，在我们神的城中，在他的圣山上，该受大赞美。"（48:1）

赞美锡安的诗篇。描述耶路撒冷的美丽和神在其中的保护。

诗篇49篇 — 财富的虚空

"你们不要倚靠财利，也不要倚靠无定的钱财。"（49:13）

智慧诗。教导人不能用财富赎生命，"一个也无法赎自己的弟兄"（49:7），唯有神能救赎人的灵魂脱离阴间。

2. 大卫的哀歌（诗51-65篇）

诗篇51篇 — 大卫的忏悔

"神啊，求你按你的慈爱怜恤我，按你丰盛的慈悲涂抹我的过犯。"（51:1）

最伟大的忏悔诗。大卫与拔示巴犯罪后，先知拿单责备他，他写了这篇忏悔诗。大卫承认自己的罪，求神洁净他："求你为我造清洁的心，使我里面重新有正直的灵"（51:10）。

诗篇53篇 — 愚顽人

"愚顽人心里说：没有神。"（53:1）

与诗14篇几乎完全相同，描绘愚顽人的败坏。"没有一个人行善"（53:3），保罗在罗马书中引用此诗说明人的全然败坏（罗3:10-12）。

诗篇55篇 — 被朋友出卖

"原来不是仇敌辱骂我，若是仇敌，还可忍耐；也不是恨我的人向我狂大，若是恨我的人，就必躲避他。"（55:12）

大卫的哀歌，表达被亲密朋友出卖的痛苦。"你要把你的重担卸给耶和华，他必抚养你"（55:22），这是信心的宣告。

诗篇57篇 — 在翅膀下避难

"神啊，求你怜悯我，怜悯我！因为我的心投靠你。我要投靠在你翅膀的荫下。"（57:1）

大卫逃避扫罗时在洞中所写。虽在危险中，大卫仍宣告："神啊，我心坚定，我心坚定！我要唱诗，我要歌颂！"（57:7）。

诗篇62篇 — 专等候神

"我的心默默无声，专等候神，我的救恩是从他而来。"（62:1）

信靠诗篇。大卫教导我们，神是唯一的磐石和拯救。"你们众民当时时倚靠他，在他面前倾心吐意。神是我们的避难所"（62:8）。

诗篇63篇 — 渴慕神

"神啊，你是我的神！我要切切地寻求你；在干旱疲乏无水之地，我渴想你。"（63:1）

大卫在犹大旷野所写。表达了对神的深切渴慕。"因你的慈爱比生命更好，我的嘴唇要颂赞你"（63:3）。

3. 赞美与感恩（诗66-70篇）

诗篇66篇 — 全地都当向神欢呼

"全地都当向神欢呼！"（66:1）

集体赞美诗。邀请全地向神欢呼，述说神在历史中的作为。

诗篇67篇 — 愿神怜悯我们

"愿神怜悯我们，赐福与我们，用脸光照我们。"（67:1）

祝福的祷告。愿万国得知神的道路，万民得知神的救恩。

诗篇68篇 — 神在圣所中兴起

"愿神兴起，使他的仇敌四散。"（68:1）

宏伟的赞美诗。描写神在历史中的得胜，从旷野到锡安的旅程。"你已经升上高天，掳掠仇敌"（68:18），新约引用此诗指基督的升天（弗4:8）。

诗篇69篇 — 大卫的哀歌

"神啊，求你救我！因为众水要淹没我。"（69:1）

大卫的哀歌，表达极度的苦难。"他们拿苦胆给我当食物；我渴了，他们拿醋给我喝"（69:21），新约引用此诗指基督在十字架上的经历（约19:28-29）。

4. 所罗门的诗（诗72篇）

诗篇72篇 — 弥赛亚的国度

"神啊，求你将判断的权柄赐给王，将公义赐给王的儿子。"（72:1）

所罗门的诗篇，描写理想的君王和国度。"他要从海极拯救被掳的人，从远方搭救受欺压的人"（72:12-13）。这不仅是所罗门的祷告，更是对弥赛亚国度的预言。"他的名要存到永远，要留传如日之久"（72:17）。

四、神学主题

1. 从困苦到拯救：以色列的历史是从奴役到自由的旅程 2. 渴慕神：在苦难中切慕神的同在 3. 忏悔与洁净：大卫的认罪成为忏悔的典范 4. 弥赛亚的国度：预言基督永远的王权 5. 国家的救恩：神对整个国家的拯救计划

五、新约关联

  • 诗45:6-7 → 来1:8-9（基督的王权）
  • 诗51:4 → 罗3:4（神的公义）
  • 诗53:1-3 → 罗3:10-12（人的败坏）
  • 诗68:18 → 弗4:8（基督的升天）
  • 诗69:21-22 → 约19:28-29（十字架的经历）

六、实际应用

1. 渴慕神的同在：在灵性低谷中仍要切慕神 2. 真实的忏悔：学习大卫在神面前真诚认罪 3. 信靠神的拯救：在困难中等候神的作为 4. 赞美神的信实：述说神在历史中的奇妙作为 5. 盼望弥赛亚：仰望基督永远的国度

七、祷告与默想

神啊，我的心切慕你，如鹿切慕溪水。求你按你的慈爱怜恤我，按你丰盛的慈悲涂抹我的过犯。求你为我造清洁的心，使我里面重新有正直的灵。我要投靠在你翅膀的荫下，直到灾害过去。奉耶稣基督的名祷告，阿们。

"神啊，我的心切慕你，如鹿切慕溪水。"（诗篇 42:1）
