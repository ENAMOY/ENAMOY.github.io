信靠诗篇 解经

一、诗篇背景

  • 经文范围：诗篇 4, 11, 16, 23, 27, 46, 62, 63, 91, 121, 125, 131, 138, 139
  • 作者：大卫（诗4, 11, 16, 23, 27, 62, 63, 131, 138, 139）、可拉后裔（诗46, 84）、未注明作者（诗91, 121, 125）
  • 历史背景：信靠诗篇是诗篇中最能安慰人心的一类，表达了在各种境遇中对神的坚定信靠
  • 写作目的：教导信徒在困难、危险、不确定中如何信靠神

二、经文概览

信靠诗篇共14篇，表达了对神各种方面的信靠：牧者的看顾、避难所的保护、磐石的稳固、亮光的引导。

三、逐节解经

1. 诗篇4篇 — 安然居住

"我必安然躺下睡觉，因为独有你耶和华使我安然居住。"（4:8）

背景：大卫的晚间祷告。

逐节分析：

  • 4:1 求神应允："显我为义的神啊，我呼吁的时候，求你应允我。"

  • 4:2-3 信心的宣告："你们这上流人哪，你们将我的尊荣变为羞辱要到几时呢？你们要知道，耶和华已经分别虔诚人归他自己。"

  • 4:4-5 劝勉："你们应当畏惧，不可犯罪；在床上的时候，要心里思想，并要肃静。"

  • 4:6-8 安然居住："耶和华啊，求你仰起脸来，光照我们。你使我心里快乐。我必安然躺下睡觉，因为独有你耶和华使我安然居住。"

神学意义：在危险中仍能安然入睡，因为信靠神的保护。

2. 诗篇11篇 — 投靠耶和华

"我是投靠耶和华，你们怎么对我说：你当像鸟飞往你的山去？"（11:1）

背景：大卫的信靠诗。

逐节分析：

  • 11:1-3 投靠神："我是投靠耶和华，你们怎么对我说：你当像鸟飞往你的山去？恶人弯弓，把箭搭在弦上，要在暗中射那心里正直的人。"

  • 11:4-7 耶和华在圣殿中："耶和华在他的圣殿里，耶和华的宝座在天上。他的慧眼察看世人。耶和华试验义人。"

神学意义：在危险中投靠神，因为神在圣殿中掌管一切。

3. 诗篇16篇 — 生命的道路

"你必将生命的道路指示我。"（16:11）

背景：大卫宣告神是他的产业和福分。

逐节分析：

  • 16:1-2 投靠神："神啊，求你保佑我，因为我投靠你。"

  • 16:3-4 圣徒的团契："论到世上的圣民，他们又美又善，是我最喜悦的。"

  • 16:5-6 神是产业："耶和华是我的产业，是我杯中的份。"

  • 16:7-8 常在神面前："我必称颂那指教我的耶和华。我将耶和华常摆在我面前，因他在我右边，我便不至摇动。"

  • 16:9-11 生命的道路："因此我的心欢喜，我的灵快乐，我的肉身也要安然居住。因为你必不将我的灵魂撇在阴间，也不叫你的圣者见朽坏。你必将生命的道路指示我。在你面前有满足的喜乐，在你右手中有永远的福乐。"

新约应验：使徒彼得在五旬节讲道时引用此诗（徒2:25-28），说明基督的复活。

4. 诗篇23篇 — 耶和华是我的牧者

"耶和华是我的牧者，我必不致缺乏。"（23:1）

背景：最受人喜爱的诗篇，以牧羊的意象描绘神的看顾。

逐节分析：

  • 23:1-3 牧者的看顾："耶和华是我的牧者，我必不致缺乏。他使我躺卧在青草地上，领我在可安歇的水边。他使我的灵魂苏醒，为自己的名引导我走义路。"

  • 23:4 死荫的幽谷："我虽然行过死荫的幽谷，也不怕遭害，因为你与我同在；你的杖、你的竿，都安慰我。"

  • 23:5-6 敌人面前的筵席："在我敌人面前，你为我摆设筵席。你用油膏了我的头，使我的福杯满溢。我一生一世必有恩惠慈爱随着我，我且要住在耶和华的殿中，直到永远。"

神学意义：神是我们的牧者，从青草地到死荫的幽谷，都有神的同在。

5. 诗篇27篇 — 耶和华是我的亮光

"耶和华是我的亮光，是我的拯救，我还怕谁呢？"（27:1）

背景：大卫对神的坚定信靠。

逐节分析：

  • 27:1-3 信心的宣告："耶和华是我的亮光，是我的拯救，我还怕谁呢？耶和华是我性命的保障，我还惧谁呢？"

  • 27:4 渴望神的同在："有一件事，我曾求耶和华，我仍要寻求：就是一生一世住在耶和华的殿中，瞻仰他的荣美，在他的殿里求问。"

  • 27:5-6 神的保护："因为我遭遇患难，他必暗暗地保守我；在他亭子里，把我藏在他帐幕的隐密处，将我高举在磐石上。"

  • 27:7-12 求神垂听："耶和华啊，我用声音呼吁的时候，求你垂听我。"

  • 27:13-14 等候神："我若不信在活人之地得见耶和华的恩惠，就早已丧胆了。要等候耶和华！当壮胆，坚固你的心！我再说，要等候耶和华！"

神学意义：在黑暗中等候神，是信心最深的操练。

6. 诗篇46篇 — 神是我们的避难所

"神是我们的避难所，是我们的力量，是我们在患难中随时的帮助。"（46:1）

背景：可拉后裔的诗篇。

逐节分析：

  • 46:1-3 不害怕："神是我们的避难所，是我们的力量，是我们在患难中随时的帮助。所以地虽改变，山虽摇动到海心，其中的水虽砰訇翻腾，山虽因海涨而战抖，我们也不害怕。"

  • 46:4-7 神的城："有一道河，这河的分汊使神的城欢喜。这城就是至高者居住的圣所。神在其中，城必不动摇。"

  • 46:8-11 你们要休息："你们来看耶和华的作为。你们要休息，要知道我是神。我必在外邦中被尊崇，在遍地上也被尊崇。"

神学意义："你们要休息，要知道我是神"（46:10），是信心最深的宣告。

7. 诗篇62篇 — 专等候神

"我的心默默无声，专等候神，我的救恩是从他而来。"（62:1）

背景：大卫的信靠诗。

逐节分析：

  • 62:1-2 专等候神："我的心默默无声，专等候神，我的救恩是从他而来。惟独他是我的磐石、我的拯救，他是我的高台，我必不很动摇。"

  • 62:3-4 反对恶人："你们大家攻击一人，把他毁坏，如同毁坏歪斜的墙、将倒的壁，要到几时呢？"

  • 62:5-8 信靠神："我的心哪，你当默默无声，专等候神，因为我的盼望是从他而来。惟独他是我的磐石、我的拯救，他是我的高台，我必不动摇。我的拯救、我的荣耀，都在乎神；我力量的磐石、我的避难所，都在乎神。"

  • 62:9-12 不倚靠财富："你们众民当时时倚靠他，在他面前倾心吐意。神是我们的避难所。"

神学意义：在动荡中默默等候神，因为他是磐石和拯救。

8. 诗篇63篇 — 渴慕神

"神啊，你是我的神！我要切切地寻求你。"（63:1）

背景：大卫在犹大旷野所写。

逐节分析：

  • 63:1-4 渴慕神："神啊，你是我的神！我要切切地寻求你；在干旱疲乏无水之地，我渴想你。我在圣所中曾如此瞻仰你，为要见你的能力和你的荣耀。因你的慈爱比生命更好，我的嘴唇要颂赞你。"

  • 63:5-8 信靠神："我在床上记念你，在夜更的时候思想你。因为你曾帮助我，我就在你翅膀的荫下欢呼。我心紧紧地跟随你，你的右手扶持我。"

  • 63:9-11 盼望："但那寻索王命的，必往地底下去。"

神学意义：在干旱之地渴慕神，如同鹿切慕溪水。

9. 诗篇91篇 — 住在至高者隐密处

"住在至高者隐密处的，必住在全能者的荫下。"（91:1）

背景：信靠诗篇。

逐节分析：

  • 91:1-2 神的保护："住在至高者隐密处的，必住在全能者的荫下。我要论到耶和华说：他是我的避难所，是我的山寨，是我的神，是我所倚靠的。"

  • 91:3-8 各种保护："他必救你脱离捕鸟人的网罗和毒害的瘟疫。他必用自己的翎毛遮蔽你，你要投靠在他的翅膀底下。"

  • 91:9-13 天使的保护："因他要为你吩咐他的使者，在你行的一切道路上保护你。他们要用手托着你，免得你的脚碰在石头上。"

  • 91:14-16 神的应许："神说：因为他专心爱我，我就要搭救他。因为他知道我的名，我要把他安置在高处。他若求告我，我就应允他。他在急难中，我要与他同在。我要搭救他，使他尊贵。我要使他足享长寿，将我的救恩显明给他。"

新约应验：耶稣在旷野受试探时，魔鬼引用此诗（太4:6），但耶稣拒绝了试探。

10. 诗篇121篇 — 我的帮助从何而来

"我要向山举目，我的帮助从何而来？"（121:1）

背景：上行之诗，朝圣者的诗歌。

逐节分析：

  • 121:1-2 帮助的来源："我要向山举目，我的帮助从何而来？我的帮助从造天地的耶和华而来。"

  • 121:3-4 神的保护："他必不叫你的脚摇动，保护你的必不打盹。保护以色列的，也不打盹也不睡觉。"

  • 121:5-6 白日夜间："保护你的是耶和华，耶和华在你右边荫庇你。白日，太阳必不伤你；夜间，月亮必不害你。"

  • 121:7-8 一切灾害："耶和华要保护你，免受一切的灾害。他要保护你的性命。你出你入，耶和华要保护你，从今时直到永远。"

神学意义：神的保护是全天候的，从今时直到永远。

11. 诗篇125篇 — 信靠的保护

"倚靠耶和华的人好像锡安山，永不动摇。"（125:1）

背景：上行之诗。

逐节分析：

  • 125:1-2 信靠的稳固："倚靠耶和华的人好像锡安山，永不动摇。众山怎样围绕耶路撒冷，耶和华也照样围绕他的百姓，从今时直到永远。"

  • 125:3-5 公义的保护："恶人的杖不常落在义人的份上。耶和华啊，求你善待那些为善和心里正直的人。"

神学意义：倚靠神的人永不动摇，因为神围绕保护他们。

12. 诗篇131篇 — 谦卑安稳

"耶和华啊，我的心不狂傲，我的眼不高大。"（131:1）

背景：大卫的谦卑诗。

逐节分析：

  • 131:1 谦卑："耶和华啊，我的心不狂傲，我的眼不高大；重大和测不透的事，我也不敢行。"

  • 131:2 安稳："我的心平稳安静，好像断过奶的孩子在他母亲怀中；我的心在我里面真像断过奶的孩子。"

  • 131:3 劝勉："以色列啊，你当仰望耶和华，从今时直到永远。"

神学意义：在神面前存谦卑的心，如同断过奶的孩子安稳。

13. 诗篇138篇 — 感谢神

"我要一心称谢你，在诸神面前歌颂你。"（138:1）

背景：大卫的感恩诗。

逐节分析：

  • 138:1-3 感谢："我要一心称谢你，在诸神面前歌颂你。我要向你的圣殿下拜，为你的慈爱和诚实称赞你的名。"

  • 138:4-6 神的看顾："耶和华虽高，仍看顾低微的人。"

  • 138:7-8 信靠："我虽行在患难中，你必将我救活。耶和华必成全关乎我的事。"

神学意义：在患难中信靠神必成全关乎我们的事。

14. 诗篇139篇 — 神的全知

"耶和华啊，你已经鉴察我，认识我。"（139:1）

背景：大卫的诗篇，赞美神的全知全能。

逐节分析：

  • 139:1-6 神的全知："耶和华啊，你已经鉴察我，认识我。我坐下，我起来，你都晓得，你从远处知道我的意念。"

  • 139:7-12 神的全在："我往哪里去躲避你的灵？我往哪里逃躲避你的面？"

  • 139:13-18 神的创造："我在暗中受造，在地的深处被联络。我未成形的体质，你的眼早已看见了。"

  • 139:19-24 求神引导："神啊，求你鉴察我，知道我的心思；试炼我，知道我的意念。"

神学意义：神的全知和全在，是我们最大的安慰和挑战。

四、神学主题

1. 神的保护：在各种危险中神的保护 2. 神的同在：在各种境遇中神的同在 3. 神的信实：神的应许必然成就 4. 信靠的稳固：倚靠神的人永不动摇 5. 谦卑的心：在神面前存谦卑的心

五、新约关联

  • 诗16:10 → 徒2:25-28（基督的复活）
  • 诗23:1 → 约10:11（好牧人）
  • 诗27:1 → 约8:12（世界的光）
  • 诗46:10 → 可4:39（平静风浪）
  • 诗91:11-12 → 太4:6（耶稣受试探）
  • 诗121:3-4 → 约10:28-29（永生的保障）
  • 诗139:1-4 → 来4:13（神的全知）

六、实际应用

1. 在危险中信靠：在各种危险中信靠神的保护 2. 安然入睡：像大卫一样安然入睡，因为神保护我们 3. 等候神：在黑暗中等候神的时间 4. 渴慕神：在干旱之地渴慕神的同在 5. 谦卑安稳：在神面前存谦卑的心

七、祷告与默想

耶和华我的神，你是我的牧者，我必不致缺乏。你使我躺卧在青草地上，领我在可安歇的水边。我虽然行过死荫的幽谷，也不怕遭害，因为你与我同在。我要向山举目，我的帮助从造天地的耶和华而来。奉耶稣基督的名祷告，阿们。
