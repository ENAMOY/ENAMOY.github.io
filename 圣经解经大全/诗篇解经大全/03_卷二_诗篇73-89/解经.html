卷三：诗篇73-89篇 解经

一、诗篇背景

  • 经文范围：诗篇 73:1 - 89:53
  • 作者：亚萨（诗73-83）、可拉后裔（诗84-85, 87-88）、大卫（诗86）、以探（诗89）
  • 历史背景：涵盖北国以色列灭亡前后的历史时期，国家面临危机
  • 写作目的：如同利未记强调神的圣洁，本卷聚焦于圣所中的敬拜和神的圣洁

二、经文概览

卷三包含17篇诗篇，核心主题是圣所中的敬拜、神的圣洁以及国家危机中的信心。本卷充满对国家命运的哀叹，却以神的信实为根基。

三、逐节解经

1. 亚萨的哀歌（诗73-83篇）

诗篇73篇 — 义人为何受苦

"神实在恩待以色列那些清心的人！"（73:1）

亚萨的智慧诗。诗人曾为恶人的兴旺而困惑，几乎失脚。"我见恶人和狂傲人享平安，就心怀不平"（73:3）。直到他进了神的圣所，才明白他们的结局。"你实在是把他们安在滑地，使他们掉在沉沦之中"（73:18）。

这首诗教导我们，在苦难中不要凭眼见判断，而要进入神的同在中寻求答案。"除你以外，在天上我有谁呢？除你以外，在地上我也没有所爱慕的"（73:25）。

诗篇74篇 — 圣所被毁的哀歌

"神啊，你为何永远丢弃我们呢？"（74:1）

国家哀歌。圣所被毁，敌人在圣所中咆哮。诗人求神记念他的会众，记念他古时所行的奇事。"求你顾念所立的约"（74:20）。

诗篇77篇 — 在患难中寻求神

"我要向神发声呼求，我向神发声，他必留心听我。"（77:1）

亚萨的哀歌。诗人在夜间患难中寻求神，回忆神在古时所行的奇事。"你的道在海中，你的路在大水中，你的脚踪无人知道"（77:19）。

诗篇78篇 — 以色列的历史教训

"我的民哪，你们要留心听我的训诲。"（78:1）

历史诗篇。回顾以色列从出埃及到大卫王朝的历史，教训百姓不要重蹈覆辙。"他们在旷野悖逆他，在荒地叫他担忧"（78:40），但神仍怜悯他们。

诗篇79篇 — 耶路撒冷被毁的哀歌

"神啊，外邦人进入你的产业，污秽你的圣殿。"（79:1）

国家哀歌。耶路撒冷被外邦人攻占，圣殿被污秽。诗人求神拯救，为他名的缘故。

诗篇80篇 — 为国家祈求

"领约瑟如领羊群之牧者啊，求你留心听！"（80:1）

国家哀歌。以"牧者"的意象呼求神拯救以色列。"求你使我们回转，使你的脸发光，我们便要得救"（80:3, 7, 19）。

诗篇82篇 — 神施行审判

"神站在有权力者的会中，在诸神中行审判。"（82:1）

神审判世上的审判官。"我曾说：你们是神，都是至高者的儿子"（82:6），耶稣引用此诗（约10:34）。

诗篇83篇 — 求神击败仇敌

"神啊，求你不要静默！神啊，求你不要闭口，也不要不作声！"（83:1）

咒诅诗。求神击败以色列的仇敌。

2. 可拉后裔的诗（诗84-85, 87-88篇）

诗篇84篇 — 渴慕神的院宇

"万军之耶和华啊，你的居所何等可爱！"（84:1）

渴慕圣殿的诗篇。"在你的院宇住一日，胜似在别处住千日"（84:10）。"靠你有力量、心中想往锡安大道的，这人便为有福"（84:5）。

诗篇85篇 — 求神复兴

"耶和华啊，你已经向你的地施恩。"（85:1）

求神复兴国家。"慈爱和诚实彼此相遇；公义和平安彼此相亲"（85:10），描绘了神国度的和谐。

诗篇87篇 — 锡安的荣耀

"耶和华所爱的门，他爱锡安的门。"（87:2）

赞美锡安的诗篇。"论到锡安，必说：这一个那一个都生在其中"（87:5），预言万民归向锡安。

诗篇88篇 — 最黑暗的哀歌

"耶和华拯救我的神啊，我昼夜在你面前呼吁。"（88:1）

诗篇中最黑暗的哀歌。全诗以哀叹结束，没有转向赞美。"你把我的良朋密友隔在远处，使我所认识的人进入黑暗里"（88:8）。这教导我们，有时信心的道路是漫长而黑暗的。

3. 大卫的祷告（诗86篇）

诗篇86篇 — 大卫的祈求

"耶和华啊，求你侧耳应允我，因我是困苦穷乏的。"（86:1）

大卫的个人祷告。求神怜悯、保护、教导。"主啊，你本为良善，乐意饶恕人，有丰盛的慈爱赐给一切求告你的人"（86:5）。

4. 以探的诗（诗89篇）

诗篇89篇 — 神与大卫的约

"我要歌唱耶和华的慈爱，直到永远。"（89:1）

以探的智慧诗，是卷三的总结。前半部分（1-37节）赞美神的信实和与大卫所立的约："我与我所拣选的人立了约，向我的仆人大卫起了誓"（89:3）。后半部分（38-51节）哀叹约似乎被破坏。

但结尾仍是信靠："耶和华是应当称颂的，直到永远。阿们，阿们。"（89:52）。

四、神学主题

1. 神的圣洁：神在圣所中彰显他的圣洁 2. 信心的挣扎：在国家危机中坚持信心 3. 历史的教训：从以色列历史中学习功课 4. 神的约：神与大卫所立的永远之约 5. 圣殿的渴慕：渴慕神的同在

五、新约关联

  • 诗82:6 → 约10:34（耶稣的引用）
  • 诗89:3-4 → 路1:32-33（基督的王权）
  • 诗89:20 → 徒13:22（大卫蒙拣选）

六、实际应用

1. 在困惑中信靠：像亚萨一样，在困惑中进入神的同在寻求答案 2. 记念神的历史：从以色列历史中学习信心的功课 3. 渴慕神的同在：渴慕圣殿中神的同在 4. 在黑暗中坚持：像诗88篇教导的，有时信心的道路是黑暗的 5. 持守神的约：相信神的应许必然成就

七、祷告与默想

耶和华我的神，除你以外，在天上我有谁呢？除你以外，在地上我也没有所爱慕的。我的肉体和我的心肠衰残，但神是我心里的力量，又是我的福分，直到永远。求你使我渴慕你的同在，如同鹿切慕溪水。奉耶稣基督的名祷告，阿们。

"除你以外，在天上我有谁呢？除你以外，在地上我也没有所爱慕的。"（诗篇 73:25）
