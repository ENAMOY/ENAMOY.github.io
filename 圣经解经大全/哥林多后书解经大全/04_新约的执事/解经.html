04_新约的执事

经文范围：哥林多后书3:1-18

核心神学命题

1. 圣灵的荐信与事工的凭据（ἐπιστολή / epistolē, "书信、推荐信"）——保罗宣称信徒本身就是他事奉的活荐信，不是用墨水写在纸上，而是用永生神的灵写在心版上（πλαξὶν καρδίας / plaxin kardias）。这荐信的真伪不在于人手所持的书函，而在于受荐者生命中圣灵工作的可见果效。此命题建立了新约事工合法性的属灵根基：事工的权柄源于神，不源于人。

2. 新旧约的对比与超越（παλαιός καινός / palaios kainos, "旧的与新的"）——保罗以"字句"（γράμμα / gramma）与"灵"（πνεῦμα / pneuma）的对偶来区分旧约与新约的本质。旧约是刻在石版上的律法，带来定罪与死亡；新约是圣灵写在心版上的恩典，带来生命与义。这并非否定旧约的价值，而是揭示旧约乃是新约的影儿，新约才是实体。

3. 帕子的除去与自由（κάλυμμα / kalymma, "遮盖、帕子"）——摩西脸上的帕子象征旧约时代启示的有限性与渐进性。当以色列人读旧约时，帕子仍在心上，直到归向基督才被除去。基督是解开一切旧约预表与律法的钥匙，在他里面人才能真正看见神荣耀的本体。

4. 从荣耀到荣耀的更新（μεταμορφόω / metamorphoō, "改变形像、更新变化"）——信徒敞着脸观看主的荣光，就变成主的形像，从一个荣耀层次进入另一个荣耀层次。这是圣灵持续不断的工作，是渐进成圣的过程。"观看"（κατοπτρίζομαι / katoptrizomai）的原文带有"照镜子"的含义，暗示信徒在基督里看见自己真实的属灵光景与神的荣耀设计。

5. 基督是那灵（τὸ πνεῦμα / to pneuma, "那灵"）——"主就是那灵"（3:17）是本段最深奥的宣告之一。这并非将基督与圣灵混淆，而是指出复活升天的基督如今以圣灵的位格与工作继续在信徒中间运行。基督的自由正是圣灵的自由，在圣灵里不再有律法的定罪、不再有帕子的遮蔽、不再有死亡的辖制。

主题概述

哥林多后书第三章是保罗神学中最具深度的段落之一，集中论述新约职事的超越性。保罗面对哥林多教会中某些假使徒的挑战——他们可能持有来自耶路撒冷教会的荐信，质疑保罗事工的合法性——保罗以一个出人意料的策略回应：真正的荐信不是羊皮纸上的墨迹，而是信徒生命中圣灵改变的明证。

本章的核心论证采用层层递进的对比修辞：字句与灵、定罪的职事与称义的职事、旧约的荣光与新约更美的荣光、帕子的遮蔽与敞脸的自由。保罗以出埃及记三十四章摩西蒙面的故事为旧约经文基础，将整个旧约时代描绘为一个"帕子时代"——启示已有，但尚未完全；荣光已显，但尚被遮蔽。唯有在基督里，帕子才被彻底除去，人才得以直面神的荣耀。

这段经文不仅是保罗为自己使徒权柄的辩护，更是整本圣经关于旧约与新约关系最系统的神学阐述之一。它直接影响了早期教会的释经方法论——旧约必须透过基督的透镜来阅读，否则帕子仍在，字句仍杀。对于今天的信徒而言，本章邀请我们反思：我们是否仍在读经时带着帕子？我们是否真正敞着脸观看主的荣光？我们的事奉是否依赖圣灵的膏抹，还是仍在追求字句的精巧？

经文与逐句解经

一、保罗的荐信 (3:1-3)

3:1

我们岂是又举荐自己吗？岂像别人用人的荐信给你们，或用你们的荐信给人吗？

保罗以反问句开篇，回应那些质疑他使徒职分的人。"举荐"（συνίστημι / synistēmi）意为"推荐、证明、立定"，保罗在哥林多后书多次使用此词（3:1; 4:2; 5:12; 6:4; 10:12, 18），说明"自我推荐"是当时假使徒攻击他的主要论点之一。"荐信"（ἐπιστολή / epistolē）在古代世界极为常见——旅行的教师或使者通常携带其所属社群或权威人士的推荐函，以证明其身份与权柄的合法性。"别人"（τινές / tines）可能指那些从耶路撒冷来到哥林多的犹太基督徒教师，他们持有雅各或彼得等耶路撒冷领袖的推荐函。保罗并非否认荐信本身的价值，而是要指出：事工的合法性不能仅凭一纸文书来建立。真正的属灵权柄，其根基在于神的呼召与圣灵的印证，而非人的背书。参考保罗在加拉太书一章的辩护——他的使徒职分"不是由于人，也不是借着人，乃是借着耶稣基督"（加1:1）。这节经文为整段论证奠定了基础：保罗要展示一种超越人为荐信的、属灵的证据。

3:2

你们就是我们的荐信，写在我们的心里，被众人所知道所念诵的。

保罗给出了惊人的回答：哥林多信徒本身就是他的荐信。"写"（ἐγγράφω / engraphō）字面意为"刻写、铭刻"，与下文"石版"形成呼应。这封荐信不是写在羊皮纸或蒲草纸上，而是"写在我们的心里"（ἐν ταῖς καρδίας ἡμῶν / en tais kardias hēmōn）——保罗将哥林多信徒的属灵生命看作他使徒职分的活证据。"被众人所知道所念诵的"（γινωσκομένη καὶ ἀναγινωσκομένη / ginōskomenē kai anaginōskomenē）——"知道"（γινώσκω / ginōskō）指被认出、被辨别；"念诵"（ἀναγινώσκω / anaginōskō）则指被宣读、被朗诵。这两个动词构成精妙的文字游戏：前者是认知性的了解，后者是字面性的阅读。保罗的意思是：哥林多信徒生命中可见的改变，就如同一封人人可以阅读和辨认的公开信。无论假使徒如何质疑保罗的权柄，哥林多人在保罗事奉下得救、成圣、蒙恩的事实是无法否认的。这一论证方式也提醒每一位属灵工人：事奉最终的果效不在讲台上的雄辩，而在会众生命的转变。参约翰福音十五章8节："你们多结果子，我父就因此得荣耀，你们也就是我的门徒了。"

3:3

你们明显是基督的信，借着我们修成的。不是用墨写的，乃是用永生神的灵写的；不是写在石版上，乃是写在心版上。

这是本段的高峰，保罗展开了"基督之信"的神学隐喻。"基督的信"（Χριστοῦ ἐπιστολή / Christou epistolē）——这封荐信的真正作者不是保罗，而是基督自己；保罗只是执笔的器皿，"借着我们修成的"（διακονηθεῖσα ὑφ᾽ ἡμῶν / diakonētheisa hyph' hēmōn），"修成"（διακονέω / diakoneō）原意为"服事、执行"，保罗自视为基督的文书或中保。接下来的对比极为重要：不是用墨（μέλαν / melan），乃是用永生神的灵（πνεύματι θεοῦ ζῶντος / pneumati theou zōntos）；不是写在石版上（πλαξὶν λιθίναις / plaxin lithinais），乃是写在心版上（πλαξὶν καρδίας σαρκίναις / plaxin kardias sarkinaiis）。这组对比明显呼应了以西结书三十六章26-27节的应许："我也要赐给你们一个新心，将新灵放在你们里面。又从你们的肉体中除掉石心，赐给你们肉心。我必将我的灵放在你们里面。"以及耶利米书三十一章33节新约的核心应许："我要将我的律法放在他们里面，写在他们心上。"保罗在这里将旧约先知的预言与新约圣灵的更新工作直接联系起来，表明五旬节圣灵的降临正是这些古老应许的应验。"心版"（πλαξὶν καρδίας）对比"石版"（πλαξὶν λιθίναις），两处都用了"版"（πλάξ / plax）这个字，强调外在律法与内在生命之律的对比。律法刻在石上是外在的、冰冷的、无力改变人心的；圣灵写在心上是内在的、有生命的、能从根本上转化人的性情。

二、新约职事的超越性 (3:4-11)

3:4

我们因基督所以在神面前才有这样的信心。

"信心"（πεποίθησις / pepoithēsis）在此不是指信仰的信心（πίστις / pistis），而是指"确信、把握、胆量"。保罗宣告他事奉的自信与确据完全来自基督（διὰ Χριστοῦ / dia Christou），而不是来自自己的才能、资历或人的推荐。"在神面前"（πρὸς τὸν θεόν / pros ton theon）表明这种确信的终极法庭是神，不是人的评价。保罗在前面两节中展示了哥林多信徒作为他事奉果效的证据，现在他将论证的焦点转向这果效的真正来源——不是保罗自己，而是基督。这节经文承上启下，从荐信的讨论自然过渡到新旧约对比的主题：保罗之所以能写出这样一封"活的荐信"，是因为他所从事的新约职事有圣灵的大能运行。参考以弗所书三章12节："我们因信耶稣，就在他里面放胆无惧，笃信不疑地来到神面前。"又参考腓立比书三章3节，保罗称真受割礼的人是"在基督耶稣里夸口、不靠着肉体的"。

3:5

并不是我们凭自己能承担什么事；我们所能承担的，乃是出于神。

保罗在此做了一个关键的限定，防止读者将前文误解为自我吹嘘。"承担"（ἱκανός / hikanos）原意为"足够的、有能力的、配得的"，保罗彻底否认自己的能力有任何独立于神的来源。"凭自己"（ἐξ ἑαυτῶν / ex heautōn）与"出于神"（ἐκ τοῦ θεοῦ / ek tou theou）形成鲜明对比——一个是出于自我的有限源泉，一个是出于神的无限泉源。这节经文是理解保罗事工神学的关键：事奉的资格（ἱκανότης / hikanotēs）不是人自我培养出来的，而是神赐予的。保罗在哥林多后书四章7节进一步阐释这一主题："我们有这宝贝放在瓦器里，要显明这莫大的能力是出于神，不是出于我们。"这与当时假使徒的自我标榜形成对比——他们可能炫耀自己的学识、血统、口才和推荐信，而保罗却夸自己的软弱。参考耶利米书九章23-24节："智慧人不要因他的智慧夸口，勇士不要因他的勇力夸口，财主不要因他的财物夸口。夸口的却因他有聪明，认识我是耶和华。"保罗正是在实践这一原则：一切事奉的能力与资格都源于神的呼召与赐予，不源于人的资质。

3:6

他叫我们能承当这新约的执事，不是凭着字句，乃是凭着精意；因为那字句是叫人死，精意是叫人活（"精意"或作"圣灵"）。

这是本章的中心经文之一，保罗在此正式提出"新约执事"（διάκονοι καινῆς διαθήκης / diakonos kainēs diathēkēs）的概念。"执事"（διάκονος / diakonos）原意为"仆人、服事者"，保罗将自己和同工定位为新约的仆人，而非主人。"他叫我们能承当"（ἱκανόσεν ἡμᾶς / hikanosen hēmas）——"承当"（ἱκανόω / hikanoō）意思是"使有资格、使有能力"，再次强调事奉的能力是神所赋予的，呼应上节。接下来是本段最具神学分量的对比："字句"（γράμμα / gramma）与"精意"（πνεῦμα / pneuma）。和合本小字注明"精意"或作"圣灵"，原文确实是"灵"（πνεῦμα），可以指圣灵，也可以指属灵的意义。"字句叫人死"（τὸ γράμμα ἀποκτενεῖ / to gramma apoktenei）——律法本身是圣洁的，但它只能显明罪，却不能赦免罪；它只能定罪，却不能拯救。律法如同X光，能照出病灶，却无法治愈疾病。"灵叫人活"（τὸ δὲ πνεῦμα ζῳοποιεῖ / to de pneuma zōopoiei）——圣灵赐予新生命，使人心从根本上被更新。这呼应了耶稣对尼哥底母说的话："从灵生的，就是灵。……叫人活着的乃是灵"（约3:6; 6:63）。保罗并非在否定旧约——他自己在下文会说旧约也有"荣光"——而是在揭示旧约职事的本质局限：它是一面镜子，照出人的罪，却无法洗净人的罪。唯有在新约中，圣灵才能从根本上改变人心。参考罗马书七章10节："那本来叫人活的诫命，反倒叫我死了。"以及罗马书八章2节："因为赐生命圣灵的律，在基督耶稣里释放了我，使我脱离罪和死的律了。"

3:7

那用字刻在石头上属死的职事尚且有荣光，甚至以色列人因摩西面上的荣光，不能定睛看他的荣光；这荣光原是渐渐退去的。

保罗开始了一个精心构建的旧约论证，以出埃及记三十四章29-35节摩西蒙面的故事为经文基础。"属死的职事"（ἡ διακονία τοῦ θανάτου / hē diakonia tou thanatou）——保罗称旧约律法的职事为"属死的"，这并非贬低旧约，而是指出律法在人堕落光景中的实际效果：它定人的罪，宣告人的死罪。律法本身是圣洁的，但面对罪人的软弱，它成了定罪的工具。"用字刻在石头上"（ἐν γράμμασιν ἐντετυπωμένη λίθοις / en grammasin entetypōmenē lithois）——"刻"（ἐντυπόω / entypoō）意思是"雕刻、铭刻"，强调律法的外在性与固定性。"有荣光"（ἐγενήθη ἐν δόξῃ / egenēthē en doxē）——即便这"属死的职事"也有荣光！这是保罗论证的关键转折：旧约也有荣光，但新约的荣光要远远超越它。摩西从西奈山下来时，脸上因与神同在而发光，以色列人因这荣光不能定睛看他。"不能定睛看"（ἀτενίσαι / atenisai）意思是"凝视、注视"，因为荣光太强烈。"渐渐退去的"（τὴν καταργουμένην / tēn katargoumenēn）——"退去"（καταργέω / katargeō）意思是"废去、使之无效、使之停止"。这个细节极为重要：摩西面上的荣光是暂时的、会消退的，象征旧约启示的过渡性与暂时性。参考加拉太书三章19节："这样说来，律法是为什么有的呢？原是为过犯添上的，等候那蒙应许的子孙来到。"

3:8

何况那属灵的职事，岂不更有荣光吗？

这是一句简洁有力的"何况"（πολλῷ μᾶλλον / pollō mallon，"更加、更何况"）论证——保罗从旧约的"有荣光"推论到新约的"更有荣光"。"属灵的职事"（ἡ διακονία τοῦ πνεύματος / hē diakonia tou pneumatos）对比上文"属死的职事"——新约的职事是属灵的，因为它的动力、内容和果效都源于圣灵。这不是逻辑上的对比，而是程度上的递进：如果旧约那属死的、暂时的、会废去的职事尚且有荣光，那么新约这属灵的、永恒的、不会废去的职事岂不更有荣光？保罗的论证策略极为精妙：他没有贬低旧约，而是承认旧约的荣光，然后以旧约的荣光为基线，论证新约荣光的超越。这就像说："如果月亮尚且有光，何况太阳呢？"保罗在本章中反复使用这种"何况"（πολλῷ μᾶλλον）论证法（参3:9, 11），这是犹太释经传统中的"轻推重"（qal waḥomer）论证方式。参考希伯来书十二章25节的类似论证："你们总要谨慎，不可弃绝那向你们说话的。因为那些弃绝在地上警戒他们的，尚且不能逃罪，何况我们违背那从天上警戒我们的呢？"

3:9

若是定罪的职事有荣光，那称义的职事，荣光就越发大了。

保罗将旧约与新约的对比从"死与灵"推进到"定罪与称义"。"定罪的职事"（ἡ διακονία τῆς κατακρίσεως / hē diakonia tēs katakriseōs）——"定罪"（κατάκρισις / katakrisis）意思是"宣判有罪、定罪"。律法的功能之一就是定罪——它揭示人的罪，宣告人无法靠自己的行为达到神的标准。保罗在罗马书三章20节总结道："所以凡有血气的，没有一个因行律法能在神面前称义，因为律法本是叫人知罪。"然而，即便这个"定罪的职事"也有荣光，因为它是神圣的、出于神的。"称义的职事"（ἡ διακονία τῆς δικαιοσύνης / hē diakonia tēs dikaiosynēs）——"称义"（δικαιοσύνη / dikaiosynē）意思是"义、公义"，在此指神在基督里白白赐予信徒的义的地位。新约的职事宣告的是赦免、称义、与神和好的信息。"荣光就越发大了"（περισσεύει τῇ δόξῃ / perisseuei tē doxē）——"越发大"（περισσεύω / perisseuō）意思是"更加丰富、更加充足、充满有余"。保罗的意思是：如果宣告定罪的律法尚且有荣光，那么宣告称义的恩典岂不更充满荣光？这就好比说：如果法官的判决书有权威，那么总统的特赦令岂不更有权威？参考罗马书五章20节的总结："律法本是外添的，叫过犯显多；只是罪在哪里显多，恩典就更显多了。"

3:10

那从前有荣光的，因这极大的荣光，就算不得有荣光了。

保罗将论证推向高潮。"极大的荣光"（τῇ ὑπερβαλλούσῃ δόξῃ / tē hyperballousē doxē）——"极大的"（ὑπερβάλλω / hyperballō）意思是"超越、超过、凌驾"，原文带有"超越一切"的含义。在新约荣光的强烈照耀下，旧约的荣光相比之下"就算不得有荣光了"（οὐ δεδόξασται / ou dedoxastai）。这不是说旧约完全没有荣光，而是说旧约的荣光在新约荣光面前被大大超越了。这就像黎明的曙光在太阳完全升起后就看不见了——不是曙光没有光，而是太阳的光完全遮盖了曙光。保罗的修辞策略极为高明：他先肯定旧约有荣光，再以新约的荣光使旧约的荣光相形见绌。这种论证方式既尊重了旧约的神圣地位，又清晰地宣告了新约的超越性。参考希伯来书一章1-2节的平行思想："神既在古时借着众先知多次多方地晓谕列祖，就在这末世借着他儿子晓谕我们。"以及希伯来书八章13节："既说新约，就以前约为旧了；那渐旧渐衰的，就必快归无有了。"

3:11

若那废掉的有荣光，那长存的就更有荣光了。

保罗以"何况"（πολλῷ μᾶλλον / pollō mallon）论证作结，用"废掉的"（τὸ καταργούμενον / to katargoumenon）与"长存的"（τὸ μένον / to menon）的对比来总结整个论证。"废掉的"呼应3:7的"渐渐退去的"，指旧约职事的过渡性；"长存的"（μένω / menō，"持续、存留"）指新约职事的永恒性。保罗的最终论点是：如果那会过去、会废去的旧约尚且有荣光，那么那永远长存、不会废去的新约岂不更有荣光？这节经文完成了一个三重递进的论证结构：旧约有荣光 → 新约更有荣光 → 旧约的荣光在新约面前算不得荣光。这一论证的神学含义极为深远：旧约并非被废弃，而是被超越；旧约并非错误，而是不完全；旧约是影儿，新约是实体。参考希伯来书十章1节："律法既是将来美事的影儿，不是本物的真像，总不能借着每年常献一样的祭物叫那近前来的人得以完全。"又参考歌罗西书二章17节："这些原是后事的影儿；那形体却是基督。"

三、帕子与自由 (3:12-18)

3:12

我们既有这样的盼望，就大胆讲说。

"这样的盼望"（τοιαύτην ἐλπίδα / toiautēn elpida）指上文所论述的新约职事的荣光与永恒性。保罗之所以大胆讲说（πολλῇ παρρησίᾳ χρώμεθα / pollē parrēsia chrōmetha），正是因为他确信新约职事的超越荣光。"大胆"（παρρησία / parrēsia）意思是"坦率、公开、无惧"，这个字在新约中常用来描述信徒在神和人面前的坦然无惧（参徒4:13, 29, 31; 弗3:12; 腓1:20; 来4:16; 约壹2:28; 3:21; 4:17; 5:14）。保罗的大胆不是出于个人性格的鲁莽，而是出于对新约荣光的确信。如果新约的职事有如此超越的荣光，那么传讲这职事的人怎能畏缩不前？这节经文也暗示了保罗与假使徒的对比：假使徒可能靠着人的荐信和外在的资格来获取权柄，而保罗靠着新约职事本身的荣光来放胆传讲。参考使徒行传四章29-31节初代教会的祷告："他们恐吓我们，现在求主鉴察，一面叫你仆人大放胆量讲你的道。……祷告完了，聚会的地方震动，他们就都被圣灵放胆讲论神的道。"

3:13

不像摩西将帕子蒙在脸上，叫以色列人不能定睛看到那将废者的结局。

保罗在此开始以出埃及记三十四章33-35节的旧约叙事为基础进行寓意释经。"帕子"（κάλυμμα / kalymma）原文指覆盖物、遮盖物。根据出埃及记的记载，摩西与神说话后下山，脸上发光，以色列人因这荣光而惧怕靠近。摩西便用帕子蒙脸，直到再次进入会幕与神说话时才揭去帕子。保罗对这个故事进行了神学诠释：摩西蒙帕子的目的不只是因为以色列人惧怕，更是"叫以色列人不能定睛看到那将废者的结局"（πρὸς τὸ μὴ ἀτενίσαι τοὺς υἱοὺς Ἰσραὴλ εἰς τὸ τέλος τοῦ καταργουμένου / pros to mē atenisai tous hyious Israēl eis to telos tou katargoumenou）。"结局"（τέλος / telos）在此有双重含义：既指"终局、终结"，也指"目的、目标"。保罗暗示摩西蒙帕子的一个目的是不让以色列人看到旧约荣光终将废去的事实。旧约律法的荣光是暂时的、过渡性的，它的终极目的是指向基督。摩西的帕子在保罗的诠释中象征旧约启示的有限性——旧约不是最终的、完全的启示，而是渐进启示过程中的一个阶段。参考加拉太书三章23-25节："但这因信得救的理还未来以先，我们被看守在律法之下，直候那将来的真道显明出来。这样，律法是我们训蒙的师傅，引我们到基督那里，使我们因信称义。但这因信得救的理既然来到，我们从此就不在师傅的手下了。"

3:14

但他们的心地刚硬了，直到今日诵读旧约的时候，这帕子还没有揭去。这帕子在基督里已经废去了。

这是保罗论证最尖锐的一节，他将摩西的故事直接应用于当代以色列人的属灵光景。"心地刚硬"（ἐπωρώθη / epōrōthē）——"刚硬"（πωρόω / poroō）原意为"使变硬、使麻木"，如同骨头硬化或皮肤结茧。这个字在福音书中也用来描述门徒的"心硬"（可6:52; 8:17），以及犹太人对耶稣信息的拒绝。"直到今日"（ἄχρι γὰρ τῆς σήμερον / achri gar tēs sēmeron）——保罗在写信时距摩西时代已有一千五百年，但以色列人的属灵帕子仍在。当犹太人在会堂中诵读摩西的书（旧约）时，"这帕子还没有揭去"（τὸ αὐτὸ κάλυμμα ἐπὶ τῇ ἀναγνώσει τῆς παλαιᾶς διαθήκης μένει / to auto kalymma epi tē anagnōsei tēs palaias diathēkēs menei）。这不是说旧约本身有帕子，而是说读旧约的人心上有帕子。帕子的存在不是因为旧约晦涩，而是因为读旧约之人拒绝看到旧约指向基督的终极目的。"这帕子在基督里已经废去了"（ὅτι ἐν Χριστῷ καταργεῖται / hoti en Christō katargeitαι）——"废去"（καταργέω / katargeō）呼应3:7, 11，保罗反复使用同一个字，强调"废去"的主题贯穿整段。基督是解开一切旧约预表、律法和预言的钥匙。在基督里，旧约的帕子被除去，人可以清楚看见旧约的真正目的——指向基督。参考路加福音二十四章25-27节：复活的基督在以马忤斯路上"从摩西和众先知起，凡经上所指着自己的话都给他们讲解明白了"。又参考路加福音二十四章44节："摩西的律法、先知的书和诗篇上所记的，凡指着我的话都必须应验。"

3:15

然而直到今日，每逢诵读摩西书的时候，帕子还在他们心上。

保罗几乎逐字重复3:14的要点，但做了一个微妙的调整：从"诵读旧约"变为"诵读摩西书"（ἀναγινώσκεται Μωϋσῆς / anaginōsketai Moysēs），强调问题的核心在于对摩西五经的理解。犹太人以拥有摩西律法为荣，视之为神与他们立约的标记，但保罗指出，正是对摩西书的错误解读——只看到律法的字句，看不到律法指向基督的目的——构成了心上的帕子。"帕子还在他们心上"（κάλυμμα ἐπὶ τὴν καρδίαν αὐτῶν κεῖται / kalymma epi tēn kardian autōn keitai）——"还在"（κεῖται / keitai）意思是"躺卧、安放"，是一个持续的状态。帕子不是神故意放在以色列人心上的惩罚，而是他们拒绝接受基督的后果。参考约翰福音五章46节耶稣的话："你们如果信摩西，也必信我，因为他书上有指着我写的话。"又参考罗马书十一章25-26节："等到外邦人的数目添满了，于是以色列全家都要得救。如经上所记：必有一位救主从锡安出来，要消除雅各家的一切罪恶。"以色列人心上的帕子终有一天要被除去，这是神信实的应许。

3:16

但他们的心几时归向主，帕子就几时除去了。

这是一句极为重要的转折，保罗给出了除去帕子的条件。"心几时归向主"（ἡνίκα δ᾽ ἐὰν ἐπιστρέψῃ πρὸς κύριον / hēnika d' ean epistrephē pros kyrion）——"归向"（ἐπιστρέφω / epistrephō）意思是"转回、回转、悔改"，这是一个主动的意志行为，指心灵的回转与悔改。"主"（κύριος / kyrios）在此指基督——当以色列人的心回转归向基督时，帕子就被除去。"帕子就几时除去了"（περιαιρεῖται τὸ κάλυμμα / periaireitai to kalymma）——"除去"（περιαιρέω / periaireō）意思是"完全拿走、彻底挪去"，与3:14的"废去"（καταργέω）不同，这个字强调彻底的移除。保罗在此引用了出埃及记三十四章34节的叙述："摩西进到耶和华面前与他说话，就揭去帕子。"保罗将摩西进到耶和华面前揭去帕子的行为类比为以色列人归向基督时帕子的除去。在旧约中，摩西在神面前揭帕子；在新约中，信徒在基督面前帕子被除去。这节经文也暗示了一个普世性的原则：无论犹太人还是外邦人，任何人要真正明白圣经、认识神，都必须归向基督。帕子不是知识的缺乏，而是心灵的拒绝。参考使徒行传三章19节："所以你们当悔改归正，使你们的罪得以涂抹，这样，那安舒的日子就必从主面前来到。"

3:17

主就是那灵；主的灵在哪里，那里就得以自由。

这是本章最深奥、最具神学争议的宣告之一。"主就是那灵"（ὁ δὲ κύριος τὸ πνεῦμά ἐστιν / ho de kyrios to pneuma estin）——这并非将基督与圣灵混淆为同一位格（这会违背三一论的基本教义），而是指出复活升天的基督如今以圣灵的位格与工作继续运行在信徒中间。基督在升天前应许门徒："我要求父，父就另外赐给你们一位保惠师，叫他永远与你们同在，就是真理的圣灵"（约14:16-17）。基督虽然肉身升天，但他借着圣灵继续与信徒同在。因此，"主就是那灵"可以从几个层面理解：（1）主基督的工作与圣灵的工作是不可分割的；（2）在圣灵里，信徒与主基督相交；（3）圣灵是基督在地上的代理者与执行者。"主的灵在哪里，那里就得以自由"（οὗ δὲ τὸ πνεῦμα κυρίου, ἐλευθερία / hou de to pneuma kyriou, eleutheria）——"自由"（ἐλευθερία / eleutheria）是本节的关键词。这自由包括：（1）脱离律法定罪的自由——不再因不能守全律法而恐惧；（2）脱离帕子遮蔽的自由——敞着脸看见神的荣光；（3）脱离死亡权势的自由——在圣灵里有新生命；（4）从奴仆的心变为儿女的心——呼叫"阿爸，父"（罗8:15; 加4:6-7）。参考约翰福音八章36节："所以天父的儿子若叫你们自由，你们就真自由了。"又参考加拉太书五章1节："基督释放了我们，叫我们得以自由，所以要站立得稳，不要再被奴仆的轭挟制。"

3:18

我们众人既然敞着脸得以看见主的荣光，好像从镜子里返照，就变成主的形状，荣上加荣，如同从主的灵变成的。

这是整段经文的巅峰总结，也是新约中最深奥的成圣教义之一。"我们众人"（ἡμεῖς πάντες / hēmeis pantes）——保罗特别使用"众人"，强调这是所有信徒的共同经历，不仅仅是使徒或特别属灵的人。"敞着脸"（ἀνακεκαλυμμένῳ προσώπῳ / anakekalymmenō prosōpō）对比以色列人的帕子蒙脸——在基督里，帕子已被除去，信徒可以敞着脸直接面对神。"得以看见"（κατοπτριζόμενοι / katoptrizomenoi）——这个字的含义历来有不同解释：（1）"照镜子"——如同在镜子中观看；（2）"观看"——直接凝视。前者更符合字源学（κάτοπτρον / katoptron = 镜子），暗示信徒在基督这面"镜子"中看见神的荣光，也看见自己在神面前的真实光景。在古代世界，铜镜映照的影像并不完全清晰，因此保罗在哥林多前书十三章12节说："我们如今仿佛对着镜子观看，模糊不清。"但即使如此模糊的看见也足以产生改变。"就变成主的形状"（μεταμορφούμεθα / metamorphoumetha）——"变成"（μεταμορφόω / metamorphoō）意思是"改变形像、转化"，这是"变形"（metamorphosis）一词的来源。马太福音十七章2节和马可福音九章2节也用同一个字描述耶稣在山上变像。信徒在观看主的荣光时，就被圣灵改变，渐渐变成基督的形像。"荣上加荣"（ἀπὸ δόξης εἰς δόξαν / apo doxēs eis doxan）——从一个荣耀层次进入另一个荣耀层次，这是渐进成圣的过程。信徒不是一步到位地达到完全，而是在圣灵的工作中不断成长、不断更新、不断更像基督。"如同从主的灵变成的"（καθάπερ ἀπὸ κυρίου πνεύματος / kathaper apo kyriou pneumatos）——最终，这更新变化的工作归功于"主的灵"——圣灵。成圣不是人意志力的成果，而是圣灵在人心里持续不断的工作。参考罗马书八章29节："因为他预先所知道的人，就预先定下效法他儿子的模样，使他儿子在许多弟兄中作长子。"又参考罗马书十二章2节："不要效法这个世界，只要心意更新而变化，叫你们察验何为神的善良、纯全、可喜悦的旨意。"以及约翰壹书三章2节："亲爱的弟兄啊，我们现在是神的儿女，将来如何，还未显明；但我们知道，主若显现，我们必要像他，因为必得见他的真体。"

神学要点总结

1. 新约职事的内在性与转化性：旧约律法是刻在石版上的外在准则，新约圣灵是写在心版上的内在生命。律法改变行为的外壳，圣灵改变人的心灵本质。这一对比奠定了保罗恩典神学的根基——救恩不是靠遵行外在律法，而是靠圣灵在内心的更新。

2. 旧约与新约的连续性与超越性：保罗既没有废弃旧约，也没有将旧约与新约等同。旧约有荣光，新约有更美的荣光；旧约是渐进启示的一个阶段，新约是启示的完全。旧约是指向基督的影儿，新约是基督成全的实体。这种"超越但不废弃"的关系是理解整本圣经的关键。

3. 基督是解开旧约的钥匙：帕子的比喻表明，没有基督，旧约就是一部被帕子遮蔽的书。以色列人虽然拥有旧约圣经，却因为拒绝基督而无法真正理解旧约的精义。这为基督徒释经学提供了根本原则：旧约必须透过基督的透镜来阅读。

4. 渐进成圣的圣灵工作："从荣耀到荣耀"描述了信徒在圣灵里的持续成长。成圣不是一次性完成的事件，而是渐进的过程；不是人意志力的挣扎，而是圣灵的工作；不是孤立的个体努力，而是在"观看主荣光"的团契生活中发生的。信徒的责任是敞着脸观看，圣灵的工作是改变形像。

5. 自由是新约的标记：圣灵在哪里，那里就有自由——脱离定罪的自由、脱离帕子的自由、脱离死亡的自由。这自由不是放纵的自由，而是从罪的权势下释放出来、甘心顺服圣灵引导的自由。参考加拉太书五章13节："弟兄们，你们蒙召是要得自由，只是不可将你们的自由当作放纵情欲的机会，总要用爱心互相服事。"

应用反思

保罗在这段经文中给我们最重要的提醒是：事奉的根基不是人的才能、资历或推荐，而是圣灵的膏抹与运行。今天教会中常常出现"荐信文化"——人们看重学位、头衔、名人推荐，却忽略了生命改变的证据。保罗说："你们就是我们的荐信。"一个牧者、传道人或教师的真正资格，不在于他拥有什么文凭或头衔，而在于他的事奉是否带来人生命的改变。当我们评估自己或他人的事奉时，应当问：有没有人因此认识基督？有没有人的生命被圣灵更新？这才是真正的"荐信"。

这段经文也挑战我们对读经的态度。如果保罗说以色列人诵读旧约时帕子仍在心上，我们是否也可能在读经时带着帕子？一个可能的帕子是先入为主的偏见——我们带着自己的想法来读经，让经文来迎合我们的观点，而不是让经文来挑战和改变我们。另一个可能的帕子是知识的骄傲——我们以为自己已经明白了一切，就停止了向圣灵敞开、让圣灵教导的心。愿我们每次打开圣经时都祈求圣灵除去我们的帕子，使我们能敞着脸看见基督的荣光。

最后，"从荣耀到荣耀"的渐进成圣应鼓励每一位灰心的信徒。属灵的成长不是一夜之间完成的，有时我们觉得自己毫无进展，有时甚至觉得自己在退步。但保罗说：我们是"从荣耀到荣耀"，这是圣灵持续不断的工作。只要我们继续敞着脸观看主的荣光，圣灵就在我们里面做工，使我们渐渐变成基督的形像。我们不需要靠自己的力量来改变自己，只需要持续注视基督，圣灵自然会在我们身上完成他美好的工作。参考腓立比书一章6节："我深信那在你们心里动了善工的，必成全这工，直到耶稣基督的日子。"

与前文的联系：哥林多后书二章14节至三章6节构成一个完整的论证单元。保罗从"凯旋行列"的比喻（2:14-17）开始，宣告自己是基督得胜行列中的使者，散播因基督而有的馨香之气。接着他回应"自我推荐"的指控（3:1-3），以信徒的生命作为他事奉的活荐信，然后转入新旧约职事的对比（3:4-11）。整个论证的逻辑链条是：保罗事奉的合法性不在于人的推荐，而在于他所服事的新约职事本身具有的超越荣光。

与后文的联系：三章18节"从荣耀到荣耀"的成圣主题自然过渡到四章1-6节保罗对自己事奉方式的描述——"乃是将那些暗昧可耻的事弃绝了"（4:2）。正因为保罗在圣灵的光中敞着脸面对神，他才能以无亏的良心、诚实的态度传讲福音。四章7-15节"瓦器中的宝贝"的比喻则是对三章5节"并不是我们凭自己能承担什么事"的具体展开——瓦器会碎裂，但宝贝（基督的荣光）永远长存。整本哥林多后书的信息在此形成了有机的统一：事奉的力量不在器皿本身，而在器皿中的宝贝；荣光不在旧约的字句上，而在新约的灵里；自由不在律法的遵行中，而在圣灵的更新里。
